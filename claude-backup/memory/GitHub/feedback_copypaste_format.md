---
name: コピペしやすい形式で出力する
description: ユーザーに入力してもらう内容は、常にコピー＆ペーストしやすいコードブロック形式で提示する
type: feedback
---

ユーザーに入力・コピペしてもらう情報は、必ず簡単にコピー＆ペーストできる形式（コードブロック等）で提示すること。

**Why:** ユーザーが手作業で入力する手間を最小限にするため。説明文の中に埋もれた形で提示すると、コピーしにくく非効率。

**How to apply:** フォーム入力、API設定、コマンド実行、設定値など、ユーザーがどこかに貼り付ける必要がある情報は、全てコードブロック（```）で囲んで提示する。手順も番号付きで順序立てて、1ステップずつ明確に指示する。
