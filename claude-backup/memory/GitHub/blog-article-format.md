## 全記事共通フォーマット（2026-03-13確定）

### ヘッダー（widget custom_html-2で制御）
- ロゴ（丸ロゴ + 南米おやじ文字）: **左寄せ**
- ナビ4項目（パラグアイ生活/海外からの稼ぎ方/お金と手続き/プロフィール）: **右寄せ**（`margin-left:auto`）
- Apple風フロストガラス背景、sticky

### サイドバー（widget custom_html-3で制御）
- Apple風角丸カードスタイル（`.nambei-sidebar-section`）
- 構成: プロフィール → 検索 → カテゴリー → 新着記事 → 目次（sticky）
- カテゴリページ・記事ページ・固定ページすべて同じスタイル

### 禁止事項（全記事共通）
- **記事本文内にプロフィール（筆者プロフィール）を書かない** — サイドバーにあるので不要
- **目次は1つだけ** — Cocoonデフォルト目次のみ。widget挿入の目次（tcd-toc, details）は無効化済み
- **heroヘッダー無効化** — タイトル・アイキャッチの重複防止のため`createHeroHeader()`は無効
- **SNSシェアボタン非表示** — `.sns-share`等はCSS非表示
- **大きなSVGアイコン禁止** — SVGには必ずwidth/height属性を付ける

### 新規記事投稿時の注意
- 記事末尾にプロフィールブロックを入れない
- excerptを必ず設定する（CSSがカテゴリページに漏れるのを防ぐ）
- Puppeteerで公開後にフォーマット確認すること

**Why:** 過去にheroヘッダーによるタイトル・画像重複、SVGアイコン巨大化、目次重複、プロフィール重複が発生した
**How to apply:** 記事作成・テーマ変更時に必ずこのルールを確認。Puppeteerで全記事一括チェック可能
