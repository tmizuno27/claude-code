name: Gumroad出品時は全項目を漏れなく提示
description: 商品出品手順では全入力欄（Name, Price, Summary, Description, Tags, Thumbnail, Content）を一括で提示する
type: feedback
---

Gumroad等のプラットフォームへの商品出品手順を案内する際、**全ての入力欄を漏れなく一度に提示する**こと。

**Why:** Summary欄を空欄のまま案内してしまい、ユーザーに二度手間をかけた。コンテンツファイルも未作成のまま出品手順を案内し、手順が前後した。

**How to apply:**
- 出品手順を提示する前に、必要なコンテンツ（PDF、画像、テンプレート等）を全て先に完成させる
- 手順提示時は全入力項目をチェックリスト的に網羅する（Name, Price, Summary, Description, Tags, Thumbnail, Content, URL slug等）
- 「空欄でOK」の項目があれば明示的に「空欄でOK」と書く
- 抜け漏れで二度手間にさせない
