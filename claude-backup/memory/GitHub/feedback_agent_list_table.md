---
name: エージェント一覧は表形式で出力
description: 「エージェント一覧」等の文脈で、全チーム・エージェント・スキルを呼び出し方付き表形式で即表示する
type: feedback
---

「エージェント一覧」「チーム一覧」はもちろん、「エージェント」「スキル」「チーム」など短い一言でも、一覧を求めている文脈と判断したら表形式で出力する。

**Why:** ユーザーがすぐに `@エージェント名` や `/スキル名` をコピペして呼び出せるようにするため。

**How to apply:** チーム（Team Leaders）→ 個別エージェント（Agents）→ カスタムコマンド（Slash Commands）→ ビルトインスキル（Skills）の4セクションに分け、各行に「名前・呼び出し方・得意作業」を記載。使い分けの目安表も添える。カスタムコマンドは `.claude/commands/` 配下の `.md` ファイルを `/コマンド名` で呼び出す。
