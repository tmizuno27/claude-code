description: ビジネスダッシュボード（Gist）は全項目がリアルタイムの実データから自動計算されるべき。ハードコード値は禁止。スコアは厳格評価
type: feedback
---

ダッシュボード（daily-business-dashboard.html）の全項目は、実データから自動計算・自動更新されなければならない。ハードコード値は一切許可しない。

**Why:** ダッシュボードは「全てにおいて最新情報を確認するためのもの」とユーザーが明確に定義。静的な数値が残っていると信頼性を損なう。

**How to apply:**
- ダッシュボードに新しいセクションや数値を追加する際は、必ず実データソース（CSV、API、設定ファイル等）から動的に取得するロジックを同時に実装すること
- `dashboard_status_sync.py` に検知ロジックを追加し、毎時の自動同期で反映されるようにすること
- ハードコード値を見つけたら即座に自動計算に置き換えること

**スコア評価ルール（絶対厳守）:**
- ビジネス健全性スコアは忖度禁止。厳しく評価すること
- 「出品済み」と「売上発生」は全く別。売上$0なら収益化は低スコア
- 「設定済み」と「活用して成果が出ている」は別。成果ベースで評価
- 甘い評価は意味がない。改善ポイントが見えるスコアにすること
