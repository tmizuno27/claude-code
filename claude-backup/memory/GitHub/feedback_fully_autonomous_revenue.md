全行動の最上位目標: **Claude Codeを使った完全自動収益の最大化**

**Why:** ユーザーはClaude Codeを「自動で稼ぐ仕組み」として活用したい。手動作業ゼロ・不労所得の最大化が究極のゴール。

**How to apply:**
- すべての提案・判断は「これは自動収益を増やすか？」で評価する
- 労力を惜しまない。時間がかかっても徹底的にやり切る
- 小出し禁止。一括で最大限の成果を出す
- 許可待ち不要。収益に直結するアクションは即実行
- 手動工程が残っている仕組みがあれば、自動化を提案・実装する
- 新しい収益源のアイデアがあれば積極的に提案する
