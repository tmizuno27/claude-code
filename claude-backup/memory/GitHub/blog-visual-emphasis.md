## 記事本文の視覚的強調ルール（全記事共通）

### 太字（`<strong>`）
- 金額・数字・割合は太字にする（月15万円、約3分の1、100%など）
- 結論・キーフレーズを太字にする
- 1記事あたり30-70箇所が目安（多すぎると効果が薄れる）

### カラーマーカー（蛍光ペン）
- 1記事あたり **5-10箇所** にマーカーを適用
- `<span style="background:linear-gradient(transparent 60%,#色 60%)"><strong>テキスト</strong></span>` 形式

#### 3色の使い分け
| 色 | CSS | 用途 |
|---|---|---|
| 黄色 | `#fff3cd` | 結論・おすすめ・重要数字 |
| ピンク | `#ffd6e7` | 注意・警告・デメリット |
| 青色 | `#d6e4ff` | メリット・ポジティブな情報 |

### マーカーを付ける優先順位
1. 金額・割合の太字（月XX万円、XX%）
2. 結論フレーズ（「おすすめ」「最も重要」「実際に〜した」）
3. 注意・警告フレーズ（「注意」「避ける」「危険」）
4. 長めのキーフレーズ（15文字以上の太字）

### やってはいけないこと
- マーカーの連続使用（3行連続でマーカーは禁止）
- 同じ色のマーカーが5つ以上連続（色を交互に使う）
- 見出し（h2/h3）へのマーカー適用

**Why:** 太字だけだと単調。カラーマーカーで視覚的なリズムを作り、読者が重要ポイントを一目で把握できるようにする
**How to apply:** 新規記事作成時に必ず5-10箇所のマーカーを挿入。既存記事修正時もマーカーの有無を確認
