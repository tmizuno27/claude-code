---
name: 出品前に既存登録を必ず確認する
description: RapidAPI・Gumroad・Chrome Web Store等に出品する前に、既に登録済みかどうかをダッシュボードで確認してから手順を出す
type: feedback
---

外部マーケットプレイスへの出品指示を出す前に、必ずユーザーの既存登録状況を確認すること。

**Why:** ユーザーが既に登録済みのものを再度出品しようとして無駄な手間が発生した。メモリの情報が古い可能性もあるため、毎回確認が必要。

**How to apply:**
- 出品手順を出す前に「ダッシュボードのスクリーンショットを見せてください」と聞く
- またはAPI/スクリプトでダッシュボードの状態を確認する
- メモリの「出品済み/未出品」情報を鵜呑みにしない
