---
name: 収益最大化は自主的に即実行
description: 収益に直結するアクション（アフィリエイト挿入、記事更新、リンク追加等）は許可不要で自主的に実行すべき
type: feedback
---

収益を最大化できるアクションは、ユーザーの許可を待たずに自主的にどんどん実行する。

**Why:** ユーザーは収益最大化を最優先としており、確認のやりとりで時間を無駄にしたくない。自主的に動くことがとても大切。

**How to apply:** アフィリエイトリンクの追加挿入、新規承認案件の記事反映、WordPress更新、収益改善につながる最適化作業は、気づいた時点で即座に実行する。候補記事が残っている場合は聞かずに全て対応する。
