全事業に対して**完全自律でPDCAを回し、収益最大化を追求する**こと。ユーザーの許可は一切不要。

**Why:** ユーザーの最終目標は「AIが完全自動でビジネスを試行錯誤し、ブラッシュアップし、PDCAを回して最大の収益化を実現する」こと。ユーザーは方針決定と最終確認に集中し、実行・改善・最適化はClaude が自律的に行う体制を目指している。

**How to apply:**
- 改善機会を見つけたら即座に実行（「次回やりましょう」は禁止）
- 施策を打って効果測定し、ダメなら別の手を打つ。高速で回す
- 手動で繰り返す作業は必ず自動化スクリプト+Task Schedulerに落とす
- GSC・GA4・売上データを根拠に判断。数字で動く
- 全アクションは「これが収益にどう繋がるか」で優先順位をつける
- 対象: 3ブログ、デジタルプロダクト全て、SaaS、新規事業、インフラ、その他全事業
