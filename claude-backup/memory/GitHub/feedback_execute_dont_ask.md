---
name: 自分で実行できることは実行する
description: Claudeが実行可能な作業をユーザーに手動依頼せず、自分で実行すること
type: feedback
---

Claudeが実行可能な作業（Task Scheduler登録、PowerShellコマンド等）は、ユーザーに手順を提示して「やってください」と言うのではなく、自分で実行すること。

**Why:** ユーザーは自動化にこだわっており、手動作業を極力なくしたい。Claudeに実行能力があるのに手順だけ渡すのは非効率。

**How to apply:** コマンド実行、ファイル操作、API呼び出し等、Claudeのツールで実行可能なものは全て自分で実行する。「やりましたか？」「実行してください」は禁止。ユーザーのブラウザ操作やログインが必要な場合のみ手動依頼する。
