description: 自動実行可能なタスクをユーザーに確認せず即座に実行する。「やりますか？」は禁止
type: feedback
---

自動実行可能なタスクは確認なしで即座に実行すること。「進めますか？」「実行しますか？」は禁止。

**Why:** 2026-03-25にユーザーから「Claudeが実行可能な作業は自分で実行すること」と指示。2026-03-27にさらに「聞くまでもないです。自動で進められるのが私に聞かずに、自律的にどんどん押し進めてください」と明確に強化された。

**How to apply:** 優先タスクや改善施策を列挙した後、自動実行可能なものは即座に並列エージェントで実行開始する。手動対応が必要なもの（Stripe KYC、AdSense申請等ブラウザログインが必要なもの）のみ残タスクとして報告する。タスク一覧を提示して「やりますか？」と聞く行為は厳禁。
