## オンラインセールス（フリーランス）

- **ステータス**: 稼働中（2025年〜現在）
- **収益モデル**: 業務委託報酬（労働収入）
- **位置づけ**: 生活費を支える現金収入源。Webメディア事業が育つまでの主力
- **バックグラウンド**: パーソナルジム店舗責任者 → 個別指導塾教室長 → Amazon QA Team Manager → オンラインセールス
- **スキル**: セールス心理学・ヒアリング技法を体系的に学習（`archive/sales/` に7章分のナレッジベース）

**Why:** Webメディア（アフィリエイト3サイト）はまだ収益化前段階のため、セールスが生活の基盤
**How to apply:** 事業全体の優先順位を考える際、セールス稼働時間とのバランスを考慮する
