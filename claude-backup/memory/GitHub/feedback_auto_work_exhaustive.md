---
name: 自動作業は徹底的にやり切る
description: 「自動で進めて」と言われたら、これ以上できることがないレベルまで全て実行する。コスト・時間制限なし
type: feedback
---

「自動的に作業を進めて」「自動で進められるものは？」と言われたら、**1回の指示で全事業・全スクリプト・全サービスを網羅的に洗い出し、自動実行できるものは全て並行で実行し切る**こと。

「これ以上自動で進められるものはありません」と言えるレベルまで徹底する。

**Why:** ユーザーは毎回「他にないですか？」と聞く手間を省きたい。1回の指示で完結させるのが理想。
**How to apply:**
- 最初の「自動で進めて」で、全事業のスクリプト・ログ・外部サービス・キュー・ヘルスチェックを一括調査
- 実行可能なものは全て並行エージェントで即実行
- コスト・時間の制限は気にしない（ユーザーが明示的に許可済み）
- 小出しにせず、最初から全部やる
