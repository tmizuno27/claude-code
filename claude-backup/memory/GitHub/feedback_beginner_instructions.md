手順説明は**常に初心者向け**に、1ステップずつ詳細に書くこと。

**Why:** ユーザーから「私はいつでも初心者だと認識してください」と明確に指示された。技術的な操作（Vercel、Google Cloud、AdSense等）は特に丁寧に。

**How to apply:**
- 「〇〇をクリック」ではなく「画面の△△にある『〇〇』という青いボタンをクリック」のように具体的に
- 1画面1ステップ、スクショを貼ってもらいながら進める
- URLは必ずコピペ形式で提示
- 「次は？」と聞かれる前に次のステップまで説明する
- 専門用語は使わない、使う場合は簡単に説明を添える
