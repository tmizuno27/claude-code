# ブログ記事作成 SEOルール

## 基本方針
ブログ記事（南米おやじのAI実践ラボ）を作成する際は、**毎回必ず最新のSEO手法とトレンドをリサーチ**してから執筆すること。

## 参考チャンネル・情報源
- **ウェブ職TV（なかじ / 中島大介）**: https://www.youtube.com/@webshokutv
  - SEO記事構成、タイトルの付け方、クリック率向上テクニック
  - ブログライティングの教科書（著書）
  - 株式会社メリル代表、ブログ歴17年のプロ
  - 2025年〜LLMO診断サービスも展開、チャンネル登録者14万人超

## 記事作成時の必須アクション
1. **WebSearchで最新SEOトレンドを調査**（直近2-3年のアルゴリズム変動・手法を確認）
2. **最新のGoogle SEOアップデート情報を確認**（コアアップデート、HCU、スパムポリシー等）
3. **LLMO/GEO/AIO対策を意識**（AI検索に引用されやすい構造にする）
4. **競合記事を調査**（ターゲットKWで上位表示されている記事の構成を参考にする）
5. **E-E-A-T（経験・専門性・権威性・信頼性）を最大化**
6. **独自性のある一次情報**を必ず含める（パラグアイ在住の実体験）

## なかじ式SEOテクニック（常に適用）
- **タイトル**: メインKWを先頭に、30字前後、具体的な数字、ベネフィット明示
- **結論ファースト**: リード文の冒頭1-2文で質問に対する端的な回答を書く（LLMO対策兼用）
- **比較表**: 読者の意思決定を助ける比較表を積極的に挿入
- **FAQ構造化データ**: 3-6問のFAQセクションでリッチリザルトを狙う
- **ブランディング**: タイトルに信頼性を高める要素（「在住者が」「実体験」等）
- **割り切れない数字**: 「5選」より「7選」のように信頼感のある数字を使う
- **KW密度1-2%**: 自然な配置、詰め込み禁止
- **リード文100字以内にKW配置**
- **内部リンク2-3箇所**: パラグアイ記事⇔AI副業記事の相互リンク
- **トピッククラスター**: ピラー記事とクラスター記事を内部リンクで繋ぎ、トピカルオーソリティを構築

## 最新SEOトレンド（2026年3月更新）

### Googleアルゴリズム動向
- **2026年2月コアアップデート**: 薄いAIコンテンツがペナルティ対象に。トピカルオーソリティとコンテンツ深度を重視。Discoverアルゴリズムも刷新され、地域関連性・専門性・独創性・時事性が重要に
- **2025年12月コアアップデート**: 「実体験（lived experience）＞形式的な資格」が評価軸に。AIと非AIコンテンツの差別化要因として実体験が決定的に重要
- **2025年3月コアアップデート**: スパムポリシー更新、「abused reputation（悪用されたレピュテーション）」を新たにスパム定義。第三者コンテンツの悪用も明確にスパム化
- **YMYL分野の厳格化**: E-E-A-Tの評価基準がさらに厳しく、特に「Experience」「Expertise」の比重増大

### AI生成コンテンツの評価
- **Google品質ガイドライン（2025年1月改定）**: 「Generative AI」を初めて14回言及
- **AI単体の自動生成**: 最低品質評価を受ける可能性あり
- **AI+人間の編集**: 独自性・付加価値がある場合は問題なし
- **AI記事の大量生成**: KW詰め込みAI記事は平均20位ダウン、テンプレ量産は検索結果から消失
- **対策**: 必ず実体験・独自データを加え、AI出力のままにしない

### LLMO/GEO/AIO（AI検索最適化）- 2026年の新パラダイム
- **LLMO（Large Language Model Optimization）**: ChatGPT等のLLMに引用される対策
- **GEO（Generative Engine Optimization）**: Google AI Overview、Perplexity等の生成AI検索対策
- **AIO（AI Optimization）**: 上記の包括的な用語
- **Gartner予測**: 2026年までに従来の検索エンジントラフィックが25%減少
- **対策方法**:
  - Schema.orgの構造化データ（FAQ、HowTo、Article等）
  - FAQ形式で端的に回答するコンテンツ
  - 複数フォーマットでの情報提供（テキスト、テーブル、リスト、図解）
  - 冒頭で質問に対する直接的な回答を書く（answer-firstセクション）
  - 権威あるソースからの引用・参照
  - エンティティマッピング（純粋なKW最適化よりエンティティ定義を重視）
  - オフページシグナル強化（AI応答でのブランド言及の85%は第三者ソースから）
  - LLMが自明に再生成できない独自インサイトを含める

### コンテンツ戦略
- **トピッククラスター**: ピラー記事+クラスター記事の内部リンク構造で「トピカルオーソリティ」を獲得
- **独自データ**: オリジナルの写真、調査データ、家計簿などの一次情報がSEO評価に直結
- **検索意図の深堀り**: 表面的なKWマッチではなく、検索者の本当の悩みに応える構成
- **既存記事の定期リライト**: Search Consoleで11-30位の記事に著者情報追記、最新データ更新、独自体験追加
- **ロングテールKW**: 3語以上の組み合わせで初期は低競合を狙う

## 記事作成のワークフロー
```
1. WebSearchで以下を調査:
   - 「[ターゲットKW] SEO 2025 2026」
   - 「Google コアアップデート 最新」
   - 「LLMO GEO 対策 ブログ」
   - 競合上位記事の構成
2. 最新のGoogleアルゴリズム変動を確認
3. 競合上位5記事の構成を分析
4. なかじ式テクニック + 最新トレンド + LLMO対策を反映した構成案を作成
5. 記事テンプレート（templates/article-template.md）に沿って執筆
6. ライティングルール（agents/article-writer-agent.md）を遵守
7. フロントマターにseo_notes（適用した最新SEO対策のメモ）を記載
```

## 記事の品質チェックリスト
- [ ] リード文冒頭で質問に端的に回答しているか（LLMO対策）
- [ ] タイトルにメインKWが先頭にあるか
- [ ] パラグアイ在住の実体験が1セクション以上あるか
- [ ] 比較表が1つ以上あるか
- [ ] FAQが3〜6問あるか
- [ ] 内部リンクが2〜3箇所あるか（トピッククラスター意識）
- [ ] アフィリエイトリンクが自然に配置されているか（1記事5本以内）
- [ ] 禁止表現（AI臭い表現）を使っていないか
- [ ] 【要追記】【写真:】マークが適切に入っているか
- [ ] 独自データ（価格、日付、具体的エピソード）が含まれているか
- [ ] フロントマターのseo_notesに適用した最新対策が記載されているか
