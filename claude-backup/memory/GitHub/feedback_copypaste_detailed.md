---
name: コピペ形式で全項目を提示する
description: 外部サービスへの入力時は、全入力項目をコードブロックで提示し、空白でOKな項目も明示する
type: feedback
---

外部サービス（Gumroad、RapidAPI、Chrome Web Store等）への出品・設定時は、コピペ形式で全項目を網羅して提示すること。

**Why:** ユーザーは項目ごとに「これは入力すべきか？」と迷う。Claudeが先回りして全項目の入力内容を提示すれば迷わず進められる。

**How to apply:**
- 各入力項目をコードブロック（```）で囲んで提示
- 空白でOKな項目も「→ 空白でOK」と明示する
- 「これも入力した方がいいですよ」とClaudeから積極的に提案する
- 1商品ずつ順番に提示し、完了を確認してから次へ進む
- サムネイル・カバー画像のファイルパスも毎回明示する
