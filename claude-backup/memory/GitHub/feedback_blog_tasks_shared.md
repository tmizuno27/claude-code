name: ブログ自動タスクは全サイト共通
description: 自動タスクを追加する際は全3ブログ（nambei-oyaji.com, otona-match.com, sim-hikaku.online）に等しく適用すること
type: feedback
---

自動タスク（Task Scheduler）は全ブログサイトで共通にすること。1サイトにタスクを追加したら、他の2サイトにも同じタスクを追加する。

**Why:** ユーザーは3サイト均等運営を望んでおり、nambei-oyaji.comだけ15タスク、他2サイトが1-3タスクという不均衡が発生していた。

**How to apply:** 新しい自動タスクを作成・提案する際は、必ず3サイト分セットで考える。ホームページ系タスクも同様に全サイト共通。
