---
name: 画像・アセット等を後回しにしない
description: サムネイル・画像・コンテンツ等の必要素材は出品手順の提示前に全て用意する。後回し禁止。
type: feedback
---

出品や公開の手順を案内する際、サムネイル画像・コンテンツファイル等の**必要素材は全て事前に用意してから手順を提示する**こと。

**Why:** 「後でやればいい」と画像作成をスキップさせた結果、作業が中断し工程が遅れた。ユーザーは一気に完了させたい。

**How to apply:**
- 出品・公開手順を案内する前に、必要な全素材（画像、PDF、テンプレート等）を先に作成する
- 「スキップしてOK」「後回しでOK」は禁止
- 素材が足りなければ、手順案内を中断してでも先に素材を作成する
- 1つのタスクは中断なく一気に完了させる
