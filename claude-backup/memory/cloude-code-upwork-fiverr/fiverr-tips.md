# Fiverr Gig作成時の重要な学び

## タイトル
- 80文字以内、**英数字のみ**（ピリオド・特殊文字不可）
- コピペすると**不可視のUnicode文字**が混入してエラーになることがある → 手打ち推奨
- 最初に保存したタイトルが**永久URL**になる（後から変更してもURLは変わらない）
- URL最適化: 最初は短いキーワード重視タイトルで保存 → 公開後に長い魅力的タイトルに変更

## 検索タグ
- 最大5個、各タグ**20文字以内**
- 文字数は半角スペースも含む

## Description（説明文）
- **1200文字以内**
- 同じ単語の繰り返しでエラー（「excessive use of terms: ○○」）
  - 例: "reviews"を7-8回使用 → "opinions", "comments", "feedback"等に分散
- 文字数カウント: `wc -m`の結果とFiverr側の計算に**5〜10文字の差**がある（Fiverrの方が多い）
  - 安全策: 自分のカウントで1190文字以下を目指す

## プロフィール
- 自己紹介文: **600文字以内**
- プロフィール写真: 最低400×400px正方形

## Extra Services（追加サービス）
- 「I will」はFiverrが自動付与するので、自分で書かない（重複エラーになる）

## Gallery
- サムネイル: **1280×769px**（比率 約5:3）
- PDF: **2MB以内**（ilovepdf.comで圧縮）

## 価格設定
- デコイ効果: Basicを意図的に簡素にしてStandard購入を促す
- 英語で会話できない場合: Premium「ビデオ通話コンサル」は外す

## W-9フォーム
- 日本在住者: 「I am NOT a U.S. person」を選択

## Briefs（旧Buyer Requests）
- 新規セラーにはすぐ表示されない（数日〜数週間待ち）
- 表示されるまではUpworkの方が積極的に営業できる

## その他
- 毎日ログインでアクティブ維持（検索順位に影響）
- スマホアプリインストールでオンライン状態維持
- 最初の数件は値下げしてでも★5レビューを集めるのが重要
