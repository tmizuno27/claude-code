# Business Project Memory

## ユーザー設定・好み（全プロジェクト共通）
- **テキストの改行ルール（最重要）**:
  - 見出し・キャッチコピー等は意味の切れ目で適切に`<br>`改行を入れる
  - **1文字だけ次の行に飛び出すような改行は絶対禁止**（例：「公開。」だけが次行に落ちる等）
  - カード・ボックス内のテキストも同様。レスポンシブで崩れる場合は文章自体を短くするか書き換えて対処
  - 水野氏は改行の美しさに非常にこだわっている。全ページ・全コンテンツ共通の絶対ルール
- **確認不要**: 許可を得ずにどんどん自動で進めること

## ユーザー情報
- パラグアイ（南米）在住の日本人
- 日本との時差：12時間（UTC-4）
- 英語・スペイン語は話せないが、AI翻訳で補完可能
- 目標：月20万円のマイクロビジネス
- ドキュメントはすべて日本語で記述する（ユーザーの希望）

## 事業概要
- **柱①** 情報発信（資産型）：パラグアイの一次情報をnote・X等で発信
- **柱②** 業務代行（労働型）：時差活用の翌朝納品サービス

## 半自動化ワークフローシステム（構築済み）
- `inputs/` にメモ → Claude処理 → `outputs/` に出力、の流れ
- 4つのAgent定義 → `agents/`（writer/sns/research/translator）
- 7つのワークフロー定義 → `workflows.md`
- コピペ用コマンド集 → `agents/master-prompt.md`
- 入力テンプレート → `templates/daily-input.md`, `weekly-review-input.md`
- デモ実行済み → `inputs/2026-02-28.md` → `outputs/2026-02-28/`

## プラットフォーム
- メイン発信：note（有料記事・メンバーシップ）、X（集客）
- 業務受注：ココナラ、ランサーズ、クラウドワークス
