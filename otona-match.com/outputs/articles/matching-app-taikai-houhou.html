「マッチングアプリを退会したいのに、やり方がわからない」「アプリを消しただけで退会できたと思っていた」――こんな経験はありませんか？

実は、マッチングアプリはアンインストールしただけでは退会にならず、アカウントもプロフィールもそのまま残り続けます。しかも有料プランに加入していた場合、アプリを消しても課金が止まらないケースがほとんどです。

筆者（南米おやじ）もパラグアイに移住する前、日本で複数のマッチングアプリを使っていましたが、退会手続きをせずにアプリだけ削除してしまい、翌月も課金されていたという苦い経験があります。海外にいると日本のサブスク管理は特に見落としやすいので、正しい退会手順を知っておくことは本当に大切です。

この記事では、主要10アプリの退会・アカウント削除手順を完全網羅。有料プランの解約方法から、退会前にやるべきこと、退会後の再登録ルールまで、すべてまとめました。

マッチングアプリの退会で絶対に知っておくべき3つの基本

アプリ削除≠退会｜アンインストールだけではダメな理由

最も多い勘違いが「アプリを消したら退会できた」と思い込むことです。ほぼすべてのマッチングアプリで、アンインストールしてもアカウントは存続し、プロフィール情報・写真・メッセージ履歴はサーバーに残ったままになります。

つまり、あなたのプロフィールは他のユーザーから引き続き閲覧可能な状態です。個人情報の観点からも、使わなくなったアプリは必ず正式な退会手続きを踏みましょう。

有料プラン解約と退会は別の手続き

多くのアプリでは、有料会員のまま退会手続きができない仕様になっています。退会の前に、まず有料プランの自動更新（サブスクリプション）を停止する必要があります。

決済方法によって解約の場所が異なるのもポイントです。

決済方法 解約場所 手順
Apple ID決済 iPhoneの「設定」アプリ 設定 → Apple ID → サブスクリプション → 該当アプリ → キャンセル
Google Play決済 Google Playストアアプリ Playストア → お支払いと定期購入 → 定期購入 → 該当アプリ → 解約
クレジットカード決済 アプリ内の設定画面 各アプリの「有料会員」設定から直接解約

退会後の再登録には制限期間がある

退会すると一定期間は再登録ができません。再登録の制限期間はアプリによって異なり、最短で即日、最長で30日程度です。「すぐ戻れる」と思って安易に退会すると、再開したいときに待たされる可能性があるので注意しましょう。

各アプリの退会手順を詳しく解説

Pairs（ペアーズ）の退会方法

国内最大級のマッチングアプリであるペアーズは、退会手順がやや分かりにくいことで知られています。

ステップ1：有料プランを解約する

有料会員の場合、まず自動更新を停止します。Apple ID決済ならiPhoneの「設定」→「Apple ID」→「サブスクリプション」からペアーズを選んでキャンセル。Google Play決済ならPlayストアの「定期購入」から解約。クレジットカード決済ならペアーズアプリ内で直接解約できます。

ステップ2：退会手続きを行う

  1. ペアーズにログイン
  2. 画面右下の「その他」をタップ
  3. 「ヘルプ・お問い合わせ」をタップ
  4. 画面最下部の「お問い合わせ」をタップ
  5. お問い合わせカテゴリで「退会」を選択
  6. 退会理由を選択して退会を確定

退会後30日間は再登録不可。退会するとプロフィール・メッセージ履歴・Pairsポイント・いいね数などすべてのデータが完全に削除されます。

with（ウィズ）の退会方法

心理テストで相性診断ができるwithは、退会までに5ステップが必要です。

退会手順

  1. 有料プランを解約して無料会員に戻す（Apple ID / Google Play / クレジットカードの各決済先で停止）
  2. withアプリを開き「マイページ」をタップ
  3. 「各種設定」→「退会する」をタップ
  4. 退会理由のアンケートに回答
  5. 「退会する」ボタンをタップして完了

withは退会後14日間で再登録が可能になります。ペアーズより短い制限期間なので、お試し感覚で使いやすいアプリです。

タップルの退会方法

退会手順

  1. 有料プラン利用中の場合、自動更新を停止する
  2. アプリ右下の「マイページ」をタップ
  3. 右上の歯車アイコン（設定）をタップ
  4. 設定画面を下にスクロールし「お問い合わせ・ヘルプ」をタップ
  5. 「退会」の項目を選択
  6. 退会理由にチェックを入れ、「退会する」をタップ

タップルではアプリ内の「設定」画面の下部に退会メニューが配置されていますが、やや見つけにくい場所にあります。見落とさないようスクロールしてください。

Omiai（オミアイ）の退会方法

退会手順

  1. 有料プランを決済方法に応じて解約する
  2. Omiaiにログインし「マイページ」へ移動
  3. 「各種設定」をタップ
  4. 画面を下にスクロールし「退会手続き」をタップ
  5. 退会理由を選択して「手続きに進む」をタップ
  6. 確認画面で「退会する」をタップして完了

Omiaiは退会後すぐに再登録が可能です。ただし退会前のデータは一切引き継げないので、やり直しになる点は覚悟しましょう。

マリッシュの退会方法

再婚・シンパパ・シンママ向けのマリッシュは、比較的わかりやすいUI設計です。

退会手順

  1. 有料プランを解約する
  2. アプリの「マイページ」をタップ
  3. 「通知・各種設定」をタップ
  4. 「退会について」をタップ
  5. 退会理由を選択して退会を確定

マリッシュも退会後すぐに再登録可能ですが、以前のマッチングやメッセージ履歴はすべて消えます。

ゼクシィ縁結びの退会方法

退会手順

  1. 有料プランを解約する
  2. アプリの設定画面を開く
  3. 「退会する」をタップ
  4. 退会理由を選択して確定

ゼクシィ縁結びは退会と有料プランの解約が完全に別の手続きになっているため、解約を忘れると退会後も課金が続く可能性があります。必ず先にサブスクリプションを停止してから退会してください。

Bumble（バンブル）の退会方法

退会手順

  1. アプリを開き、プロフィールアイコンをタップ
  2. 「設定」をタップ
  3. 画面下部の「アカウントを削除」をタップ
  4. 確認画面で削除を確定

Bumbleは有料会員のままでもアカウント削除が可能な数少ないアプリです。ただし、アカウントを削除してもサブスクリプションは自動停止されないので、Apple IDやGoogle Playからの課金停止は別途行ってください。

youbride（ユーブライド）の退会方法

退会手順

  1. 有料プランを解約する
  2. アプリの設定画面を開く
  3. 「休会・退会」をタップ
  4. 「退会する」を選択して退会理由を入力
  5. 退会を確定

youbrideには「休会」機能もあり、データを残したまま一時的に活動を休止できます。完全にやめるつもりがない場合は、退会ではなく休会を選ぶのも手です。

  4. 「退会する」をタップ

退会前にやるべき5つのチェックリスト

退会ボタンを押す前に、以下の項目を必ず確認しましょう。

1. 有料プランの自動更新を停止したか

最も重要な確認事項です。アプリ内で退会しても、Apple IDやGoogle Playの定期購入が残っていると課金が続きます。退会手続きとは別に、スマホの「設定」からサブスクリプションを確認してください。

2. 有料期間の残日数を確認したか

多くのアプリでは有料プランの残り期間があっても退会すると残日数分の返金はありません。有料期間が終了するタイミングで退会するのがもっとも無駄のない方法です。

3. やり取り中の相手に一言伝えたか

マッチング中やメッセージのやり取り中に突然退会すると、相手には「退会済み」と表示されます。真剣に活動している相手に対してはマナーとして一言伝えてから退会するのが良いでしょう。詳しくは「マッチングアプリのやめどき」もご覧ください。

4. 連絡先を交換しているか

退会するとアプリ内のメッセージはすべて消えます。今後も連絡を取りたい相手とはLINEなどの外部連絡先を事前に交換しておきましょう。

5. スクリーンショットを保存したか

プロフィール文やもらった「いいね」数など、次にアプリを使う際の参考にしたいデータがあればスクリーンショットを撮っておくと便利です。

退会がうまくできないときのトラブル対処法

「退会ボタンが見つからない」場合

多くのアプリでは退会メニューを意図的に目立たない場所に配置しています。基本的には「マイページ」→「設定」→「お問い合わせ」や「ヘルプ」の中に隠れていることが多いです。アプリ内の検索機能で「退会」と検索すると直接ヘルプページにたどり着ける場合もあります。

「有料会員のため退会できません」と表示される場合

有料プランの解約が完了していない状態です。アプリ内ではなく、Apple IDまたはGoogle Playの定期購入画面から解約手続きを行ってください。クレジットカード決済の場合はアプリ内の設定から解約できます。

解約したのに課金が止まらない場合

解約手続き後も、次回更新日までは有料サービスが利用可能な場合があります。それを過ぎても課金が止まらない場合は、Apple（App Store）またはGoogle（Google Play）に直接問い合わせてください。

退会と休会の違い｜一時停止という選択肢

アプリによっては「退会」の代わりに「休会」「一時停止」機能が用意されています。

項目 退会 休会・一時停止
アカウント 完全削除 保持される
プロフィール 消える 非表示になるが残る
マッチング履歴 全消去 保持される
再開時 新規登録からやり直し そのまま再開
課金 停止 停止（アプリによる）

「しばらく休みたいだけ」「また使うかもしれない」という場合は、退会ではなく休会を選ぶことでデータを残したまま活動を休止できます。特にマッチング数やいいね数が多い方は、退会するとそれらが全てリセットされるため、休会のほうがメリットが大きいでしょう。

マッチングアプリに疲れを感じている方は「マッチングアプリ疲れの対処法」も参考にしてみてください。

退会後に再登録するときの注意点

再登録の制限期間を確認する

前述の通り、ペアーズは30日間、withは14日間の再登録制限があります。一方でOmiaiやマリッシュは退会直後でも再登録可能です。「少し休んでまた始めたい」なら、退会ではなく休会を選ぶか、制限期間が短いアプリを選ぶと柔軟に対応できます。

再登録は完全に新規扱い

退会後に再登録すると、プロフィール・マッチング履歴・いいね数などは一切引き継がれません。すべて最初からやり直しです。ただしこれは「新規会員」として表示されるため、マッチング率が一時的に上がるという副次的なメリットもあります。

同じ電話番号・メールアドレスで再登録できるか

ほとんどのアプリでは、退会時に使用していた電話番号やメールアドレスで再登録が可能です。ただしFacebook連携で登録した場合は、同じFacebookアカウントでの再登録に制限がかかるケースもあるので事前に確認してください。

パラグアイ在住者が学んだ退会管理のコツ

筆者は現在パラグアイのアスンシオンに在住していますが、海外に住んでいると日本のサブスクリプション管理が疎かになりがちです。実際に私が経験した失敗と、そこから得た教訓を共有します。

日本のアプリを海外で解約する際の注意点

Apple IDやGoogle Playの決済は海外からでも管理可能ですが、アプリによってはVPN経由でないとアクセスできない場合があります。また、クレジットカード決済の場合、日本のカードの有効期限切れに気づかず「解約したつもりが更新されていた」ケースもあります。

サブスク管理アプリを使う

私は複数のマッチングアプリを使っていた時期に、どのアプリが有料でどれが無料かわからなくなった経験があります。iPhoneの「設定」→「サブスクリプション」画面を定期的にチェックする習慣をつけるか、サブスク管理アプリを活用することをおすすめします。

退会後の次のステップ｜自分に合ったアプリの選び直し

退会の理由が「出会えなかった」「合わなかった」という場合、アプリを変えることで状況が改善することも多いです。

30代・40代の方には、年齢層や目的に合ったアプリ選びが重要です。「マッチングアプリおすすめランキング2026」で最新の人気アプリを比較していますので、次に使うアプリ選びの参考にしてください。

よくある質問（FAQ）

Q. アプリをアンインストールすれば退会できますか？

いいえ、できません。アプリを削除してもアカウントは残り、プロフィールも公開されたままです。必ずアプリ内から退会手続きを行ってください。

Q. 有料期間の途中で退会した場合、返金されますか？

ほとんどのアプリでは途中退会による返金はありません。有料期間が終了するタイミングで退会するのが最も経済的です。

Q. 退会したら相手にどう表示されますか？

多くのアプリでは「退会済み」または「このユーザーは退会しました」と表示されます。プロフィール写真やメッセージは相手側からも見えなくなります。

Q. 退会と「ブロック」は違いますか？

まったく別の機能です。ブロックは特定の相手に自分のプロフィールを見せなくする機能で、アカウント自体は存続します。退会はアカウントそのものを削除する手続きです。

Q. 退会後、同じアプリにすぐ再登録できますか？

アプリによります。Omiaiやマリッシュは即日再登録可能ですが、ペアーズは30日間、withは14日間の制限があります。本記事の比較表を参考にしてください。

まとめ：正しい退会手順で安心してアプリを卒業しよう

マッチングアプリの退会で最も重要なポイントは次の3つです。

  1. アプリ削除だけでは退会にならない――必ずアプリ内で退会手続きを行うこと
  2. 有料プランは先に解約する――Apple ID・Google Play・クレジットカードの各決済元でサブスクリプションを停止すること
  3. 退会と休会を使い分ける――また使うかもしれないなら休会でデータを残す選択肢もある

退会手続きは面倒に感じるかもしれませんが、個人情報保護と無駄な課金防止のために必ず正しい手順で行いましょう。

新しいアプリを試してみたい方は「2026年版マッチングアプリランキング」をチェックしてみてください。自分に合ったアプリとの出会いが、素敵なパートナーとの出会いにつながるはずです。
