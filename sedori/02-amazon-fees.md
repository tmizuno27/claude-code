# Amazon 手数料ガイド
## せどりで必ず理解すべき4つの手数料

作成日: 2026年3月6日

---

## ざっくり計算ルール

**販売価格の約30〜35%が手数料で消える**

→ 販売価格3,000円なら、手数料は約900〜1,050円
→ 仕入れ値が2,000円以下なら利益が出る可能性あり
→ さらにポイント還元を加味して最終判断

---

## ① 販売手数料（カテゴリーごとに異なる）

売上金額に対して一定のパーセンテージが引かれる。

**計算式**: 販売価格 × 手数料率 × 1.1（消費税）

### カテゴリー別手数料率

| カテゴリー | 手数料率 |
|-----------|---------|
| 本 | 15% |
| CD・DVD | 15% |
| TVゲーム | 15% |
| 家電・カメラ | 8% |
| パソコン・周辺機器 | 8% |
| おもちゃ | 10% |
| ホーム・キッチン | 15% |
| ドラッグストア・ビューティー | 8〜10% |
| 食品・飲料 | 8〜10% |
| 服・シューズ | 12〜15% |
| スポーツ・アウトドア | 10% |

**2026年4月1日改定**: 全カテゴリーで+0.4%の引き上げ予定

### 計算例

3,000円の家電 → 3,000 × 8% × 1.1 = **264円**

## ② FBA配送代行手数料（サイズと重量で決まる）

Amazonが梱包・発送を代行する費用。

| サイズ区分 | 目安条件 | 手数料（税込） |
|-----------|---------|--------------|
| 小型 | 25×18×2cm以下、250g以下 | 約288円 |
| 標準60 | 60サイズ相当、2kg以下 | 約434円 |
| 標準80 | 80サイズ相当、5kg以下 | 約514円 |
| 標準100 | 100サイズ相当、9kg以下 | 約603円 |
| 大型140 | 140サイズ相当 | 約895円〜 |

**ポイント**: 小さくて軽い商品ほど手数料が安い → 初心者は小型・軽量の商品から始める

---

## ③ 在庫保管手数料（倉庫に置いてる間ずっとかかる）

FBA倉庫に商品を預けている限り、日割りで発生。

- **1〜9月**: 5.160円 ×（商品体積 ÷ 1,000cm³）× 保管日数 ÷ 当月日数
- **10〜12月**: 9.170円 ×（同上）← 年末は約1.8倍

実感: 普通サイズの商品1個で **月10〜50円程度**

**注意**: 3ヶ月以上売れ残ると「長期在庫保管手数料」が別途発生 → 回転率が命

---

## ④ その他の手数料

| 手数料 | 内容 | 金額 |
|--------|------|------|
| 大口出品月額 | 毎月固定 | 5,390円（税込） |
| カテゴリー成約料 | 本・CD・DVDのみ | 80〜140円/個 |
| 返品手数料 | 購入者が返品した場合 | 配送代行手数料相当額 |

## 利益計算の具体例

### 例: ドラッグストアカテゴリーの日用品

```
販売価格:           2,500円
━━━━━━━━━━━━━━━━━━━━━
仕入れ値（楽天）:    1,500円（ポイント還元前）
楽天ポイント還元15%:  −225円
実質仕入れ値:        1,275円

販売手数料（10%×1.1）: −275円
FBA配送代行（小型）:   −288円
保管手数料（1ヶ月）:   −15円
FBA納品代行:          −100円
━━━━━━━━━━━━━━━━━━━━━
純利益:              547円
利益率:              21.9%
ROI（投資利益率）:    42.9%
```
