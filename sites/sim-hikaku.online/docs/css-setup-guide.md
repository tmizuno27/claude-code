# CSS適用手順 — sim-hikaku.online

## Step 1: カスタムCSSを適用

### 方法A: Cocoon子テーマのstyle.css（推奨）

1. WordPress管理画面 → **外観 → テーマファイルエディター**
2. 「Cocoon Child」のstyle.cssを選択
3. `theme/css/sim-global.css` の内容をすべてコピー＆ペースト
4. 「ファイルを更新」をクリック

### 方法B: カスタムHTMLウィジェット経由

1. **外観 → ウィジェット** → 「コンテンツ上部」に「カスタムHTML」を追加
2. 以下のコードを貼り付け:
```html
<style>
/* ここにsim-global.cssの内容をすべて貼り付け */
</style>
```

## Step 2: トップページを設定

1. **固定ページ → 新規追加**
2. タイトル: 「ホーム」、スラッグ: `home`
3. **「カスタムHTML」ブロック**を選択
4. `theme/front-page.html` の内容を貼り付け
5. 公開

6. **設定 → 表示設定**
7. 「ホームページの表示」→ 「固定ページ」を選択
8. ホームページ: 「ホーム」を選択
9. 保存

## Step 3: 固定ページを作成

以下の4ページをWordPressで作成:

| ページ名 | スラッグ | テンプレートファイル |
|---------|---------|-------------------|
| 運営者情報 | about | `templates/about.html` |
| プライバシーポリシー | privacy-policy | `templates/privacy-policy.html` |
| 免責事項 | disclaimer | `templates/disclaimer.html` |
| お問い合わせ | contact | Contact Form 7プラグインで作成 |

### 手順:
1. 固定ページ → 新規追加
2. タイトルとスラッグを設定
3. 「カスタムHTML」ブロックを選択
4. 各テンプレートの内容を貼り付け
5. 公開

## Step 4: メニュー設定

1. **外観 → メニュー**
2. 新しいメニュー「ヘッダーメニュー」を作成
3. 以下の項目を追加:
   - 用途別おすすめ（カテゴリ: yoto-betsu）
   - 料金比較（カテゴリ: ryokin-hikaku）
   - レビュー（カテゴリ: review）
   - 乗り換え（カテゴリ: norikae）
   - 海外SIM（カテゴリ: kaigai-sim）
4. メニューの位置: 「ヘッダーメニュー」にチェック
5. 保存

## Step 5: フッターメニュー

1. 新しいメニュー「フッターメニュー」を作成
2. 以下の固定ページを追加:
   - 運営者情報
   - プライバシーポリシー
   - 免責事項
   - お問い合わせ
3. メニューの位置: 「フッターメニュー」にチェック
4. 保存

## Step 6: Cocoon設定の確認

1. **Cocoon設定 → スキン** → なし（カスタムCSSを使うため）
2. **Cocoon設定 → ヘッダー** → サイトロゴ画像があれば設定
3. **Cocoon設定 → SEO** → メタタグ出力OFF（Rank Mathを使用）
4. **Cocoon設定 → SNSシェア** → 非表示に設定（CSSでも非表示にしているが念のため）
