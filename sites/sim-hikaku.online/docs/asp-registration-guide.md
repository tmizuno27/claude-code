## 登録すべきASP（優先順）

### 1. アクセストレード（最優先）
- **URL**: https://www.accesstrade.ne.jp/
- **理由**: 格安SIM案件が最も豊富。SIM特集ページあり
- **報酬**: 1,000円から振込可能、振込手数料無料
- **特徴**: 提携ランク制度で成果に応じて報酬単価UP

**登録手順**:
1. https://www.accesstrade.ne.jp/ にアクセス
2. 「パートナー登録（無料）」をクリック
3. メールアドレス登録 → 確認メール → 本登録
4. サイト情報登録:
   - サイト名: SIM比較オンライン
   - URL: https://sim-hikaku.online
   - カテゴリ: 通信・インターネット
   - サイト説明: 格安SIM・格安スマホを用途別に比較するサイト
5. 審査通過後（通常1-3営業日）→ ログイン

**提携申請する案件**:
- UQモバイル
- 楽天モバイル
- mineo
- IIJmio
- NUROモバイル
- BIGLOBEモバイル
- 日本通信SIM
- HISモバイル

> 注意: 提携はサイトごとに申請が必要。sim-hikaku.onlineで別途提携申請すること

### 2. バリューコマース
- **URL**: https://www.valuecommerce.ne.jp/
- **理由**: UQモバイル高単価案件、SIMフリー特集あり
- **特徴**: Yahoo!ショッピング連携、LinkSwitch機能

**登録手順**:
1. https://www.valuecommerce.ne.jp/ にアクセス
2. 「無料会員登録」
3. サイト情報・報酬振込先を入力
4. 審査（通常数日〜1週間）

**提携申請する案件**:
- UQモバイル（6,480円/件）
- LINEモバイル（3,726円/件）
- mineo（2,700円/件）
- ワイモバイル

### 3. A8.net
- **URL**: https://www.a8.net/
- **理由**: 最大手ASP、楽天モバイル案件あり
- **特徴**: 審査なしで即登録可能、セルフバックあり

**登録手順**:
1. https://www.a8.net/ にアクセス
2. 「無料登録でアフィリエイトを始める」
3. メールアドレス → 基本情報 → サイト情報 → 即完了（審査なし）

**提携申請する案件**:
- 楽天モバイル（通話SIM: 1,851円、データSIM: 925円）
- TONEモバイル
- エキサイトモバイル
- ConoHa WING（サーバー紹介用: 5,000-12,000円/件）

### 4. もしもアフィリエイト
- **URL**: https://af.moshimo.com/
- **理由**: W報酬制度（通常報酬+12%ボーナス）、初心者に優しい
- **特徴**: Amazon・楽天の商品紹介も可能

**登録手順**:
1. https://af.moshimo.com/ にアクセス
2. 「会員登録する（無料）」
3. メール → 基本情報 → サイト登録 → 審査

**提携申請する案件**:
- 格安SIM関連案件
- ConoHa WING
- スマホ端末関連

## ASP登録時の注意点

### 審査通過のコツ
- **最低5記事は公開してから申請**する（審査落ちリスクを減らす）
- プライバシーポリシー・運営者情報ページを必ず設置
- 記事は1,500字以上の質の高い内容にする
- アダルト・違法コンテンツがないことを確認

### 推奨フロー
1. WordPress設置 → 固定ページ（about, privacy-policy）作成
2. 初期記事5本を公開
3. A8.net 登録（審査なし・即日）
4. アクセストレード・バリューコマース登録（審査あり・数日）
5. もしもアフィリエイト登録
6. 各ASPで格安SIM案件に提携申請
7. 承認後、記事にアフィリエイトリンクを設置

### アフィリエイトリンク管理

`config/affiliate-links.json` で一元管理:

```json
{
  "uq-mobile": {
    "name": "UQモバイル",
    "asp": "バリューコマース",
    "url": "（提携後に取得したリンク）",
    "commission": 6480,
    "status": "pending"
  },
  "rakuten-mobile": {
    "name": "楽天モバイル",
    "asp": "A8.net",
    "url": "（提携後に取得したリンク）",
    "commission": 1851,
    "status": "pending"
  }
}
```
