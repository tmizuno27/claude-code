## 6. ✅ アクションアイテム（今日やること）

### 🔴 最優先：計測環境の整備

#### ① GA4の即時設定
```
手順:
1. Google Analytics (analytics.google.com) でプロパティ作成
2. GA4 計測IDを取得（G-XXXXXXXXXX形式）
3. WordPressに設定
   → 推奨プラグイン:「Site Kit by Google」または「GTM経由」
4. リアルタイムレポートで計測確認
```
> ⏱ 所要時間: 30〜60分 / 優先度: 🔴 最高

#### ② Search Consoleのインデックス促進
```
手順:
1. Search Console（search.google.com/search-console）にログイン
2. 「URL検査」ツールで主要記事のURLを1件ずつ入力
3. 「インデックス登録をリクエスト」を実行
4. XMLサイトマップが送信済みか確認
   → WordPress: Yoast SEO / All in One SEO で自動生成されたものを送信
```
> ⏱ 所要時間: 20〜30分 / 優先度: 🔴 最高
