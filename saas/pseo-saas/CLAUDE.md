# pSEO AIツール比較サイト — プロジェクトガイド

## 概要
AIツールの「[A] vs [B]」比較ページを数千〜数万ページ生成し、Google検索流入→アフィリエイト+広告で収益化するプログラマティックSEOサイト。

## なぜAIツール比較か（2026-03-15 最終検証済み）
- **競合が極めて弱い**: 「jasper vs chatgpt」でBlogspotが上位表示。権威構造が未確立
- **市場成長率180%+**: 毎月新ツールが生まれる→ページが自然に増え続ける
- **ページ生成ポテンシャル**: 300〜500ツール → 44,000+ページ
- **検索ボリューム**: 「ChatGPT vs Claude」で月15K〜25K
- メールマーケティング比較は競合が強く断念（EmailToolTester $3.8M ARR、Google HCUリスク）

## 事業モデル
- **ターゲット言語**: 英語（グローバル市場）
- **ニッチ**: AIツール全般（ライティング、画像、コーディング、動画、音声、SEO等）
- **収益化**: アフィリエイト（リカーリング報酬）+ 広告（Mediavine/AdSense）
- **集客**: Google検索のみ（営業ゼロ）

## コンテンツ品質ルール（Google HCU対策）
- テンプレコピペの量産は**絶対禁止**
- 各ページに独自の分析・評価スコアリングを含める
- 「実際に使ってみた」フレーミングで体験ベースの文体
- 最低800語のユニークコンテンツ
- 独自スコアリングシステム（価格、使いやすさ、機能、サポート等）
- 段階的公開: 月500〜1,000ページ

## 技術スタック
- **フレームワーク**: Next.js (App Router, SSG)
- **ホスティング**: Vercel
- **データ**: JSON → SQLite（ツール情報DB）
- **コンテンツ生成**: Claude API（質の高い比較文章）
- **監視**: Google Search Console API + GA4
- **デプロイ**: GitHub → Vercel自動デプロイ

## ディレクトリ構成
```
pseo-saas/

├── scripts/             ← データ収集・ページ生成・監視スクリプト
```
