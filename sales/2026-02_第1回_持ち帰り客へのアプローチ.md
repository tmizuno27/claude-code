## 2. メインテーマ: 持ち帰りを希望するお客様への有効なアプローチ

**前提:**
- 持ち帰らせない（無理な即決）ではなく、**持ち帰っても制約になるアプローチ**を考える
- 特商法の観点からも無理な即決は避ける

---

### 伊藤さんのアプローチ: 「合格」ではなく「内定」にする

- 合格 → ゴール感・ウイニングラン感が出てしまい、そこで満足して終わる可能性がある
- **内定止まり**にして、行動を完了した方を「正式合格」とする

**伝え方のロジック:**
- 「口頭での意思表示と実際に行動できるかは別物。私たちは行動まで完了した方を正式合格としています」
- ご褒美を少し後ろに持っていくことで、行動を促す

**成さんの解説:**
- 入るかどうかがゴールではなく、**理想の3年後を叶えるところがゴール**
- 3年後に向けてしっかり動ける人に入っていただきたいという前提
- 行動の1歩目が1番大事な1歩
- 合格を出した後に持ち帰りになった場合 → 「一旦内定という形にさせていただきます。ご決断いただければ改めて合格とさせていただきます」
- 注意: 「4つの確認ができてないから内定」ではなく、「3年後に向けた1歩目を踏み出していただくための内定」というニュアンス

---

### ゆりさんのアプローチ

1. **持ち帰りたい理由を深掘り** — 何がネックなのか明確にする
2. **理想の未来を出して本当に叶えたいか確認** → 環境を変えなきゃいけないことを言語化してもらう
3. **例え話を用意して伝える**
4. **2回目のZoom提案をその場で設定** — 日程を握っておく

---

### 竹本さんのアプローチ

1. **次回のアポイントを設定する**
2. **パートナーへの相談時の対処方法を一緒に考える**
   - 「大体反対ありますよね。その時にどう話しますか？」を明確にしておく
3. **持ち帰りそうだと感じたら最終金額は伝えない** → 99万でクロージングしてから
4. **「推薦合格なので、こちらも畑さんや岡さんに相談してきます」** → 次回への期待感を作る
5. **やった方がいいができていないこと**: 面談後のお礼＋「あなたには期待しています」を伝える

---

### 弘樹さんのアプローチ

- **「自分を変えたい」「成長したい」という思いに刺さるアプローチ**
- 持ち帰った後にも印象に残るように
- **面談担当者の「顔」を印象に残す** — 振り返った時にその人の顔が浮かぶように
- プロダクトの魅力 + 自分自身の印象を残す

---

### 金子さんのアプローチ: 人間の性善説を前提にした「顔を刻み込む」戦略

**前提: 人間は性善説である**

- 人は「自分を信じてくれた人」「恩を感じた人」を裏切ることが1番きつい
- 最後の最後、感情で決める。その感情の因数分解 → **お世話になった人を裏切りたくない**

**具体的なトーク（パートナーの反対への対処）:**

1. 「必ず反対してきます。なぜなら、あなたのことをめちゃくちゃ愛してるから、心配してるからです」
2. 「その人の判断基準は**あなたの過去**を見ています」
3. 「きっとこんな言葉が来るでしょう — "だってこれまで色々やっても結局うまくいかなかったじゃん"」
4. 「それに対して — 私は今日話してあなたの意思と可能性にかけます。確信して背中を押します」

**過去と未来の対比構造を作る:**
- 過去を引っ張ってくる周りの人 vs 未来の最大の応援者である自分
- 「今まで誰も信じてくれなかった私の思いを信じてくれた金子さんだけは裏切れない」
- **このパワーが198万円を超えていた**

### 成さんのアプローチ: 包括的な持ち帰り対応

#### ステップ1: 持ち帰りの受容とリフレーミング

- 持ち帰りになったら → **「大歓迎です。むしろ持ち帰ってください」**
- お客様の「ひっくり返されるぞ」という防御を解く
- 「全然確認だけなんですけど、やりたいお気持ちってぶっちゃけありますか？なければはっきり言ってお終いにしちゃって全然大丈夫です」
  - 「なければはっきり言ってOK」と言うと → 「いやいや、あるんですよ」と引き出せる
  - 逆に「やりたい気持ちありますよね？」と押すと → 「まぁでもちょっと…」と逃げる

#### ステップ2: 悩みの種類を分ける

- 「やるかどうかを悩みたいのか」vs「やる決断を固めるための時間が欲しいのか」をセパレート

#### ステップ3: 熟成のリフレーミング

- 「熟成ですね」→ 熟成焼肉のメタファー
- 「今すぐもいいけど、熟成させてより強い決断ができるのもリッチなこと」
- 「この持ち帰った時間をリッチに熟成させるためには何があったらいいですか？」
- セールスマンではなく、**土手で夕焼けを見ながら聞いている感じ**で質問する

#### ステップ4: 不安への対処（先出し）

- 「この後に心で起きるのは"大丈夫かな？本当にやっていいのかな？怪しくない？"です。普通です」
- 「その気持ちは、本気で考えてる証拠」
- 「覚悟は日常の延長線上では起こらない。新たなことをやろうとする時に初めて"大丈夫？"と思う」
- **「やりたい理由」か「今を変えたい思い」を振り返って、あるならご決断ください。なければ無理しなくていい」**

#### ステップ5: パートナーの反対への対処

- まず何があったら応援してくれるかを前提に聞く
- 「パートナーに言ったらどう話しそうですか？」「何があったら笑顔で頑張りなよって言ってくれそうですか？」
- **人によっては**: 「大切な誰かの理由でご決断を決めないでください」
  - 理由: 逆の立場だったら？ あなたの大切な人が「あなたを理由にやめます」と言われたら寂しくないですか？
  - 「愛する人だからこそ、自分でご決断することが大事」
  - 入るか入らないかより、**3年後に向かう腹を決められるか**

#### ステップ6: ベイビーステップの提案

- 「買う買わないは一旦置いといて、3年後のために今日明日で何か1つやってみましょうか」
- ネクストベイビーステップを聞く
- セールスマンではなく**ライフコーチ**として最後のコミュニケーションを取る

## 3. 持ち帰りがちなお客様の特徴

### 竹本さんの観察
- ヒアリングで「転職など大きな決断をどう決めてきたか」を聞く
- 「奥さんに相談して決めました」→ 持ち帰り傾向
- 「自分で全部決めてきました」→ 即決傾向

### 弘樹さんの観察
- **明るすぎる人、トントン進みすぎる人** → 意外と最後に持ち帰りがち
- スムーズに行きすぎて自分も「いい感じ」だと思ってしまうが、金額を伝えた後に持ち帰りに
- 自分の印象を残せていない → 持ち帰り後に気持ちが落ちて失注

### 金子さんの観察
- **左脳タイプの人**は持ち帰りやすい（情報を淡々と受け取る）
- **右脳タイプ**はその場で決める傾向
- ただし左脳タイプでも**感情が動いた瞬間**（リズムが崩れる、間がある、沈黙、泣く）があれば即決する
- ロジカルを超えた感情の起伏がその瞬間に生まれたかどうか

### 成さんの補足: 「意思ある楽しさ」を作る
- 持ち帰りやすい**人**だけでなく、持ち帰りやすい**空気**がある
- 楽しくトントン行く → 金額でリアルが出る → 「夢の国のチケット代こんなですか」状態
- **「意思がない楽しさ」より「意思ある楽しさ」を面談中に作る**
- 楽しい → 意思確認 → 楽しい → 意思確認 — このリズムで地に足のついた楽しさを作る
- ビビらせる = 意思を固める。脅すのではなく、しっかりリアルで意思を確認していく

### 家電セールス部隊の立ち上げ（金子さん）

**ビデオセールス vs 家電セールスの違い:**

| | ビデオセールス（現在） | 家電セールス（新規） |
|---|---|---|
| 方法 | Zoom（顔出し・画面共有） | 電話 |
| 対象 | Webセミナー視聴 → 個別面談申込者 | Webセミナー視聴 → **個別面談に申し込まなかった人** |
| 報酬 | 同一 | 同一 |
| トーク | 画面共有しながら説明 | 口頭で説明、メールでGMOサイン送付 |
| 初回着金額 | 高い傾向 | やや低い傾向（分割多め） |

**なぜ家電セールスが有効か:**
- 個別面談に申し込まない理由の1つ: **ITリテラシー**（LINE操作やZoom設定のハードル）
- 電話番号にかかってくるならOKという層が一定数いる
- 卓ロ6の実績: 月間売上の約**20%が家電セールス**から発生
- これを36プロジェクト全体に展開したい
