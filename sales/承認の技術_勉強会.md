# 承認の技術 — 相手が自然とチャレンジしたくなる承認の技術

## 1. 承認とは

- 相手の存在・変化・頑張りという事実に気づき、そのまま言葉にして伝えること
- 褒めるとの違いを気にしすぎず、**相手に届けばOK**
- 参加者の定義：「受け入れる」（竹本さん）、「背景を理解しての寄り添い」（加藤さん）

## 2. 承認がもたらす効果

- **やる気・安心感を引き出す** → 自己開示の促進 → 自己肯定感・効力感UP
- **ラポール（信頼関係）形成**に不可欠
  - 橋のたとえ：山と山の間に橋がかかっていれば伝わる。なければどんなスクリプトも途中で落ちる
  - 同じことを言われても、信頼関係があるかないかで届き方が全く違う
- 「この人から買いたい」「この人の話を聞きたい」「この人が大丈夫だと思うなら自分もチャレンジしたい」と思える関係値を作る

### 営業の3要素
1. **ラポール**（信頼関係）
2. **技術**（スクリプト・スキル）
3. **あり方**（商品を売るのか、お客様の未来を向いているのか）

### 失注理由トップ6
1. お金がない
2. 家族・パートナーに相談したい
3. 他と比べたい
4. タイミングが今じゃない
5. **自分に続けられるか不安** ← 承認でダイレクトにアプローチ可能
6. 必要性を感じない

## 3. 何を承認するか

| 承認ポイント | 例 |
|---|---|
| 成果・結果 | すごいですね、よくこんな結果出せたね |
| 行動・プロセス | よくここまで諦めずにやってきましたね |
| 発言 | ずっとこういう風にしてきた（覚悟・決意） |
| 気づき | その経験から何に気づかれたんですか？ |
| 経験 | これだけ経験を積んできたんですね |
| 感情 | 不安・喜び・怒りなどの感情にアクセス |
| **存在** | こうやってお会いできたこと自体が嬉しい |

### 参加者の実践例
- **加藤さん**: ユーモア承認（背景のセンスを褒めるなど軽い冗談から入り、相手のタイプを見極める）
- **竹本さん**: 50-60代の方が「声を使った仕事をやりたい」と思い続けてきた姿勢への承認
- **伊藤さん**: 覚悟や決意に対しての承認

## 4. どのように承認するか — 言葉編

### 3つの「祭り」に注意（口癖化の落とし穴）

| 祭り | 相手の反応 |
|---|---|
| 「すごいですね」祭り | 別にすごくねえよ |
| 「いいですね」祭り | 別によくねえよ |
| 「素敵ですね/素晴らしいですね」祭り | 別に素敵じゃねえよ |

- コミュニケーションは無意識だと1番楽な方法に流れる
- 口癖を見抜いて意識的にコントロールすることが大事

### 承認のサイズ感 = 漫才のツッコミ

- ボケに対してちょうどいいツッコミでないと客が笑えない
- 承認も**相手が求めてるちょい上**がバランスのいいポイント
- 大きすぎると「やられてるな」と感じさせてしまう

### 語彙を増やしておく

- すごいですね、いいですね、素晴らしいですね（定番）
- 初めて知りました、さすがですね、尊敬します
- かっこいい、頼もしい（男性向け）
- 同じ女性として尊敬します（伊藤さん）
- 無理して出すのではなく、引き出しとして持っておく → 無意識的にパッと出やすくなる

### 加藤さんの「ギャップ承認」テクニック
- 前半でユーモア・ボケを挟み「正直な人」という印象を作る
- その上で本当にいいところを真剣に褒める → ギャップで深く刺さる
- ストレートと変化球の使い分け

## 5. どのように承認するか — 非言語編

### 非言語の承認方法
- **表情**: 目を見開く、口を大きく開けて無音で笑いながら聞く
- **目線**: カメラ目線で承認する（オンラインでは特に重要）
- **頷き**: 大きく頷く
- **ボディアクション**: 拍手、距離感の使い分け（前のめり↔後ろに引く）、天を仰ぐ

### 加藤さんの「3D承認」
- 画面に近づく（前のめり）と後ろに引くを使い分ける
- 「興味あります」を最大限に身体で表現

### メラビアンの法則
- 視覚情報: 55%
- 聴覚情報: 38%
- 言語情報: 7%
- 同じセリフでもトーン・表情で全く違う伝わり方になる

### 「すごいですね」のバリエーション（トーンで変わる）
- 「すごいですね」（普通）
- 「マジすごいっすね」（驚き）
- 「これはすごいですよ」（断定・強調）
- 「いやぁ、すごいっすね」（感嘆）

## 6. 裏技

### 名前を呼ぶ（存在承認に直結）
- 1番簡単かつインパクトがある承認
- 苗字で丁寧に呼ぶ or 下の名前・あだ名で距離を縮める
- 「この時間、リラックスして本気で向き合ってほしいので、呼び名を決めさせてください」
- 自分から先に「僕は○○と呼ばれてます」と開示する

### 頷きを大きくする
- 一瞬で承認力UP、反射的にやりやすい
- タイミングが大事（関係ないところで頷かない）
- 目をつぶって頷く / 開けて頷く でニュアンスが変わる

### カメラ目線
- オンライン相談では相手の顔を見てしまいがちだが、カメラを見ないと「話を聞いてもらえてない」と無意識に感じさせる
- カメラ目線で承認することが重要

### リフレーミング承認
- フレームを変えて捉え直す
- 「頑固なんです」→「自分を持ってるってことですよね」
- 「優柔不断なんです」→「それだけ本気で考えてる証拠じゃないですか」
- 「不安なんです」→「本気だから不安なんですよね。どうでもいいチャレンジには不安は感じない」

## 7. 承認するための準備 — ステートを良くする

### フィジカル + メンタル
- **メンタル**: CSに集中しよう、お客様の未来にしっかりビルドしよう
- **フィジカル**: 疲れてないか、眠くないか、肩が凝ってないか
- ステートが落ちると無意識的に承認の質が下がる

### ミラーニューロン（鏡の神経）
- こちらの状態は相手に伝染する（コロナやインフルより映る）
- 「この人買わないだろうな」→ 相手に伝わる
- 「これ決まったらあといくらだ」→ 相手に伝わる
- クロージング時に商品のことを考えると相手が違和感を感じて後ろに引く
- **相手の未来を考え続けること**が大事

## 8. 承認がうまくなるには

### 学習方法
1. **気持ちのいい承認を思い出す**: 自分が受けてきた中で良かった承認を振り返る
2. **承認が上手い人を観察する**: なぜうまいのかを言語化する
3. **逆パターンも使える**: 「こう言われたらもっと頑張ろうと思えたな」を考える
4. **自分の相談動画を見直す**: 承認が入った瞬間・外した瞬間を見抜く

### 最終的には「練習」あるのみ
- 承認はほぼ**反射**で行われる → 無意識レベルまで落とし込む必要がある
- 知らない → 知ってるけどできない → 意識したらできる → **意識しなくてもできる**
- 意識して繰り返し練習するしかない

## 9. 金子さんの総括

### ポイント1: 不安からエフィカシーを上げる
- 不安を受け止めつつ、その瞬間に0.1mmでもエフィカシーを上げる
- 「なんで不安なんですか？」→ 自己分析ができている → 「分析力が高い＝あなたの武器」
- ピンチをチャンスに変える

### ポイント2: 感情を大事に扱う
- 人が力を発揮できる時 = **自分を大切に扱ってもらえたと感じた時**
- 安心・安全な場だから力が出せる
- **ネガティブな感情を排除しない** — ポジティブもネガティブも両方抱きしめる（陰と陽の統合）
- 不安という感情をどれだけ大事に扱えるかが勝負の分かれ目
- 感情を大事に扱ってくれるからこそ → 信頼 → パワー発揮 → エフィカシーUP → 決断できる

## まとめ: 承認の全体像

```
【何を承認するか】
  成果 / 行動・プロセス / 発言 / 気づき / 経験 / 感情 / 存在

【どのように承認するか】
  言葉編: 口癖祭りを避ける / サイズ感 / 語彙を増やす / ギャップ承認
  非言語編: 表情 / 目線 / 頷き / ボディアクション / 3D承認

【裏技】
  名前を呼ぶ / 頷きを大きく / カメラ目線 / リフレーミング承認

【準備】
  ステート管理（フィジカル＋メンタル）/ ミラーニューロンを意識

【上達法】
  良い承認を観察・言語化 → 意識的に練習 → 反射レベルへ
```
