# Free PDF Generator API - HTML/Markdown/URL to PDF

> **Free tier: 500 requests/month** | Generate PDFs from HTML, Markdown, or web URLs with custom settings

Generate production-ready PDFs from HTML strings, Markdown content, or live web URLs. Supports custom page sizes, margins, headers, footers, and landscape orientation. Built on Cloudflare Workers.

## Why Choose This PDF Generator API?

- **Multiple inputs** -- HTML, Markdown, or URL as source
- **Custom layout** -- page size (A4, Letter, Legal), margins, orientation
- **Headers/footers** -- add custom headers and footers to generated PDFs
- **Markdown support** -- direct Markdown-to-PDF without HTML conversion step
- **Free tier** -- 500 requests/month at $0

## Quick Start

```bash
curl -X POST "https://pdf-generator-api.t-mizuno27.workers.dev/generate" \
  -H "Content-Type: application/json" \
  -H "X-RapidAPI-Key: YOUR_KEY" \
  -d '{"html": "<h1>Invoice #001</h1><p>Total: $100</p>", "format": "A4"}' \
  -o invoice.pdf
```

### Python Example

```python
import requests

url = "https://pdf-generator-api.p.rapidapi.com/generate"
headers = {"X-RapidAPI-Key": "YOUR_KEY", "X-RapidAPI-Host": "pdf-generator-api.p.rapidapi.com"}
payload = {"html": "<h1>Report</h1><p>Generated by API</p>", "format": "A4"}

response = requests.post(url, headers=headers, json=payload)
with open("report.pdf", "wb") as f:
    f.write(response.content)
```
