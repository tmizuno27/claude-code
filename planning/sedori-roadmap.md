## 推奨モデル: Amazon FBA × 電脳せどり

### なぜこのモデルか

1. **FBA（Fulfillment by Amazon）** を使えば、保管・梱包・発送・返品対応をAmazonが代行。パラグアイにいても運営可能
2. **電脳せどり**（ネット仕入れ）なので店舗に行く必要がない
3. **FBA納品代行業者** を使えば、仕入れた商品を自宅を経由せずAmazon倉庫に直送できる
4. 日本最大のECプラットフォームで集客力が圧倒的

### 商品の流れ

```
ネットショップ(楽天/ヤフショ等)
    ↓ 購入・発送先をFBA納品代行業者に指定
FBA納品代行業者（日本国内）
    ↓ 検品・ラベル貼り・梱包・FBA倉庫へ発送
Amazon FBA倉庫
    ↓ 注文が入ったら自動で発送
購入者
```

**あなたがやること**: リサーチ → 仕入れ（ネットで購入）→ 価格管理 → 売上管理
**全てPC/スマホで完結**

### 1-2. 初心者向け仕入れ先と手法

#### A. 楽天ポイントせどり（最も初心者向け）

**仕組み**: 楽天市場で商品を購入 → ポイント還元を含めた実質仕入れ値で利益を出す

**手順**:
1. 楽天SPUを最大限上げる（楽天カード作成、楽天モバイル等）
2. お買い物マラソン・スーパーSALE時にまとめて仕入れる
3. ポイント還元率が10〜20%以上になるタイミングを狙う

**例**:
- 商品A: 楽天で3,000円で購入（ポイント15%還元 = 実質2,550円）
- Amazonで4,200円で販売
- FBA手数料+販売手数料: 約1,000円
- 納品代行費: 約200円
- 利益: 4,200 − 2,550 − 1,000 − 200 = **450円**
- 利益率: 約10.7%（ポイント込みで考えると実質もう少し高い）

#### B. Yahoo!ショッピングせどり

**仕組み**: PayPayポイント還元を活用して仕入れコストを下げる

- 5のつく日（5日・15日・25日）にポイントアップ
- 超PayPay祭りなどのキャンペーン時が狙い目
- クーポン併用で実質20〜30%オフになることも

#### C. その他の仕入れ先

| 仕入れ先 | 特徴 |
|---------|------|
| メルカリ | 個人出品で相場より安い商品を拾える |
| ヤフオク | オークション形式で安く仕入れ可能 |
| Qoo10 | 韓国コスメ等が安い |
| 家電量販店のオンラインショップ | ビックカメラ、ヨドバシ等のセール品 |
| コストコオンライン | 限定品・セット品 |

### 目標: 月利10〜15万円 / 仕入れ資金15〜25万円

### 2-1. 仕入れの精度を上げる

- **得意ジャンルを2〜3つに絞る**（例: 食品・日用品・おもちゃ）
- **リピート仕入れ**: 一度利益が出た商品を繰り返し仕入れる
- **セール・イベントカレンダーを作成**: 楽天スーパーSALE、お買い物マラソン、PayPay祭り等のスケジュールを把握
- **Keepaの通知機能**: 狙い商品が値下がりしたらアラート

### 2-2. 利益を最大化する工夫

- **楽天SPUを最大化**: 楽天カード（+2倍）、楽天モバイル（+4倍）、楽天銀行（+0.5倍）等
- **ポイント二重取り**: ハピタス等のポイントサイト経由で購入
- **クーポン・割引の活用**: 各ショップのクーポンを必ずチェック
- **まとめ買い送料対策**: 送料無料ラインを超えて購入

### 2-3. 資金回転を意識する

- **回転率重視**: 1ヶ月以内に売れる商品を中心に
- **売れ残りは早めに値下げ**: 在庫保管料（月額）がかかるため、長期在庫は損失に
- **ポイントの現金化**: 楽天ポイントは楽天ペイで日用品購入に使い、浮いた現金を仕入れに回す

## リスクと対策

### 1. Amazon アカウント停止リスク
- **対策**: 真贋調査に備えて購入履歴（領収書）を必ず保管。怪しいブランド品は扱わない

### 2. 価格競争による利益減少
- **対策**: ニッチなカテゴリー・セット品・限定品を狙う。「薄利多売」より「適正利益率」を重視

### 3. 為替リスク（パラグアイ→日本への送金時）
- **対策**: Wise等の低手数料送金サービスを利用。まとまった額でまとめて送金

### 4. 在庫リスク
- **対策**: 回転率重視のリサーチ。売れ残りは1ヶ月で損切り

### 5. ポイント改悪リスク
- **対策**: 複数の仕入れ先を持つ。ポイントに依存しすぎない利益構造を作る
