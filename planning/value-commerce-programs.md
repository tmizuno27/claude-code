## 提携申請の共通手順

1. バリューコマース管理画面にログイン
2. 「広告」→「詳細検索」でプログラムを検索
3. 対象プログラムの「広告素材を選ぶ」ボタンをクリック
4. **即時提携**のプログラムはすぐに広告掲載可能
5. **審査あり**のプログラムは数日〜1週間で結果通知（メール＋管理画面メッセージ）
6. 承認後、広告素材（バナー・テキスト）を取得して記事に設置

> 参考: https://help.valuecommerce.ne.jp/aff/adlist/alliance/02/

## 報酬単価についての注意

- 上記の報酬単価は **業界相場の目安**。バリューコマース上の実際の単価は管理画面でのみ確認可能
- バリューコマースは「おすすめ報酬アップ広告主」キャンペーンを随時実施 → [キャンペーンページ](https://www.valuecommerce.ne.jp/campaign/)
- 同一広告主でもASPごとに単価が異なる。A8.net・アクセストレード・もしもと比較して高い方を使うべき

## 承認率を上げるコツ

1. **サイトの記事数**: 最低10記事以上あること（3サイトとも条件クリア済み）
2. **ジャンルの一致**: サイトテーマとプログラムの関連性が高いこと
3. **コンテンツ品質**: E-E-A-T（経験・専門性・権威性・信頼性）が評価される
4. **更新頻度**: 直近で更新があること
5. **PV数**: 審査なしのプログラムから始めて実績を作るのも手
