# 事業計画書：パラグアイ発・日本向けマイクロビジネス

## 1. 事業概要

### ミッション
パラグアイ在住の日本人として「現地でしか得られない一次情報」を武器に、
日本市場向けの情報発信＋リモート業務代行で月20万円を安定的に稼ぐ。

### ビジネスモデル（2本柱）

| 柱 | 種別 | 内容 | 特徴 |
|---|---|---|---|
| ①情報発信（資産型） | ストック収益 | パラグアイ・南米の一次情報を有料コンテンツ化 | 一度作れば継続収益 |
| ②リモート業務代行（労働型） | フロー収益 | 時差を活かした翌朝納品サービス | 即金性が高い |

### なぜ勝てるか（競合優位性）

1. **情報の希少性** — パラグアイから日本語で発信している人がほぼゼロ
2. **コスト優位** — パラグアイの生活費は日本の1/3〜1/4。日本の相場以下で提供しても十分な利益
3. **時差の価値** — 日本の夜間に稼働＝「寝ている間に仕事が完了」という付加価値
4. **AI活用** — Claude Agentチームで一人でも高い生産性を実現

## 5. コンテンツ戦略

### 柱①：情報発信の記事カテゴリ

#### カテゴリA：移住・生活情報（最も需要が高い）
- パラグアイの生活費リアルレポート（食費、家賃、光熱費）
- ビザ取得の具体的手順と費用
- 日本人がパラグアイに住むメリット・デメリット
- 現地の治安・医療事情
- 日本食材の入手方法
- 子育て・教育環境

#### カテゴリB：ビジネス・投資情報
- パラグアイでの会社設立方法
- 不動産投資の実態（物件価格、利回り）
- 現地の税制・銀行口座開設
- 日本→パラグアイ輸出入ビジネスの可能性

#### カテゴリC：旅行・文化情報
- パラグアイ観光スポット（日本語情報が皆無）
- 現地の食文化・おすすめレストラン
- パラグアイ人の国民性・付き合い方
- 南米周遊のハブとしてのパラグアイ

#### カテゴリD：ノマド・リモートワーク情報
- パラグアイのインターネット事情
- コワーキングスペース情報
- 生活コスト比較（vs タイ、vs ジョージア、vs ポルトガル）
- デジタルノマドビザ情報

## 6. 柱②：業務代行サービス設計

### サービスメニュー

#### A. AI活用ライティング代行
- **内容**: ブログ記事、SEO記事、メルマガ作成
- **単価**: 3,000〜5,000円/本（2,000〜3,000文字）
- **納品**: 日本時間の翌朝9時まで
- **AI活用**: Claude で下書き → 人間が編集・品質チェック
- **月間キャパ**: 20〜30本

#### B. リサーチ・データ収集代行
- **内容**: 市場調査、競合分析、情報収集
- **単価**: 5,000〜15,000円/件
- **納品**: 1〜3営業日
- **AI活用**: Claude でデータ分析・レポート作成
- **月間キャパ**: 5〜10件

#### C. EC運営サポート
- **内容**: 商品登録、在庫更新、レビュー返信
- **単価**: 月額30,000〜50,000円（固定契約）
- **対応時間**: 日本時間 21:00〜翌9:00
- **AI活用**: Claude で商品説明文自動生成

#### D. 南米現地調査代行（ユニーク商品）
- **内容**: パラグアイ現地の写真撮影、価格調査、店舗訪問
- **単価**: 10,000〜30,000円/件
- **他にない強み**: 現地に住んでいるからできるサービス

## 7. Claude Agentチーム構成

### チーム概要

日常業務を効率化するため、以下のClaude Agentチームを構築する。

```
┌─────────────────────────────────────────┐
│          あなた（オーナー）               │
│    戦略決定・品質管理・現地取材           │
└────────────┬────────────────────────────┘
             │
    ┌────────┼────────┬──────────┐
    ▼        ▼        ▼          ▼
┌────────┐┌────────┐┌────────┐┌────────┐
│Research││Writer  ││Translator││SNS     │
│Agent   ││Agent   ││Agent     ││Agent   │
│        ││        ││          ││        │
│市場調査 ││記事執筆 ││翻訳     ││投稿作成│
│情報収集 ││構成作成 ││スペイン語││分析    │
│競合分析 ││編集支援 ││↔日本語  ││運用    │
└────────┘└────────┘└──────────┘└────────┘
```

### 各Agentの役割

#### 1. Research Agent（リサーチ担当）
- パラグアイの最新ニュース・法律変更の収集
- 移住関連の検索トレンド分析
- 競合コンテンツの調査
- 顧客の業務代行案件のリサーチ

#### 2. Writer Agent（ライティング担当）
- note記事の構成案・下書き作成
- 業務代行案件の記事執筆
- 有料コンテンツの原稿作成
- SEOキーワードを意識した記事最適化

#### 3. Translator Agent（翻訳担当）
- スペイン語の現地情報を日本語に翻訳
- 現地での買い物・手続きのサポート翻訳
- 顧客向け南米調査レポートの翻訳

#### 4. SNS Agent（SNS運用担当）
- X（Twitter）投稿文の作成
- ハッシュタグ戦略の提案
- エンゲージメント分析
- 投稿スケジュール管理

## 9. リスクと対策

| リスク | 対策 |
|---|---|
| コンテンツが売れない | 最初は無料記事でファン獲得→有料化は信頼構築後 |
| 業務代行の品質問題 | 少数案件から始めて実績・評価を積む |
| プラットフォーム依存 | 複数プラットフォーム運用で分散 |
| ネタ切れ | カテゴリを4つに分け、ローテーション |
| 為替リスク | 日本円建てで請求、現地通貨への換金タイミングを調整 |
| AI翻訳の精度 | 重要な場面では現地の知人にダブルチェック依頼 |
