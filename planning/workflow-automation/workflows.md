# 半自動化ワークフローガイド

## 全体像

```
あなたの日常メモ（5分）
        │
        ▼
  ┌─────────────┐
  │  inputs/     │  ← あなたがメモを置く場所
  │  YYYY-MM-DD  │
  └──────┬──────┘
         │
         │  Claude Codeで以下を実行：
         │  「今日のメモから記事とツイートを作って」
         │
         ▼
  ┌──────────────────────────────────────┐
  │  Claudeが自動で4つのAgentを起動       │
  │                                        │
  │  1. Research Agent → SEO・競合調査     │
  │  2. Writer Agent  → 記事の下書き作成   │
  │  3. SNS Agent     → X投稿案10個作成   │
  │  4. Translator Agent → 必要に応じ翻訳  │
  └──────────────────┬───────────────────┘
                     │
                     ▼
  ┌─────────────┐
  │  outputs/    │  ← 完成物がここに出る
  │  YYYY-MM-DD  │
  └──────┬──────┘
         │
         ▼
  あなたが確認・体験談を追記（15分）→ 投稿
```

---

## 使い方（3ステップ）

### ステップ1：メモを書く（5分）

`inputs/` フォルダに今日の日付でファイルを作成する。
テンプレート → `templates/daily-input.md` をコピーして使う。

書く内容は箇条書きでOK：
- 今日体験したこと
- 撮った写真のファイル名
- 気づいた数字（値段、距離など）
- 感じたこと

### ステップ2：Claudeに指示する（1分）

Claude Codeで以下のように指示する：

```
inputs/2026-02-28.md のメモを元に、note記事の下書きとX投稿案を作ってください。
outputs/2026-02-28/ に出力してください。
```

または、定型作業ごとにコマンドを使い分ける：

| やりたいこと | 指示文 |
|---|---|
| 記事作成 | 「inputs/日付.md から記事を書いて outputs/ に出力」 |
| ツイート案 | 「inputs/日付.md から1週間分のツイート案を作って」 |
| 競合調査 | 「"パラグアイ移住" でnoteの人気記事を調査して」 |
| 翻訳 | 「このスペイン語の記事を日本語に翻訳して：URL」 |
| ココナラ出品文 | 「この内容でココナラの出品文を作って：サービス内容」 |

### ステップ3：確認＋投稿（15分）

`outputs/` に生成されたファイルを確認し：
1. 自分の体験談を2〜3行追加する
2. 不自然な表現を修正する
3. noteやXにコピペして投稿する

## 7つのワークフロー

### WF1：note記事作成ワークフロー

**頻度**: 週2回（火・木）
**所要時間**: あなた20分 + AI自動処理

```
入力：あなたのメモ（体験・数字・写真）
  ↓
  ↓ agents/writer-agent.md のプロンプトで処理
  ↓
出力：
  ├── article-draft.md     （記事の下書き 2000〜3000文字）
  ├── article-titles.md    （タイトル案3つ）
  └── article-hashtags.md  （ハッシュタグ5つ）
```

### WF2：X（Twitter）投稿作成ワークフロー

**頻度**: 週1回まとめて生成 → 毎日3投稿
**所要時間**: あなた10分 + AI自動処理

```
入力：今週のテーマ or 記事URL
  ↓
  ↓ agents/sns-agent.md のプロンプトで処理
  ↓
出力：
  └── weekly-tweets.md     （21投稿分：3投稿×7日分）
```

### WF3：ココナラ出品文作成ワークフロー

**頻度**: 随時
**所要時間**: あなた5分 + AI自動処理

```
入力：サービスの概要メモ
  ↓
  ↓ agents/writer-agent.md のプロンプトで処理
  ↓
出力：
  ├── listing-title.md     （出品タイトル案3つ）
  ├── listing-body.md      （出品説明文）
  └── listing-faq.md       （よくある質問3つ）
```

### WF4：業務代行（ライティング案件）ワークフロー

**頻度**: 案件受注時
**所要時間**: あなた15分（品質チェック）+ AI自動処理

```
入力：クライアントからの依頼内容（テーマ、キーワード、文字数）
  ↓
  ↓ agents/writer-agent.md のプロンプトで処理
  ↓
出力：
  └── client-article.md    （納品用記事）
```

### WF5：リサーチ・競合分析ワークフロー

**頻度**: 週1回
**所要時間**: あなた5分（結果確認）+ AI自動処理

```
入力：調査テーマ or キーワード
  ↓
  ↓ agents/research-agent.md のプロンプトで処理
  ↓
出力：
  ├── trend-report.md      （トレンドレポート）
  └── competitor-analysis.md （競合分析）
```

### WF6：スペイン語翻訳ワークフロー

**頻度**: 随時
**所要時間**: あなた2分（確認）+ AI自動処理

```
入力：スペイン語テキスト or URL
  ↓
  ↓ agents/translator-agent.md のプロンプトで処理
  ↓
出力：
  └── translation.md       （日本語訳 + 要約）
```

### WF7：週次レビューワークフロー

**頻度**: 毎週月曜日
**所要時間**: あなた10分 + AI自動処理

```
入力：今週の数値（フォロワー数、PV、売上）
  ↓
  ↓ agents/research-agent.md のプロンプトで処理
  ↓
出力：
  ├── weekly-review.md     （振り返りレポート）
  └── next-week-plan.md    （来週のアクション提案）
```

## 1日のタイムライン例

```
08:00  スマホで写真撮影・メモ（日常のついで）
09:00  inputs/ にメモを保存（5分）
09:05  Claude Codeに記事作成を指示（1分）
09:06  〜 AIが自動処理中 〜 自由時間
09:30  outputs/ の下書きを確認、体験談追記（15分）
09:45  noteに投稿（5分）
09:50  今日のX投稿3つを確認して投稿（5分）
09:55  ココナラのメッセージ確認・返信（10分）
10:05  完了！（合計 約40分）
```

|---|---|---|
| 月 | 週次レビューの数値入力（10分）、来週計画確認 | WF7: 振り返り＋計画作成 |
| 火 | メモ入力 → 記事確認 → 投稿（30分） | WF1: note記事作成 |
| 水 | ココナラ対応、業務代行作業チェック（30分） | WF4: 案件処理 |
| 木 | メモ入力 → 記事確認 → 投稿（30分） | WF1: note記事作成 |
| 金 | 現地取材（写真・メモ）（30分） | WF5: リサーチ |
| 土 | 1週間分ツイート確認（15分） | WF2: ツイート21本生成 |
| 日 | 休み | — |
