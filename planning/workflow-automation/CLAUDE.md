# CLAUDE.md

このファイルは、Claude Code（claude.ai/code）がこのリポジトリで作業する際のガイドです。

## プロジェクト概要

パラグアイ在住の日本人が運営するマイクロビジネスの事業計画・運用ワークスペース。月20万円の収益を以下の2本柱で目指す。

1. **情報発信（資産型）** — パラグアイの一次情報をnote・X等で日本市場向けに発信
2. **リモート業務代行（労働型）** — 日本との12時間の時差を活かした「翌朝納品」サービス（ライティング、リサーチ等）

## ディレクトリ構成

```
business/
├── CLAUDE.md              ← このファイル
├── business-plan.md       ← 事業計画書（収益モデル、KPI等）
├── phase1-roadmap.md      ← フェーズ1実行ロードマップ（ネタ50選含む）
├── workflows.md           ← 半自動化ワークフローガイド（7つのWF定義）
├── docs/                  ← Agentプロンプト定義・参照ドキュメント
│   ├── master-prompt.md   ← コピペ用コマンド集（7コマンド）
│   ├── writer-agent.md    ← 記事執筆Agent
│   ├── sns-agent.md       ← X投稿作成Agent
│   ├── research-agent.md  ← リサーチAgent
│   └── translator-agent.md← 翻訳Agent
├── templates/             ← 入力テンプレート
│   ├── daily-input.md     ← 日々のメモ用
│   └── weekly-review-input.md ← 週次レビュー用
├── inputs/                ← オーナーが日々のメモを置く場所（日付別）
└── outputs/               ← AI生成物の出力先（日付別フォルダ）
```

## 半自動化ワークフロー

オーナーの作業は1日40分。残りはAIが処理する。

**基本の流れ**: `inputs/YYYY-MM-DD.md` にメモを書く → Claudeが `docs/` のルールに従い処理 → `outputs/YYYY-MM-DD/` に出力

各Agentの詳細なプロンプトは `docs/` フォルダ内のファイルを参照。
コピペで使えるコマンド集は `docs/master-prompt.md` を参照。

## 言語・表記ルール

- すべてのドキュメントは**日本語**で記述する
- オーナーは英語・スペイン語を話せないため、必要な場合はAI翻訳を使用する
- 通貨はJPY（円）を基本単位とし、PYG（パラグアイ・グアラニー）は補助的に使用する

## Claude Agentチームの役割

| エージェント | 担当業務 | プロンプト定義 |
|---|---|---|
| Writer Agent | 記事の構成案・下書き作成、SEO最適化 | `docs/writer-agent.md` |
| SNS Agent | X投稿文の作成、ハッシュタグ戦略 | `docs/sns-agent.md` |
| Research Agent | 市場調査、トレンド分析、競合分析 | `docs/research-agent.md` |
| Translator Agent | スペイン語↔日本語の翻訳 | `docs/translator-agent.md` |

## ライティングルール（全Agent共通）

- 一人称は「私」（オーナー本人として書く）
- カジュアルな文体（友達に話すようなトーン）
- 具体的な数字を必ず入れる（「安い」→「200円」）
- AI的な表現は禁止（「いかがでしたでしょうか」「〜と言えるでしょう」等）
- メモにない体験は捏造しない（不足箇所は「【要追記】」と記載）

## 重要な背景情報

- オーナーはパラグアイ在住（UTC-4、日本との時差12時間）
- 対象プラットフォーム：note（有料記事）、X/Twitter（集客）、ココナラ（スキル販売）、ランサーズ/クラウドワークス（業務受注）
- コンテンツは4カテゴリ：移住・生活、ビジネス・投資、旅行・文化、ノマド・リモートワーク
- フェーズ1目標：3ヶ月以内に月5万円達成
