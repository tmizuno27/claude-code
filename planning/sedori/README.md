```
仕入れ先: 楽天市場 / Yahoo!ショッピング（ポイント還元活用）
    ↓
FBA納品代行業者（日本国内で受取→Amazon倉庫へ発送）
    ↓
販売先: Amazon.co.jp（FBA利用で自動発送）
```
