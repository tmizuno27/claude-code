# せどり自動化プラン
## AIで自動化できること / できないこと

作成日: 2026年3月6日

---

## 結論

工程の **70〜80%は自動化・半自動化できる**。ただし全自動は不可能。

---

## 自動化できる部分（70〜80%）

### リサーチ（大幅に効率化可能）
- Keepa APIやスクレイピングで価格・ランキングデータを自動取得
- 楽天・ヤフショの商品価格を自動収集してAmazonと比較
- 条件に合う商品（利益率20%以上、ランキング○万位以内など）を自動抽出
- スプレッドシートに候補リストを自動生成

### 利益計算（完全自動化可能）
- FBA手数料・販売手数料・送料の自動計算
- ポイント還元率を加味した実質仕入れ値の算出
- 利益が出る商品だけをフィルタリング

### 価格管理（完全自動化可能）
- プライスター等のツールで自動価格改定（月額5,280円）
- 競合の価格変動に自動追従
- 赤字ラインを下回ったらアラート

### 出品登録（ほぼ自動化可能）
- 相乗り出品はASINさえ分かれば一括登録可能
- CSVアップロードで大量出品

### 売上管理・分析（完全自動化可能）
- 日次の売上・利益レポート自動生成
- 在庫回転率の監視
- 仕入れ判断の振り返り分析

---

## 自動化できない部分（20〜30%）

### 仕入れの「購入ボタンを押す」行為
- 楽天やヤフショでのBOTによる自動購入は利用規約で禁止
- 違反するとアカウントBAN

### 最終的な仕入れ判断
- 出品規制の確認（セラーセントラルにログインが必要）
- 季節性・トレンドの判断
- ライバル急増の察知

### FBA納品代行業者とのやり取り
- イレギュラー対応（商品の破損、数量の不一致など）

### Amazonアカウントの健全性管理
- 真贋調査への対応
- 知的財産権の侵害警告への対応
- パフォーマンス指標の維持

---

## 理想的なワークフロー

```
【AIが自動でやる部分】
毎日: 楽天・ヤフショの価格を巡回 → Amazon価格と比較
  → 利益率20%以上の商品をスプレッドシートにリスト化
  → Keepaデータから売れ行きを自動判定
  → 「仕入れ候補リスト」を通知

【あなたが10〜15分でやる部分】
リストを確認 → 「これ買う/買わない」を判断
  → 楽天・ヤフショで購入（発送先はFBA納品代行業者）

【AIが自動でやる部分】
出品登録 → 価格改定 → 売上トラッキング → 日次レポート
```

あなたの作業は「リストを見て買うかどうか決めて、購入する」だけ。
→ **1日30分〜1時間で運用可能**

## 作成予定のツール

| ツール | 内容 | 前提条件 |
|--------|------|---------|
| 利益自動計算シート | 商品URL貼り付けで利益計算 | なし（すぐ作れる） |
| 仕入れリスト管理シート | 仕入れ→出品→販売の全フロー記録 | なし |
| 価格差リサーチスクリプト | 特定カテゴリーの商品を自動収集 | Keepa API（月€19） |
| Amazon出品一括登録 | CSV生成→セラーセントラルにアップロード | Amazon出品アカウント |

## 必要なAPI・サービス

| サービス | 用途 | 費用 |
|---------|------|------|
| Keepa API | 価格・ランキング推移データ取得 | 月€19（約3,000円） |
| Amazon SP-API | 出品・在庫・注文管理 | 無料（開発者登録要） |
| Google Sheets API | スプレッドシート連携 | 無料 |
| プライスター | 自動価格改定 | 月5,280円 |
