# Fiverr・Upwork・Payoneer アカウント登録チェックリスト

サンプルレポート完成後、すぐにGigを出品できるよう、先に登録を済ませておく。

---

## 最優先: Payoneer（審査に2〜5日かかる）

```
□ https://www.payoneer.com にアクセス
□ 「無料登録」をクリック
□ 以下の情報を入力:
  - 氏名（パスポートと同じローマ字表記）
  - メールアドレス
  - 生年月日
  - 住所（英語表記）
    例: 1-2-3 Shibuya, Shibuya-ku, Tokyo, 150-0002, Japan
□ 本人確認書類をアップロード:
  - パスポート or 運転免許証（英語表記があるもの）
  - 住所確認書類（公共料金の請求書 or 銀行明細）
□ 銀行口座を登録（日本の銀行口座でOK）
□ 審査完了を待つ（2〜5営業日）

※ FiverrもUpworkもPayoneerが最も便利な出金方法
```

## Fiverr アカウント登録

### ステップ1: 基本登録
```
□ https://www.fiverr.com にアクセス
□ 「Join」をクリック
□ メールアドレス or Googleアカウントで登録
□ ユーザーネームを決める
  推奨: japanresearcher、tokyoresearch、jpmarketpro 等
  ※ 後から変更できないので慎重に
□ メール認証を完了
```

### ステップ2: プロフィール設定
```
□ プロフィール写真をアップロード
  - 顔がはっきり見える正面写真
  - 白 or 無地の背景
  - ビジネスカジュアル（スーツ不要、清潔感重視）
  - スマホのポートレートモードで撮影OK
  - サイズ: 最低 400×400px

□ Display Name を設定
  - 本名 or ビジネスネーム

□ Description（自己紹介）を入力
  → 13_日本市場リサーチ_No1獲得戦略.md の「1-2. Fiverrプロフィール文」をコピペ

□ Languages を設定
  - Japanese: Native/Bilingual
  - English: Fluent or Conversational

□ Skills を追加
  - Market Research
  - Competitive Analysis
  - Data Analysis
  - Japanese Translation
  - Business Strategy

□ Education を入力（あれば）

□ Certification を入力（あれば）
```

### ステップ3: Seller アカウントの有効化
```
□ 「Become a Seller」をクリック
□ Seller onboarding を完了（質問に回答）
□ 「Create a Gig」で最初のGigを作成
  → 13_日本市場リサーチ_No1獲得戦略.md の「Gig 1: 集客用」を参照
□ 本人確認（ID verification）を完了
  - パスポート or 運転免許証
  - セルフィー（自撮り）
```

### ステップ4: Gig を出品
```
□ Gig 1（集客用）を作成・公開
  - タイトル、説明文、価格、FAQ を設定
  - サムネイル画像をアップロード
  - サンプルレポートをポートフォリオに添付

□ Gig 2（ソーシャルリスニング）を作成・公開

□ Gig 3（月額レポート）を作成・公開

□ 各Gigの設定を確認:
  - Search Tags（5つまで）を設定
  - Delivery Time を確認
  - Requirements（注文時にクライアントに聞く質問）を設定:
    "What industry or market are you interested in?"
    "What specific questions do you need answered?"
    "Do you have a specific competitor you want analyzed?"
    "What is the purpose of this research? (market entry, investment, etc.)"
```

## Upwork アカウント登録

### ステップ1: 基本登録
```
□ https://www.upwork.com にアクセス
□ 「Sign Up」→ 「I want to work」を選択
□ メールアドレスで登録
□ メール認証を完了
```

### ステップ2: プロフィール設定
```
□ プロフィール写真をアップロード（Fiverrと同じものでOK）

□ Title を設定:
  "Native Japanese Market Researcher | Japan Market Entry Specialist"

□ Overview を入力:
  → 13_日本市場リサーチ_No1獲得戦略.md の「1-3. Upworkプロフィール文」をコピペ

□ Hourly Rate を設定:
  - 最初は $30/hr（実績ゼロなので控えめに）
  - レビュー5件以上で $50/hr に引き上げ
  - Top Rated になったら $75〜100/hr

□ Skills を追加:
  - Market Research
  - Market Analysis
  - Competitive Analysis
  - Japanese
  - Business Research
  - Data Analysis
  - Report Writing
  - Industry Analysis

□ Portfolio にサンプルレポートをアップロード

□ Employment History（あれば）

□ Education（あれば）
```

### ステップ3: スキルテストを受験
```
□ 「Japanese Proficiency Test」を受験（あれば）
□ 「English Proficiency Test」を受験（あれば）
□ 「Market Research」関連のテストを受験（あれば）

※ トップ10%以内のスコアだとバッジがつき、信頼性が上がる
```

### ステップ4: Connects を購入
```
□ Upworkでの提案にはConnects（ポイント）が必要
□ 最初は無料で40 Connects もらえる
□ 追加購入: $0.15/Connect
□ 1提案あたり2〜6 Connects消費
□ 最初の月は$15〜30分（100〜200 Connects）を購入推奨
```

### ステップ5: 最初の提案を送る
```
□ 「Find Work」で以下のキーワードで検索:
  - "Japanese market research"
  - "Japan market entry"
  - "Japanese translation research"
  - "Japan competitive analysis"
  - "Japanese consumer research"

□ 良い案件に1日3〜5件の提案を送る
  → 13_日本市場リサーチ_No1獲得戦略.md の「Upwork提案文テンプレート」を使用

□ 提案文は毎回カスタマイズする（テンプレのコピペは絶対NG）
```

## 全体の実行スケジュール

```
【Day 1（今日）】
□ Payoneer登録（最優先。審査待ちの間に他を進める）
□ Fiverr登録 + プロフィール設定
□ Upwork登録 + プロフィール設定

【Day 2〜3】
□ サンプルレポートの最終確認・PDF化
□ Canvaでサムネイル画像を作成
□ 自己紹介動画を撮影（スマホでOK）

【Day 4〜5】
□ Fiverr: 3つのGigを出品
□ Upwork: ポートフォリオをアップロード
□ Upwork: 最初の提案を5件送る

【Day 6〜7】
□ メッセージへの返信を開始（1時間以内が理想）
□ Upwork: さらに5〜10件の提案を送る
□ Gigの統計を確認、改善点を探す

【Week 2以降】
□ 毎日の提案ルーティンを開始（朝30分 + 夜30分）
□ 最初の受注を目指す
```

## Canva サムネイル画像の作成ガイド

```
サイズ: 1280 × 769 px（Fiverrの推奨サイズ）

デザイン要素:
├── 背景: 東京のスカイライン or 日本地図（半透明のオーバーレイ）
├── メインテキスト: "Japanese Market Research"（白、太字）
├── サブテキスト: "Native Researcher Based in Japan"（白、細字）
├── アクセント: 日本の国旗アイコン 🇯🇵
├── サンプルレポートの画像（斜めに配置してチラ見せ）
└── 色使い: 紺色(#1a365d) + 白(#ffffff) + 赤アクセント(#c53030)

Canvaでの手順:
1. Canva.com にアクセス
2. 「カスタムサイズ」→ 1280 × 769 px
3. テンプレートから「ビジネスプレゼンテーション」系を選ぶ
4. 上記の要素を配置
5. PNG or JPG でダウンロード

※ 3つのGigそれぞれ別のサムネイルを作る
  Gig 1: "Market Research" を強調
  Gig 2: "Consumer Insights" を強調
  Gig 3: "Monthly Reports" を強調
```
