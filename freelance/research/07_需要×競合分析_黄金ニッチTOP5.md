## 黄金ニッチ TOP 5

**「需要が多い」×「競合が少ない」** の交差点にあるもの。

---

### 1位: Amazon Japan 出品最適化

```
需要: ★★★★★   競合: ★☆☆☆☆   → スコア最高
```

| 項目 | データ |
|---|---|
| **市場規模** | Amazon Japan GMV **$726億**（世界第3位のAmazonマーケットプレイス） |
| **アクセス数** | 月間 **5.5億アクセス** |
| **EC市場成長率** | 年10.15%成長、越境EC は年13%成長 |
| **Upwork上の競合** | Amazon FBA全体で1,196件の求人があるが、Japan特化のフリーランサーは約8名 |
| **なぜチャンス** | Amazon USの専門家は大量にいるが、日本特化が極端に不足。直訳では「売れない」ことがデータで証明されており、ネイティブの需要が構造的に存在 |
| **単価** | 1商品 $30〜80、月額管理 $500〜1,500 |

> **月間5.5億アクセスの市場に対して、専門家が8人しかいない。これが最大のチャンス。**

## 需要が「ない」or「プラットフォームに出てこない」もの

正直な評価。

| ニッチ | 現実 |
|---|---|
| **UXライティング** | 需要は存在するが、Upwork/Fiverr上での求人は極めて少ない。概念自体がまだ新しい |
| **文化コンサルティング** | 需要はあるが、単独では成立しにくい。市場リサーチやローカライズに「付加価値」として付けるのが現実的 |
| **トランスクリエーション** | 大企業の需要はあるが、代理店経由が主流。プラットフォーム上での直接案件は少ない |
| **データ入力（一般）** | AI自動化で衰退中。ただしAI学習データ収集は特需あり（単発・不安定） |

## 詳細需要データ（ニッチ別）

### 日本語翻訳（ベースライン・参考）

- Upwork求人: **903件**が常時オープン（JP→EN: 520件、EN→JP: 383件）
- 日本の言語サービス市場: **$26.4億（2026年）**
- 翻訳に投資した企業は**収益増加の可能性が1.5倍**
- 単価: 一般 $20/時、専門（法律/医療/技術）$30〜65/時
- **判定: 最大の需要だが競合も最大。そのまま参入すると価格競争に巻き込まれる**

### 日本語字幕

- 日本ストリーミング市場: **$72億（2025年）、前年比15%成長**
- Netflix, Prime Video, U-Nextが日本語コンテンツを積極拡大中
- Upwork字幕カテゴリ: **250件**の求人（日本語特化はその一部）
- AI自動字幕は日本語ではまだ精度不足 → 人間の需要が続く
- 単価: 1行 $0.08〜0.15、急ぎ $25〜60/時
- **判定: 安定した中程度の需要。1件あたりの単価は低いが量でカバー可能**

### 日本語QAテスト

- Glassdoor: 日本で**QAテスター10件、ゲームテスター9件**の求人
- Games Jobs DirectにLQA求人あり
- ただし大半が代理店経由 or 直接雇用。フリーランスプラットフォーム上では薄い
- 単価: 標準LQA $15〜30/時、ゲーム特化 $25〜45/時
- **判定: 業界内では需要あり。ただしUpwork/Fiverr上では案件が見つけにくい**

### 日本語データ入力/収集

- 一般的なデータ入力: AI自動化により衰退傾向
- AI学習データ（画像ラベリング、音声収録、言語アノテーション）: 特需あり
- 実例: $100,000規模の日本語音声収録プロジェクト（2025年7月）が確認済み
- 単価: 一般 $8〜20/時、AI言語データ $15〜30/時
- **判定: 一般データ入力は避けるべき。AI学習データは高単価だが不安定**

## 最終結論：推奨戦略

```
【即効性 × 高需要 × 低競合】

Step 1: Amazon Japan 出品最適化 で参入
        → 需要◎、競合8名、即日開始可能

Step 2: 日本市場リサーチ を追加
        → Claude と協業で効率化、高単価

Step 3: 実績が溜まったら SaaS ローカライズ に特化
        → 長期契約で安定収入の柱に
```
