実行可能な具体的ステップに落とし込んだ完全ガイド。

## Phase 0: 準備（1週間）

### やること

**1. 自分の「売り物」を決める**

以下から最大2〜3つ選ぶ（最初から広げすぎない）：

| カテゴリ | あなたのスキル次第で選択 |
|---|---|
| 翻訳・ローカライズ | 英⇔日翻訳、UI翻訳、字幕翻訳 |
| リサーチ代行 | 日本市場調査、競合分析 |
| コンテンツ作成 | 日本語記事、商品説明、SNS投稿 |
| VA（バーチャルアシスタント） | メール対応、スケジュール管理 |
| テック系 | コーディング、データ入力、自動化 |

**2. 英語力の確認と強化**

- プラットフォームでのやり取りは全て英語
- 最低限「提案文」と「チャットでのやり取り」ができればOK
- 不安ならClaude と一緒に英文を作ればいい

**3. アカウントに必要なものを揃える**

```
□ プロフェッショナルな顔写真（スマホでOK、背景は白か無地）
□ 英語の職務経歴（簡潔に箇条書き）
□ ポートフォリオサンプル 3点以上
□ PayPal または Payoneer アカウント（報酬受取用）
□ 本人確認用の身分証明書
```

## Phase 1: アカウント構築（1〜2週間目）

### Fiverr のセットアップ

**Gig を3つ作る（これが最低ライン）**

Gig 1 - 入門用（低価格で実績を作る）
```
タイトル: I will translate English to natural Japanese
Basic:    500 words / $10 / 2日納品
Standard: 1,000 words / $20 / 3日納品
Premium:  2,500 words / $45 / 5日納品
```

Gig 2 - 中価格帯
```
タイトル: I will localize your website or app into Japanese
Basic:    1ページ / $25 / 3日納品
Standard: 3ページ / $60 / 5日納品
Premium:  全サイト / カスタム見積もり
```

Gig 3 - 高単価を狙う
```
タイトル: I will research the Japanese market for your business
Basic:    簡易レポート / $50 / 5日納品
Standard: 詳細レポート / $120 / 7日納品
Premium:  戦略提案付き / $250 / 10日納品
```

**各Gigに必要なもの:**
- サムネイル画像（Canvaで無料作成）
- Gig説明文（SEOキーワードを入れる）
- FAQ 3つ以上
- できれば自己紹介動画（30〜60秒）

## Phase 2: 最初の5件を取る（2〜6週間目）

**ここが最大の壁。全エネルギーを集中させる。**

### 毎日のルーティン

```
朝（30分）
├─ Upwork: 新着案件をチェック
├─ 良い案件に3〜5件提案を送る
└─ Fiverr: Buyer Requestに応募

夜（30分）
├─ メッセージ確認・返信（1時間以内が理想）
├─ Upwork: さらに2〜3件提案
└─ Fiverr: Gigの最適化（タグ、説明文の改善）
```

### Upwork 提案文のテンプレート

```
Hi [Client Name],

I noticed you need [specific task from job post].
As a native Japanese speaker, I can [specific solution].

[1-2 sentences showing you READ the job post carefully]

Here's my approach:
1. [Step 1]
2. [Step 2]
3. [Step 3]

I've attached a brief sample relevant to your project.

Happy to discuss details. When would be a good time?

Best,
[Your Name]
```

**重要ポイント:**
- テンプレのコピペは絶対NG（クライアントは見抜く）
- 毎回、案件に合わせてカスタマイズする
- 冒頭で「あなたの案件を理解している」ことを示す

### 最初の5件の心構え

```
× 利益を最大化する
○ 5つ星レビューを確実に取る

これが今後の全てを決める。
```

- 納期より1日早く納品する
- 求められた以上の品質を出す
- 納品後に「何か修正があればお気軽に」と伝える
- 丁寧に5つ星レビューをお願いする

## Phase 3: 単価を上げる（2〜4ヶ月目）

### レビュー5件を超えたらやること

**1. 値上げ**
```
Fiverr:  全Gigの価格を20〜30%上げる
Upwork:  時給を$5〜10上げる
```

**2. 専門特化を始める**

広く浅くではなく、特定分野に絞る：

```
例:
「翻訳全般」→「SaaS/テックアプリの日本語ローカライズ専門」
「リサーチ全般」→「越境ECの日本市場参入リサーチ専門」
```

**3. Upwork で継続契約を提案**

良いクライアントに対して：
```
"I've enjoyed working on this project. If you have ongoing
Japanese translation needs, I'd be happy to offer a monthly
retainer at a discounted rate. For example, up to 10,000 words
per month for $XXX."
```

## Phase 4: 安定化（4〜6ヶ月目）

### 収入の柱を3本作る

```
柱1: Fiverr の自動収入    → 月 $500〜1,000
柱2: Upwork の継続契約    → 月 $1,500〜3,000
柱3: 直接契約（紹介など）  → 月 $500〜1,000
─────────────────────────────────
合計目標:                    月 $2,500〜5,000
```

### AI（Claude）の活用で効率化

```
翻訳案件:
  Claude が初稿 → あなたが修正・品質管理 → 納品
  作業時間を50〜70%短縮

リサーチ案件:
  Claude が情報収集・整理 → あなたが分析・レポート作成

提案文作成:
  Claude が下書き → あなたが調整 → 送信
```

---

## Phase 5: スケール（6ヶ月目〜）

- 単価をさらに引き上げる（Top Rated になれば強気でOK）
- 自分のWebサイトを作り、プラットフォーム外でも集客
- 特定クライアントとの直接契約に移行（手数料0%）
- 余裕があれば他のフリーランサーに下請けに出す側になる

```
Month 1    $0〜200      ← 実績作り。赤字でもOK

Month 5    $1,500〜3,000 ← 安定期に入る

```
