# 【ココナラにコピペ】3サービスの出品テキスト

出品手順：
1. ココナラにログイン → 「出品する」→「サービスを出品」
2. 各サービスの「タイトル」「説明文」「よくある質問」をコピペ
3. 【要追記】の箇所を自分の情報で埋める
4. サムネイル画像を設定（Canvaで作成推奨）
5. 公開する

## サービス①：パラグアイ移住・生活相談（3,000円）
==============================================

### カテゴリ
暮らし・住まい > その他

### タイトル（以下から1つ選ぶ）
- パラグアイ在住者がリアルに答える移住相談

- 南米パラグアイ在住の日本人が相談に乗ります

### 説明文（コピペ用）

パラグアイに実際に住んでいる日本人が、移住・生活に関するご質問にお答えします。

ネットの情報だけでは分からない「現地のリアル」をお伝えします。

◆こんな方におすすめ
・パラグアイへの移住を検討している方
・パラグアイの生活費や物価を知りたい方
・ビザの取得方法や手続きの流れを知りたい方
・治安や医療事情が気になる方
・南米でのビジネスに興味がある方

◆ご提供内容
・テキストチャットでの質問回答（5問まで）
・質問は移住・生活・物価・治安・ビザ・仕事なんでもOK
・回答は24時間以内にお返しします

◆私について
パラグアイ在住の日本人です。現地のスーパー、病院、役所、銀行など日常的に利用しており、生活者目線でリアルな情報をお伝えできます。

※投資・法律の専門的な助言はできません
※情報は個人の経験に基づくものです

### 購入にあたって

・質問は具体的にいただけると、より詳しくお答えできます
 （例：「生活費はいくら？」→「2人暮らしの1ヶ月の食費はいくらくらいですか？」）
・回答は24時間以内を目安にお送りします
・1回のご購入で5問までご質問いただけます

### よくある質問

Q. スペイン語が話せなくても生活できますか？
A. 日常生活では苦労する場面もありますが、翻訳アプリやジェスチャーで乗り切れる場面も多いです。具体的にどんな場面で困るか・困らないかをお伝えします。

Q. パラグアイの治安は大丈夫ですか？
A. エリアによって大きく違います。私が住んでいるエリアの治安状況や、気をつけるべきポイントを実体験ベースでお伝えします。

Q. 日本からパラグアイへの行き方も教えてもらえますか？
A. はい。乗り継ぎルートや所要時間、航空券の相場など、実際に利用した経験をもとにお答えします。

### 設定
- 価格：3,000円
- お届け日数：3日
- 購入にあたってのお願い：上記をコピペ
