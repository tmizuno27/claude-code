# AI自動化ビジネス 比較検討表（放置自動化特化版）

作成日: 2026-03-03（更新）
条件: **放置・自動化で収益が得られるモデル**に限定
前提: 日本語ネイティブ / 英語は英検3級 / スペイン語は挨拶程度
→ ただし**AIツール（Claude/DeepL等）で言語の壁を越えられるなら対象に含める**

---

## 選定基準

以下の条件をすべて満たすもののみ掲載：
- 一度仕組みを作ったら、**日常的な営業・クライアント対応が不要**（または最小限）
- 収益が**自動的・継続的**に発生する構造
- Claude Codeで構築可能
- 言語の壁は**AI翻訳でカバー可能**であればOK

**除外したもの：**
- クライアントワーク型 → 営業・対応が常時必要で「放置」にならない
- コンサル・代行系 → 労働集約型で放置不可
- リアルタイムの外国語会話が必須のもの → AI翻訳でもカバー困難

## 言語カバー力の評価基準

| AI翻訳カバー力 | 説明 | 例 |
|:---:|------|------|
| ◎ 完全カバー | テキストベースのやり取りのみ。AIが翻訳すれば問題なし | 商品説明文、メール、テンプレート販売ページ |
| ○ ほぼカバー | 定型的なやり取りが中心。AI翻訳+テンプレートで対応可能 | カスタマーサポート（FAQ型）、コース説明文 |
| △ 部分カバー | 即座の対話や微妙なニュアンスが必要な場面がある | リアルタイム商談、複雑なクレーム対応 |

## 各案の詳細

### 1. アフィリエイトブログ自動量産

| 項目 | 内容 |
|------|------|
| **仕組み** | Claude Code + cronで定期的にSEOキーワード調査→記事生成→WordPress API経由で自動投稿 |
| **収益源** | Google AdSense + アフィリエイト（A8.net、もしもアフィリエイト等） |
| **放置ポイント** | スクリプトをTask Schedulerで定期実行。週1回の品質チェック推奨 |
| **活かせる経験** | 既存の「南米おやじのAI実践ラボ」を拡張 |
| **リスク** | GoogleのAIコンテンツ評価変動、低品質コンテンツはペナルティリスク |
| **現実的な収入推移** | 3ヶ月目: 月1万円 → 6ヶ月目: 月5万円 → 12ヶ月目: 月10-15万円 |
| **Claude Codeで作るもの** | 記事自動生成パイプライン（リサーチ→執筆→投稿→内部リンク最適化） |

## 注意事項

1. **完全放置は幻想に近い**: 「ほぼ放置」であっても月1-2回の品質チェック・メンテは必要
2. **収益化まで時間がかかる**: 大半は3-6ヶ月で初収益。即金ではない
3. **量が勝負**: デジタル商品は1個だけでは月数千円。10個、20個と積み上げて初めて安定収入になる
4. **品質管理は必須**: AI生成コンテンツでも、最終チェックは人間がやるべき（信頼性に直結）
5. **プラットフォーム依存リスク**: Kindle/Udemy/YouTubeの規約変更で収益が突然0になるリスクあり。複数プラットフォームに分散が重要
6. **AI翻訳の品質は十分実用レベル**: Claude/DeepLの英語出力はネイティブに近い品質。商品説明や記事レベルなら人間チェック不要で通用する
7. **英語圏市場は日本の10倍以上**: 同じ労力をかけるなら英語展開した方が効率的。AI翻訳のコストはほぼゼロ
