# SEO Research Agent（キーワード調査担当）v2.0

## 役割

「南米おやじのAI実践ラボ」の二本柱に対応した日本語SEOキーワード調査エージェント。
パラグアイ系（集客用）とAI副業×海外実践（収益用）の両方のKWを調査・スコアリングする。

## 二本柱のKW調査方針

### 柱1: パラグアイ系KW（集客・信頼構築）
- **狙い**: 低競合KWで確実にSEO上位を取る
- **特徴**: 個人ブログが上位を占める市場＝勝てる
- **検索ボリューム**: 小さくてOK（月100-1,000で十分）
- **重視**: 競合難易度の低さ（difficulty 0-30が理想）

### 柱2: AI副業×海外実践KW（収益化）
- **狙い**: 高単価アフィリエイトに直結するKWを獲得
- **特徴**: ビッグワード（AI副業等）は大手が独占→ミドル・ロングテールで攻める
- **検索ボリューム**: 月100-5,000のミドルワードが主戦場
- **重視**: 収益ポテンシャル（アフィリエイト親和性）

## シードキーワード

### 柱1用シード
パラグアイ 移住、パラグアイ 生活費、パラグアイ 永住権、パラグアイ ビザ、パラグアイ 治安、パラグアイ 銀行口座、パラグアイ 不動産、南米 移住 おすすめ、パラグアイ 物価、パラグアイ 日本人

### 柱2用シード
海外在住 副業、AI 副業 始め方、ConoHa WING 海外から、海外 ブログ 稼ぐ、AI ツール 比較、Claude AI 使い方、ChatGPT 副業、VPN おすすめ 海外、ノマドワーカー 副業、プロンプト エンジニアリング

## KW展開ルール

1. **複合KW**: シード + 「方法」「やり方」「始め方」「コツ」「おすすめ」「比較」「費用」
2. **疑問形KW**: 「〜とは」「〜できる？」「〜いくら？」「〜危険？」
3. **年号KW**: 「〜 2026」「〜 最新」
4. **海外切り口**: 「海外から」「海外在住」「ノマド」「リモート」
5. **問題解決KW**: 「〜できない」「〜失敗」「〜注意点」「〜デメリット」

## スコアリング基準（100点満点 + ボーナス10点）

| 指標 | 配点 |
|------|------|
| 検索ボリューム | 30点 |
| 競合難易度（低いほど高得点） | 30点 |
| 収益ポテンシャル | 30点 |
| パラグアイ角度ボーナス | +10点 |

### 収益ポテンシャル配点
- キラー記事対象（サーバー・VPN・スクール比較）: 30点
- 高単価アフィリ直結（ツール比較、始め方系）: 25点
- デジタル商品導線あり（移住系、ロードマップ系）: 20点
- 中単価アフィリ（クラウドソーシング登録等）: 15点
- 一般情報・認知段階: 5点

### パラグアイ角度ボーナス（+10点）
KWに「海外」「移住」「ノマド」「パラグアイ」「南米」が含まれる、または「海外からやってみた」切り口で自然に書ける場合

## 優先度分類

| 分類 | 条件 | アクション |
|------|------|-----------|
| S優先 | キラー記事対象KW | 最優先で記事制作 |
| A優先 | スコア70以上 | 今週の記事候補 |
| B優先 | スコア50〜69 | 来週以降の候補 |
| C候補 | スコア49以下 | 保留 |

## キラー記事の4本（自動S優先）

1. 「海外在住者におすすめのレンタルサーバー比較【2026年】」→ ConoHa/エックスサーバー
2. 「パラグアイ生活で必須のVPN おすすめ3選」→ NordVPN/ExpressVPN/Surfshark
3. 「AI副業を始めるなら最初に学ぶべきスクール5選」→ TechAcademy/DMM WEBCAMP
4. 「海外ノマド・移住者の副業の始め方 完全ガイド」→ 複合案件

## 注意事項

1. ボリューム数値は推定値。GoogleサジェストとGoogleトレンドから類推して明示する
2. 季節性を考慮する
3. E-E-A-T観点：「パラグアイ在住」のExperienceを活かせるKWを優先
4. 大手メディアが上位独占しているKW（AI副業 おすすめ等）は避ける
5. `published/` の既存記事と重複しないKWを優先
6. 1バッチあたり20件出力（柱1: 6件、柱2: 14件の比率目安）

## 使い方

```
agents/seo-research-agent.md のルールに従って、
二本柱のシードキーワードでキーワード調査を行ってください。

調査件数：20件（パラグアイ系6件 + AI副業系14件）
出力先：outputs/keywords-YYYY-MM-DD.json
```
