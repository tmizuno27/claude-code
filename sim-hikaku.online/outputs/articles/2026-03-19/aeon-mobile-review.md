# イオンモバイルの評判・口コミ｜全国イオン店舗サポートの実力を徹底検証

「イオンモバイルって安いらしいけど、速度や品質は大丈夫？」「店舗でサポートしてもらえるって本当？」

格安SIMへの乗り換えを検討するとき、こんな不安を感じる方は少なくありません。

筆者（南米おやじ）は海外在住の立場から日本の格安SIM市場を継続的にリサーチしています。その中でもイオンモバイルは**「全国200店舗以上のイオンで対面サポートが受けられる」**という、格安SIMとしては異例の手厚さが際立つサービスです。

オリコン顧客満足度調査で2022年〜2024年の3年連続「格安SIM第1位」を獲得している実績もあり、利用者からの評価は高水準を維持しています。

この記事では、2026年3月時点の最新情報をもとに、イオンモバイルの料金プラン・通信速度・口コミ・メリットとデメリットを徹底的に検証します。

## イオンモバイルとは？基本情報まとめ

イオンモバイルは、イオンリテール株式会社が運営するMVNO（格安SIM）サービスです。イオングループの強みを活かした店舗サポート体制が最大の特徴です。

| 項目 | 内容 |
|------|------|
| 運営元 | イオンリテール株式会社 |
| 回線 | ドコモ回線 / au回線（選択制） |
| 料金プラン | さいてきプラン（0.5〜10GB）/ さいてきプランMORIMORI（20〜200GB）/ やさしいプラン（60歳以上限定） |
| 店舗サポート | 全国200店舗以上のイオンで対面対応 |
| 契約事務手数料 | 3,300円（税込） |
| 解約金 | なし（最低利用期間なし） |
| 対応SIM | 物理SIM / eSIM |
| 支払い方法 | クレジットカードのみ |
| 通話料 | 11円/30秒（イオンでんわ使用時） |

格安SIMの中で「実店舗でスタッフに相談しながら契約・サポートを受けたい」というニーズに最も応えているのがイオンモバイルです。

## イオンモバイルの通信速度【実測データで検証】

格安SIMを選ぶ上で気になるのが通信速度です。イオンモバイルの実測データを時間帯別に確認しましょう。

### 時間帯別の通信速度（みんそく調べ・2026年データ）

| 時間帯 | 下り速度（平均） | 上り速度（平均） |
|--------|----------------|----------------|
| 朝（6〜8時） | 約40Mbps | 約10Mbps |
| 昼（12〜13時） | **約5〜6Mbps** | 約8Mbps |
| 夕方（17〜19時） | 約20Mbps | 約9Mbps |
| 夜（21〜23時） | 約35Mbps | 約10Mbps |
| 深夜（0〜5時） | 約100Mbps | 約12Mbps |
| **全時間帯平均** | **約28Mbps** | **約10Mbps** |

朝・夜・深夜は快適に使える速度が出ていますが、**昼12時〜13時の混雑時間帯は5〜6Mbps程度**まで低下します。これはイオンモバイルに限らず、ドコモ回線を借りているMVNO全般に共通する傾向です。

5〜6Mbpsあれば、LINEのメッセージ送受信・Webブラウジング・SNSの閲覧は問題なく行えます。ただし高画質動画のストリーミングや大容量ファイルのダウンロードは、昼休み時間帯には厳しくなることがあります。

実際のユーザーの口コミから、良い評判と悪い評判の両方を整理しました。

### 良い評判・口コミ

- 「3GBで月額1,078円。大手キャリアの3分の1以下で十分使えている」
- 「子供3人分のスマホ代がシェアプランで月5,000円以下に収まった」
- 「1GB刻みで選べるから、無駄がなくて助かる」

**店舗サポートへの安心感**

- 「近所のイオンで対面で相談できるから乗り換えの不安がなかった」
- 「親のスマホをイオンモバイルにしたら、わからないことがあるとイオンに行って聞いてくれるようになった」
- 「初期設定をスタッフがやってくれたので、機械が苦手な母でも問題なかった」

**プランの柔軟性への評価**

- 「余ったデータは翌月繰り越しできるし、プラン変更も無料で何度でもできるから自由度が高い」
- 「0.5GBプランがあるのが良い。Wi-Fi環境が整っていれば月803円で十分」

### 悪い評判・口コミ

**通信速度への不満**

- 「お昼休みの時間帯は明らかに遅くなる。動画は見られない」
- 「大手キャリアと比べると、体感で速度差を感じる場面がある」

- 「クレジットカードしか使えないのが不便。口座振替に対応してほしい」

**速度制限後の使いにくさ**

- 「データ容量を使い切ると200kbpsに制限される。これはかなり遅い」

### 口コミのまとめ

全体として、**「料金の安さ」「店舗サポートの安心感」への満足度は非常に高い**一方で、**昼時間帯の速度低下と支払い方法の制限**が主な不満点となっています。

## イオンモバイルのメリット5つ

### 1. 全国200店舗以上のイオンで対面サポート

格安SIMの中で、これほどの店舗数で対面サポートを提供しているのはイオンモバイルだけです。契約手続き・プラン変更・故障受付・初期設定サポートまで、スタッフが対応してくれます。

「格安SIMにしたいけど、ネットだけで手続きするのは不安」という方や、ご高齢の家族に格安SIMを持たせたい方にとって、**買い物ついでにイオンで相談できる**のは大きな安心材料です。

### 2. 1GB刻みの細かいプラン設定

0.5GBから200GBまで、非常に細かくプランが用意されています。「3GBだと少し足りないけど5GBは多すぎる」という微妙なニーズにも4GBプランで対応可能です。

しかも**プラン変更は無料で何度でもOK**。翌月から新プランが適用されるため、季節や生活スタイルの変化に合わせて柔軟に調整できます。

### 3. 余ったデータは翌月繰り越し

使い切れなかったデータ容量は自動的に翌月に繰り越されます。「今月はWi-Fi環境にいることが多くて余った」という場合でも、翌月に上乗せして使えるので無駄がありません。

### 4. 60歳以上限定のやさしいプランが安い

やさしいプランは通常のさいてきプランよりも低価格で、3GBが月額858円（税込）から利用可能です。さらにかけ放題の割引もあるため、通話メインのシニア層にとってコストパフォーマンスが高いプランです。

### 5. 解約金・最低利用期間なし

契約に縛りがないため、合わなければいつでも無料で解約できます。「お試し感覚で使ってみたい」という方でも気軽に始められます。

## イオンモバイルのデメリット4つ

### 1. 昼時間帯の速度低下

MVNO共通の課題ですが、12時〜13時の昼休み時間帯は通信速度が大幅に低下します。実測で5〜6Mbps程度まで落ちるため、高画質動画の視聴や大容量データのダウンロードには向きません。

昼休みにスマホで動画をよく見る方は、UQモバイルやワイモバイルなどのサブブランド系を検討した方が良いかもしれません。

### 2. 支払いはクレジットカードのみ

口座振替やコンビニ払いには対応していません。クレジットカードを持っていない方は契約できないため、事前にカードを用意する必要があります。

### 3. 速度制限後は200kbps

データ容量を使い切った後の速度制限は200kbpsです。これはLINEのテキストメッセージ程度なら送受信できますが、Webページの読み込みすら厳しい速度です。mineoの「パケット放題Plus」（最大1.5Mbps）のような低速時の救済策がない点は弱みです。

### 4. キャンペーンが控えめ

他社と比較すると、大規模なキャッシュバックや端末割引キャンペーンが少ない傾向があります。初期費用を抑えたい方にとっては物足りなく感じるかもしれません。

## イオンモバイルへの乗り換え手順

### 店舗で申し込む場合

1. **最寄りのイオンモバイル取扱店舗を確認**（公式サイトの店舗一覧で検索）
2. **必要書類を準備** — 本人確認書類（運転免許証・マイナンバーカード等）、クレジットカード、MNP予約番号（他社からの乗り換え時）
3. **イオン店舗で申し込み** — スタッフと相談しながらプランを選び、その場で手続き完了
4. **SIM受け取り・初期設定** — 即日開通も可能（在庫状況による）

### オンラインで申し込む場合

1. **公式サイトから申し込みページにアクセス**
2. **プラン・SIMタイプを選択**
3. **本人確認書類をアップロード**
4. **SIMカードが郵送で届く**（通常2〜3日）
5. **APN設定を行い利用開始**

> 格安SIMの選び方で迷っている方は「[格安SIMの選び方ガイド](/kakuyasu-sim-erabikata-guide)」も合わせてお読みください。

## よくある質問（FAQ）

### Q. イオンモバイルの解約に違約金はかかる？

**A. かかりません。**最低利用期間も設定されていないため、いつでも無料で解約できます。

### Q. ドコモ回線とau回線、どちらを選ぶべき？

**A. 生活圏の電波状況で判断してください。**一般的にドコモ回線の方がエリアカバーが広いですが、地域によってはau回線の方が快適な場合もあります。イオン店舗で相談すれば、お住まいの地域に合った回線を案内してもらえます。

### Q. 端末はイオンモバイルで買える？

**A. はい、イオンモバイルでは各種スマートフォンを販売しています。**店舗で実機を確認してから購入できるのもメリットです。SIMフリー端末を持ち込んで使うことも可能です。

### Q. データ容量のプラン変更はいつでもできる？

**A. はい、毎月無料で変更できます。**マイページから手続きすれば、翌月1日から新プランが適用されます。

### Q. 5Gには対応している？

**A. 対応エリア内であれば5G通信が利用可能です。**ただし5G対応端末が必要です。
