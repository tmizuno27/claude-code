## 夫婦・カップルの格安SIM選びで重要な3つのポイント

### ポイント1：「2人合計の月額」で比較する

格安SIMの比較では1人あたりの月額料金に注目しがちですが、夫婦・カップルの場合は<span style="background: linear-gradient(transparent 60%, #fff3cd 60%)"><strong>「2人合計の月額料金」で比較するのが正解</strong></span>です。

例えば、1人あたりの料金はAサービスの方が安くても、Bサービスには2人目以降の家族割があるため、2人合計ではBサービスの方が安くなるケースがあります。

### ポイント2：家族割の「適用条件」を確認する

格安SIMの家族割には、サービスによって適用条件が異なります。

| 確認すべき項目 | 内容 |
|--------------|------|
| **家族の定義** | 血縁・婚姻のみ？同一住所ならOK？ |
| **同居が必要か** | 別居（単身赴任等）でも適用されるか |
| **割引対象** | 2回線目以降のみ？全回線に適用？ |
| **最大回線数** | 最大何回線まで割引されるか |
| **事実婚・カップル** | 未婚のカップルでも適用されるか |

<span style="background: linear-gradient(transparent 60%, #d6e4ff 60%)"><strong>同一住所であれば、結婚していないカップルやルームシェアの友人同士でも家族割が適用できるサービスが増えています</strong></span>。ワイモバイル、UQモバイル、楽天モバイルはいずれも事実婚やカップルにも対応しています。

### ポイント3：2人のデータ使用量を把握する

夫婦それぞれのデータ使用量を把握することで、最適なプランが選べます。

| タイプ | 月間データ使用量 | 特徴 |
|-------|----------------|------|
| ライトユーザー | 3GB以下 | LINE・メール中心。Wi-Fiがあれば十分 |
| ミドルユーザー | 5〜15GB | SNS・ニュース・軽い動画 |
| ヘビーユーザー | 20GB以上 | 動画・テザリング・外出先でも大量通信 |

夫婦で使用量が大きく異なる場合（例：夫20GB・妻3GB）は、別々のプランまたは別キャリアを選んだ方がお得になることもあります。

## 家族割 vs 個別契約｜夫婦はどちらがお得？

### 家族割のメリット

| メリット | 説明 |
|---------|------|
| **割引が毎月適用** | ワイモバイルなら2回線目以降1,100円/月、UQモバイルなら全員550円/月 |
| **管理がラク** | 同一キャリアなら請求をまとめられる |
| **データシェア** | IIJmioなら余ったデータを家族間でシェア可能 |
| **ポイント共有** | 楽天モバイルはポイント統合も可能 |

### 家族割のデメリット

| デメリット | 説明 |
|-----------|------|
| **柔軟性が低い** | 片方だけ解約・変更しにくい |
| **通信障害リスク** | 同一キャリアだと2人同時に使えなくなる |
| **プランの最適化** | 使用量が違う2人に同じキャリアが最適とは限らない |

### 個別契約（別キャリア）のメリット

| メリット | 説明 |
|---------|------|
| **通信障害リスク分散** | <span style="background: linear-gradient(transparent 60%, #d6e4ff 60%)"><strong>異なる回線なら、片方に障害が起きてももう片方は使える</strong></span> |
| **最適なプラン選択** | 各自の使い方に合ったベストなプランを個別に選べる |
| **乗り換えの自由度** | 片方だけの変更が容易 |

### 結論：どちらが正解？

<span style="background: linear-gradient(transparent 60%, #fff3cd 60%)"><strong>2人の使用量が似ているなら「家族割で同一キャリア」、使用量に差があるなら「別キャリアで最適化」がお得</strong></span>です。

また、通信障害のリスク分散を重視するなら、あえて別回線（ドコモ系＋au系など）にするのも賢い選択です。2021年にドコモ、2022年にau（KDDI）で大規模通信障害が発生したことを考えると、<span style="background: linear-gradient(transparent 60%, #ffd6e7 60%)"><strong>夫婦が同じ回線に依存するリスクは無視できません</strong></span>。


## データシェアの活用法｜夫婦で賢くギガを節約

### IIJmioのデータシェア

IIJmioでは、同一mioID（契約者名義）で複数回線を契約すると、回線間でデータ容量をプレゼント（シェア）できます。

**活用例**：夫20GBプラン、妻5GBプラン → 月末に夫が5GB余ったら妻にプレゼント

| 項目 | 内容 |
|------|------|
| シェア方法 | アプリから簡単にプレゼント操作 |
| 対象 | 毎月のデータ容量＋追加購入したデータ |
| 家族割引 | 同一mioIDで音声SIM 2回線以上 → 1回線あたり100円引き |
| 最大回線数 | 10回線 |

### mineoのパケットギフト＆パケットシェア

mineoでは、家族でグループを組んでパケットをシェアする「パケットシェア」機能があります。使い切れなかった容量は翌月に自動的に家族間で共有されます。

| 項目 | 内容 |
|------|------|
| パケットシェア | 余ったパケットを翌月に家族間で自動共有 |
| パケットギフト | 任意のタイミングで好きな容量を家族にプレゼント |
| 家族割引 | 1回線あたり55円/月引き（3回線目〜は165円引き） |
| 最大回線数 | 5回線 |

### データシェアが夫婦にとってお得な理由

<span style="background: linear-gradient(transparent 60%, #d6e4ff 60%)"><strong>データシェアの最大のメリットは「データを無駄にしない」こと</strong></span>です。

夫婦でそれぞれ10GBずつ契約している場合、片方が7GBしか使わず3GB余った月に、もう片方が12GB必要になったら、シェア機能で余ったデータを渡せます。この仕組みがあれば、両方とも余裕を持って15GBプランにする必要はなく、10GBプランのまま柔軟に運用できます。

## よくある質問（FAQ）

<details>
<summary><strong>Q1. 結婚していないカップルでも家族割は使えますか？</strong></summary>

はい、多くの格安SIMでは結婚していなくても家族割が適用できます。ワイモバイルは「同一住所」であれば別姓でも適用可能。UQモバイルも同一住所で家族セット割が使えます。楽天モバイルは同性パートナーや事実婚も家族割の対象です。同居していることが条件のサービスが多いため、住民票の住所が同じであることが重要です。
</details>

<details>
<summary><strong>Q2. 夫婦で別々の格安SIMにした場合、デメリットはありますか？</strong></summary>

主なデメリットは「家族割が適用されない」「請求が2社に分かれる」「データシェアができない」の3点です。ただし、通信障害時のリスク分散、各自に最適なプラン選択ができるメリットもあります。使用量に差がある夫婦や、リスク分散を重視する場合は、あえて別キャリアにするのも賢い選択です。
</details>

<details>
<summary><strong>Q3. 夫婦で同時に乗り換える必要がありますか？</strong></summary>

いいえ、別々のタイミングで乗り換えても問題ありません。ただし、同一キャリアで家族割を適用する場合は、2人目が契約した時点で家族割の申し込みが必要です。また、MNPの乗り換えキャンペーンは時期によって内容が変わるため、<span style="background: linear-gradient(transparent 60%, #fff3cd 60%)"><strong>できれば同じキャンペーン期間中にまとめて乗り換えるのがお得</strong></span>です。
</details>

<details>
<summary><strong>Q4. 今のスマホ端末はそのまま使えますか？</strong></summary>

ほとんどの場合、今使っているスマホはそのまま格安SIMで使えます。ただし、SIMロックがかかっている端末は事前にSIMロック解除が必要です（各キャリアのマイページから無料で手続き可能）。2021年10月以降に購入した端末はSIMロックフリーなので、そのまま使えます。
</details>

<details>
<summary><strong>Q5. 通話が多い夫婦はどのプランがおすすめですか？</strong></summary>

<span style="background: linear-gradient(transparent 60%, #d6e4ff 60%)"><strong>通話が多い夫婦には楽天モバイルが最強</strong></span>です。Rakuten Linkアプリを使えば国内通話が完全無料。ahamoは5分以内の通話が無料で標準付帯されています。ワイモバイルやUQモバイルはかけ放題オプション（月額1,870円〜）を追加できます。夫婦間の通話が多い場合は、LINEの無料通話で代替するのも一つの手です。
</details>

<details>
<summary><strong>Q6. 子どもができた場合、3人以上の家族割に拡張できますか？</strong></summary>

はい、主要な格安SIMの家族割はすべて3回線以上に拡張可能です。ワイモバイル（最大9回線）、UQモバイル（最大10回線）、IIJmio（最大10回線）、楽天モバイル（最大20回線）と、十分な回線数に対応しています。将来的に家族が増えることを見越してキャリアを選ぶのも賢い判断です。
</details>

<details>
<summary><strong>Q7. 夫婦で格安SIMに乗り換える際の注意点は？</strong></summary>

最も注意すべきは「キャリアメール」です。ドコモ・au・ソフトバンクのキャリアメール（@docomo.ne.jp等）は、格安SIMに乗り換えると使えなくなります（有料の「メール持ち運び」サービスで継続可能：月額330円）。銀行やECサイトの登録メールアドレスをGmail等に事前に変更しておきましょう。また、2人同時に乗り換えると、開通手続き中に一時的に2人とも電話が使えなくなる場合があるので、日をずらすか、開通タイミングを調整するのがベターです。
</details>

<details>
<summary><strong>Q8. 単身赴任中の夫婦でも家族割は使えますか？</strong></summary>

サービスによります。ワイモバイルは別居でも血縁・婚姻関係が証明できれば家族割が適用可能です。UQモバイルも「契約者が50歳以上の場合」に限り、別住所でも家族関係の証明書があれば適用されます。楽天モバイルは招待リンクで家族グループに参加する仕組みのため、住所に関わらず適用可能です。IIJmioは同一mioID（同一名義）での契約が条件のため、住所は問いません。
</details>
