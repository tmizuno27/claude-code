## デザインコンセプト

**「信頼感 × 比較のしやすさ」** — Apple風のクリーンなデザインをベースに、
比較表・CTA・料金情報が一目で分かるUI特化の設計。

## コンポーネント設計

### ヘッダー（frosted glass）
- `backdrop-filter: blur(20px)`, `background: rgba(255,255,255,0.92)`
- ロゴ左 + ナビ右
- ナビ: 用途別 | 料金比較 | レビュー | 乗り換え | 海外SIM

### 比較表（SIM比較サイト特有）
```
┌─────────────────────────────────────────────┐
│  キャリア名  │ 月額料金 │ データ量 │ 速度 │ 公式  │
├─────────────────────────────────────────────┤
│ ★おすすめ    │          │          │      │ [CTA] │
│ UQモバイル   │ 1,628円  │ 4GB      │ ◎    │ 詳細  │
├─────────────────────────────────────────────┤
│ 楽天モバイル │ 1,078円  │ 3GB      │ ○    │ 詳細  │
└─────────────────────────────────────────────┘
```
- 行ホバーで薄いハイライト
- 「おすすめ」行に薄い背景色（#E8F4FD）
- CTAボタンは各行に配置
- スマホ: カード型に変形（テーブル → スタック）

### トップページ構成
1. **ヒーロー**: 「あなたに最適な格安SIMが見つかる」+ 簡易診断CTA
2. **用途別クイックナビ**: 6枚カード（アイコン+テキスト）
   - 👶 子供向け | 👴 シニア向け | 📱 副回線 | 📞 通話のみ | 📊 大容量 | ✈️ 海外旅行
3. **人気キャリアTOP5**: ランキング形式
4. **最新記事**: 新着5件
5. **カテゴリ一覧**: 5カテゴリ導線

### 記事ページ構成
- 2カラム: コンテンツ（左）+ サイドバー（右, 336px）
- サイドバー: 検索 → 人気記事 → おすすめSIM TOP3 → 目次（sticky）
- 比較表は記事幅いっぱいに表示

### 視覚的強調（nambei-oyaji.comと同じルール）
- 黄マーカー（#fff3cd）: 結論・おすすめ・料金
- ピンクマーカー（#ffd6e7）: 注意点・デメリット
- 青マーカー（#d6e4ff）: メリット・お得情報
- 1記事あたり5-10箇所

## レスポンシブ

| ブレークポイント | 変更点 |
|---------------|--------|
| 1024px以下 | サイドバー非表示 |
| 768px以下 | ナビ → ハンバーガー、比較表 → カード型 |
| 600px以下 | フォントサイズ調整、余白縮小 |

## CMS: WordPress + Cocoon（推奨）

nambei-oyaji.comと同じCocoonテーマを使用し、カスタムCSSで上書き。
- `!important` によるCocoonデフォルト上書きが必要
- ウィジェット（custom_html）でヘッダー・フッターのHTML/CSS/JSを注入
- 比較表はカスタムHTMLブロックで作成
