## URL構造

```
sim-hikaku.online/
├── /                              ← トップページ（比較ツール + 最新記事）
├── /category/
│   ├── /yoto-betsu/               ← 用途別おすすめ
│   ├── /norikae/                  ← 乗り換え・手続きガイド
│   ├── /ryokin-hikaku/            ← 料金比較
│   ├── /review/                   ← 個別キャリアレビュー
│   └── /kaigai-sim/              ← 海外SIM・eSIM
├── /about/                        ← 運営者情報
├── /privacy-policy/               ← プライバシーポリシー
└── /sitemap.xml                   ← サイトマップ
```

## WordPressカテゴリ設定

| カテゴリ名 | スラッグ | 説明 |
|-----------|---------|------|
| 用途別おすすめ | yoto-betsu | ターゲット別・用途別の格安SIM比較 |
| 乗り換えガイド | norikae | MNP・乗り換え手順の解説 |

| 海外SIM・eSIM | kaigai-sim | 海外旅行・留学・在住者向けSIM情報 |
