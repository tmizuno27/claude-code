## 4. CTR改善戦略

### 即効性のある施策

1. **タイトルに数字を入れる**: 費用記事に具体金額（月8万円、初期50万円等）
2. **疑問形タイトル**: 「危険？」「本当に稼げる？」「どうする？」で検索意図マッチ
3. **ネガティブKW活用**: 「失敗」「後悔」「デメリット」「落とし穴」は高CTR
4. **年号を最新に**: 全記事のタイトルを「2026年」に統一更新
5. **「在住者が」を活用**: E-E-A-Tシグナルかつ差別化要素

### 中期施策（1-2週間）

1. **治安記事の統合**: 治安記事が5本（#5,#58,#62,#63,#67,#69）と重複過多。最も強い1本に統合し、残りは301リダイレクト
2. **海外送金記事の統合**: 送金比較記事も3本（#14,#62,#64,#68）重複。メインの#14に集約
3. **FAQ構造化データ追加**: 全キラー記事にFAQスキーマを追加してリッチスニペット獲得
4. **パンくずリスト最適化**: 「ホーム > パラグアイ移住 > 費用」の構造でサイトリンク獲得
