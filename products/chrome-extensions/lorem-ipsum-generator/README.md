# Lorem Ipsum Generator

A Chrome extension for generating placeholder text quickly. Supports paragraphs, sentences, words, and lists.

## Features

- **Multiple Formats**: Paragraphs, sentences, words, or lists
- **Adjustable Quantity**: 1 to 100 units
- **One-click Copy**: Copy generated text to clipboard instantly
- **Fully Offline**: No internet needed
- **Clean UI**: Minimal, fast interface

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Runs entirely locally. No data collected or transmitted.

## Tech Stack

- Manifest V3
- Pure JavaScript (no dependencies)
