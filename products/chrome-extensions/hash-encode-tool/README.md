# Hash & Encode Tool

A Chrome extension for hashing, encoding, and decoding text. MD5, SHA-1, SHA-256, SHA-512, Base64, URL encoding, HTML entities, and JWT decode.

## Features

- **Hash**: MD5, SHA-1, SHA-256, SHA-512
- **Encode/Decode**: Base64, URL encoding, HTML entities
- **JWT Decoder**: Decode JWT tokens instantly
- **Context Menu**: Right-click selected text to hash
- **One-click Copy**: Copy results to clipboard
- **100% Local**: Uses Web Crypto API, zero network requests

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

All processing happens locally using Web Crypto API and built-in JavaScript functions. No data transmitted externally.

## Tech Stack

- Manifest V3
- Web Crypto API
- Chrome Context Menus API
