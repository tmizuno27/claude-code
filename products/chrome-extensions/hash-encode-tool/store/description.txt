Hash & Encode Tool — A Developer Utility for Hashing and Encoding

A lightweight browser tool for developers who need to quickly hash, encode, or decode text. Everything runs locally — no data leaves your browser.

WHAT IT DOES:
This extension provides common hashing algorithms (such as MD5, SHA-1, SHA-256, and SHA-512) alongside encoding and decoding tools (Base64, URL encoding, HTML entities) and a JWT token decoder — all in one popup.

HOW TO USE:
1. Click the extension icon or right-click selected text on any page
2. Enter or paste the text you want to process
3. Choose the operation you need
4. Copy the result with one click

PRIVACY & SECURITY:
All processing happens locally using the Web Crypto API and built-in browser functions. This extension makes zero network requests and never transmits your data anywhere.

Built for developers, DevOps engineers, and security professionals who want a fast, private utility always one click away.
