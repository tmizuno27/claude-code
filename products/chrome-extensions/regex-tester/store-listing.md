**Regex Tester — Real-Time Pattern Matching in Your Browser**

Stop switching between your code editor and regex101. Test regular expressions instantly without leaving your browser tab.

- Matches highlight as you type — zero delay
- See match count and execution time instantly
- No "Run" button needed, results are live

- Every capture group displayed with its index
- Instantly see what your parentheses are grabbing
- Debug complex patterns in seconds

🔧 Full Flag Support
- Global (g), Case-insensitive (i), Multiline (m)

- Toggle flags with one click

📦 Built-In Presets
- Email validation
- URL detection
- Phone numbers
- IP addresses
- And more — no Googling required

💾 Save Your Own Patterns
- Save custom patterns you use frequently
- Load them instantly with one click
- Your personal regex library, always accessible

🔒 100% Privacy
- All processing happens locally in your browser
- Zero network requests — works completely offline
- No analytics, no tracking, no data collection

**PERFECT FOR:**
- Frontend & backend developers debugging regex in code
- QA engineers testing input validation patterns
- Data engineers building extraction patterns
- Students learning regular expressions

1. Click the extension icon
