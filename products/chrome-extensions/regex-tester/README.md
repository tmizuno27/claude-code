# Regex Tester

A Chrome extension for testing regular expressions directly in your browser. Enter a pattern and test string to see matches highlighted in real time.

## Features

- **Real-time Highlighting**: Matches appear as you type
- **Capture Groups**: View captured groups with their index
- **Flags**: Global, case-insensitive, multiline, dotAll, unicode
- **Presets**: Built-in patterns for email, URL, phone number, and more
- **Save Patterns**: Save your own regex patterns for later use
- **Performance**: Shows match count and execution time
- **Fully Offline**: No data sent to any server

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

All processing happens locally in your browser. Zero network requests.

## Tech Stack

- Manifest V3
- Chrome Storage API
- Pure JavaScript regex engine (browser-native)
