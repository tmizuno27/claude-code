Test regular expressions directly in your browser. Enter a pattern and a test string to see matches highlighted instantly.

Features:
- Matches are highlighted as you type
- Capture groups are displayed with their index
- Supports flags: global, case-insensitive, multiline, dotAll, unicode
- Includes preset patterns for common use cases (email, URL, phone number)
- Shows match count and execution time
- Save your own patterns for later use

How it works:
1. Click the extension icon to open the popup
2. Enter your regex pattern and test string
3. Matches appear highlighted in the text below

Privacy:
All processing happens locally in your browser. No data is sent to any server.
