# Color Picker & Converter

A Chrome extension for picking colors from any webpage, converting between formats, checking WCAG contrast, and generating palettes.

## Features

- **Eyedropper**: Pick any color from a webpage
- **Format Conversion**: HEX, RGB, HSL, CMYK -- instant conversion
- **WCAG Contrast Checker**: Verify accessibility compliance
- **Palette Generator**: Complementary, analogous, triadic palettes
- **Color History**: Recently picked colors saved locally
- **One-click Copy**: Copy any format to clipboard

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Everything processed locally. No data collected, transmitted, or stored externally.

## Tech Stack

- Manifest V3
- EyeDropper API
- Chrome Storage API
