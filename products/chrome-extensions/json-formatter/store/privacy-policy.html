Privacy Policy — JSON Formatter Pro

Last updated: March 16, 2026

Overview

JSON Formatter Pro is a browser extension that formats and highlights JSON content. Your privacy is important to us. This policy explains what data the extension accesses and how it is handled.

Data Collection

We do not collect any data. JSON Formatter Pro processes all JSON content entirely within your browser. No data of any kind is transmitted to any external server, API, or third party.

Data Storage

The extension stores only your user preferences (theme and indentation settings) using the browser's local storage API (chrome.storage.local). This data never leaves your device.

Permissions

  • activeTab — Used to read the current page content to detect and format JSON.
  • storage — Used to save your theme and indentation preferences locally.

Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.
