# Chrome Web Store Listing — JSON Formatter Pro

## Short Description (132 chars max)
Auto-format JSON responses with syntax highlighting, collapsible tree view, JSON path copy & dark mode. 100% local processing.

## Full Description

**JSON Formatter Pro — Beautiful JSON in Your Browser, Automatically**

Open a JSON API endpoint and see it formatted instantly. No copy-pasting into external tools. No manual triggering. Just clean, readable, color-coded JSON.

**AUTO-DETECTION:**

JSON Formatter Pro automatically detects JSON content on any page and formats it in place. Navigate to an API endpoint and the response is already formatted. Zero clicks required.

**SYNTAX HIGHLIGHTING:**

- Strings → green
- Numbers → blue
- Booleans → orange
- Null → red
- Keys → distinct color
- Instantly scan large payloads and find what you need

**COLLAPSIBLE TREE VIEW:**

- Click any object or array to collapse/expand
- Navigate deeply nested JSON without scrolling endlessly
- Fold entire sections to focus on what matters
- Perfect for large API responses (1,000+ lines)

**JSON PATH:**

- Hover over any value to see its full path: `$.data[0].user.name`
- Click to copy the path to your clipboard
- Paste directly into your code — no manual path counting

**DARK & LIGHT THEMES:**

- Beautiful dark mode (default) — easy on the eyes
- Light mode available
- Preference saved automatically

**CUSTOMIZABLE INDENTATION:**

- 2 spaces, 4 spaces, or tab
- Settings persist across sessions

**100% LOCAL PROCESSING:**

- All formatting happens in your browser
- Zero data sent to external servers
- No analytics, no tracking
- Works offline after page load

**PERFECT FOR:**

- **API developers** debugging REST endpoints
- **Frontend developers** inspecting API responses
- **QA engineers** verifying JSON payloads
- **DevOps engineers** checking config responses
- **Anyone** who reads JSON regularly

**HOW TO USE:**

1. Install the extension
2. Navigate to any JSON URL
3. It's already formatted — enjoy

No setup. No configuration. No API key. Just formatted JSON.

**COMPARED TO OTHER JSON FORMATTERS:**

- Auto-detects JSON (no button click needed)
- JSON path on hover (most formatters don't have this)
- Collapsible tree (not just colored text)
- Dark mode built-in
- Zero permissions beyond what's needed
