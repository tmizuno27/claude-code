# JSON Formatter Pro

A Chrome extension that auto-detects and formats JSON responses with syntax highlighting, tree view, and JSON path display.

## Features

- **Auto-Detection**: Automatically formats JSON content on any page
- **Syntax Highlighting**: Color-coded strings, numbers, booleans, nulls, keys
- **Tree View**: Collapsible/expandable nested structure
- **JSON Path**: Hover any value to see its path; click to copy
- **Dark and Light Themes**: Preference saved automatically
- **Custom Indentation**: 2 spaces, 4 spaces, or tabs
- **100% Local**: No data ever sent to any server

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

All formatting happens in your browser. No data collection, no tracking.

## Tech Stack

- Manifest V3
- Content Script (auto-detection)
- Chrome Storage API
