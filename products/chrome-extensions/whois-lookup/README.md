# Domain WHOIS Lookup

A Chrome extension for instant WHOIS registration data, DNS records, and SSL certificate information.

## Features

- **One-click WHOIS**: Registrar, creation/expiry dates, nameservers
- **DNS Records**: A, AAAA, MX, NS, TXT, CNAME, SOA
- **SSL Certificate**: Validity, issuer, subject, chain details
- **Lookup History**: Recent lookups saved locally
- **One-click Copy**: Copy all WHOIS/DNS/SSL data at once
- **Auto-detect Domain**: Reads current tab domain automatically

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Only sends domain name to lookup APIs. No personal data collected. History stored locally only.

## Tech Stack

- Manifest V3
- WHOIS/DNS/SSL lookup APIs
- Chrome Storage API
