Privacy Policy — AI Text Rewriter

Developer: t.mizuno27

Last updated: March 16, 2026

Data Collection

This extension sends user-selected text to the OpenAI API for processing. The extension uses a Bring Your Own Key (BYOK) model — your OpenAI API key is stored locally in Chrome storage and is never transmitted to any server other than OpenAI API endpoints.

Data Usage

Text you select is sent to OpenAI solely for the purpose of rewriting. The developer does not collect, store, or have access to any of your data, API keys, or browsing activity.

Third-Party Services

This extension communicates with the OpenAI API (api.openai.com). OpenAI privacy policy applies: https://openai.com/privacy

Data Storage

Your OpenAI API key and preferences are stored locally using Chrome storage API. No data is stored on external servers by the developer.

Contact

For questions about this privacy policy, contact: t.mizuno27 via the Chrome Web Store developer page.
