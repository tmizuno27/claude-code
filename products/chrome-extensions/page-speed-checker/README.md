# Page Speed Checker

A Chrome extension for one-click PageSpeed Insights scores and Core Web Vitals analysis.

## Features

- **One-click Analysis**: Check current page performance instantly
- **Lighthouse Scores**: Performance, Accessibility, Best Practices, SEO
- **Core Web Vitals**: LCP, FID/INP, CLS
- **Mobile and Desktop**: Results for both viewports
- **No API Key**: Uses Google free public PageSpeed Insights API
- **Clean Dashboard**: Easy-to-read score display

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Current page URL is sent to Google PageSpeed Insights API. No other data collected.

## Tech Stack

- Manifest V3
- Google PageSpeed Insights API (public, free)
