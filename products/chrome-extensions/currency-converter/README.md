# Quick Currency Converter

A Chrome extension for instant currency conversion with real-time exchange rates. Supports 160+ currencies.

## Features

- **Real-time Rates**: Live exchange rates from reliable API sources
- **160+ Currencies**: Major, exotic, and crypto currencies
- **One-click Swap**: Instantly swap from/to currencies
- **Quick Rates Dashboard**: Monitor popular pairs at a glance
- **Favorites**: Save frequently used currency pairs
- **Right-click Conversion**: Select a number on any page, right-click, convert
- **Remembers Preferences**: Last-used pair saved automatically

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Only sends currency pair to exchange rate API. No personal data collected.

## Tech Stack

- Manifest V3
- Exchange Rate API
- Chrome Storage API, Context Menus API
