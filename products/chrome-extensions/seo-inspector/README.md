# SEO Inspector - Instant SEO Analysis

A Chrome extension for one-click on-page SEO analysis. Get a 0-100 score, a 20-point checklist, and actionable recommendations.

## Features

- **SEO Score**: 0-100 based on 20 weighted checks
- **7 Categories**: Title, meta description, headings, images, links, OG tags, technical
- **Actionable Recommendations**: Pass/fail/warning with specific fixes
- **Dark Mode UI**: Color-coded scores, expandable sections
- **Free Forever**: No ads, no account required

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore).

## Privacy

Only sends the URL you choose to analyze. No personal data collected, no cookies, no tracking.

## Tech Stack

- Manifest V3
- Content analysis via DOM inspection
- activeTab permission only
