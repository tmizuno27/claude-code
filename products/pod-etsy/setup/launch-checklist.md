# AsuInk 立ち上げチェックリスト

## Phase 1: アカウント開設（所要時間: 1時間）

- [ ] **Etsyアカウント作成**
  - https://www.etsy.com/sell にアクセス
  - 「Open your Etsy shop」をクリック
  - メールアドレス・パスワード設定
  - Shop preferences: Language=English, Country=Paraguay, Currency=USD

- [ ] **Etsyショップ設定**
  - Shop name: **AsuInk**
  - Shop icon: ロゴ画像（500×500px）をアップロード
  - Shop banner: ヘッダー画像（3360×840px）をアップロード
  - About section: ストア説明文を記入
  - Shop policies: 返品・発送ポリシーを設定（Printful準拠）

- [ ] **Printfulアカウント作成**
  - https://www.printful.com にアクセス
  - 無料アカウント登録

- [ ] **Printful × Etsy 連携**
  - Printful Dashboard → Stores → Add Store → Etsy
  - Etsyの認証画面でアクセス許可
  - 連携完了を確認

- [ ] **決済設定**
  - Etsy Payment: 銀行口座またはPayPal登録
  - Printful: クレジットカード登録（製造費の支払い用）

## Phase 2: デザイン制作（所要時間: 3-5時間）

- [ ] **Geminiでデザイン生成**
  - `designs/prompts/` のプロンプトを使用
  - 各ニッチ10デザイン × 5ニッチ = 50デザイン
  - 出力形式: PNG, 透過背景, 300DPI

- [ ] **Tシャツ用デザイン調整**（50枚）
  - サイズ: 4500×5400px
  - 背景: 透過
  - ファイル名: `{niche}-{number}-tshirt.png`

- [ ] **マグカップ用デザイン調整**（50枚）
  - サイズ: 2850×1074px
  - 同じデザインを横長にリサイズ/再配置
  - ファイル名: `{niche}-{number}-mug.png`

- [ ] **ポスター用デザイン調整**（50枚）
  - サイズ: 5400×7200px
  - 縦長レイアウトに調整
  - ファイル名: `{niche}-{number}-poster.png`

  - Product: Enhanced Matte Paper Poster 18×24
  - デザインをアップロードし配置を確認

## Phase 4: Etsyリスティング作成（所要時間: 3-4時間）

- [ ] **Printful同期で自動リスティング作成**
  - Printful商品がEtsyに自動同期されることを確認

- [ ] **リスティング情報を更新**（`listings/` ファイル参照）
  - タイトル: SEO最適化済みタイトルに変更
  - タグ: 13個のロングテールKWを設定
  - 説明文: テンプレートから貼り付け
  - 価格: Tシャツ$24.99、マグ$17.99、ポスター$19.99
  - カテゴリ: 適切なEtsyカテゴリを選択
  - 画像: モックアップ画像（Printful自動生成）を確認

- [ ] **SEOチェック**
  - 全150リスティングのタイトルに主要KWが含まれているか
  - タグが13個フルに設定されているか
  - 説明文の最初の2行にKWが含まれているか

## Phase 5: ストア最終確認（所要時間: 30分）

- [ ] **ショップ外観チェック**
  - ロゴ・バナーが正しく表示されるか
  - About ページが完成しているか
  - ポリシーページが設定されているか

- [ ] **商品ページチェック**（ランダムに10商品確認）
  - モックアップ画像がきれいに表示されるか
  - 価格が正しいか
  - 説明文が正しく表示されるか
  - バリエーション（サイズ・色）が正しいか

- [ ] **テスト注文**
  - 自分で1商品を注文してみる（品質確認用）
  - Printfulでの処理フローを確認
