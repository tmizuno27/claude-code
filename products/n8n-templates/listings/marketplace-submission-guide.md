# n8n Workflow Template Marketplace Submission Guide

Last updated: 2026-03-15

---

## 1. HaveWorkflow (haveworkflow.com)

### Overview
n8n/Zapier/Make向けワークフローテンプレートの専門マーケットプレイス。クリエイター向けページも用意されている。

### 出品手順
1. **haveworkflow.com/share-workflow/** にアクセス
2. アカウント作成（メールアドレスで登録）
3. n8nでワークフローを作成し、**JSONファイルとしてエクスポート**
4. 以下の情報を入力して投稿:
   - ワークフロー名
   - 詳細な説明文（使い方・ユースケース）
   - カテゴリ選択
   - JSONファイルのアップロード
5. レビュー後、マーケットプレイスに掲載

### 料金モデル
- テンプレートの価格は出品者が設定（ユーロ建て）
- 無料テンプレートも掲載可能
- コミッション率の具体的な数値は非公開（要確認）

### 特徴
- クリエイター専用ページが付与される（ポートフォリオとして活用可能）
- コミュニティフォーラムあり（community.haveworkflow.com）
- カスタムワークフロー開発の依頼も受けられる

### 出品のコツ
- 説明文を詳細に書く（どんな課題を解決するか）
- ノードにラベル・コメントを付けて可読性を上げる
- エラーハンドリングを組み込む

## 2. n8nMarket (n8nmarket.com)

### Overview
n8n/Zapier/Make向けのプレミアム自動化マーケットプレイス。テンプレート販売に加え、カスタム自動化プロジェクトの依頼も扱う。

### 出品手順
1. **n8nmarket.com/login** からアカウント作成
2. セラーとして登録（詳細な登録要件はサイト上で確認が必要）
3. テンプレートをアップロード（JSON + 説明文 + カテゴリ）
4. 価格を設定して公開

### 料金モデル
- 「プレミアムレート」でセラーに報酬が支払われると記載
- 具体的なコミッション率は非公開
- 価格ページ（n8nmarket.com/pricing）に詳細あり（要直接確認）

### カテゴリ例
- Marketing, CRM, Sales, Lead Generation, AI/ChatGPT, etc.

### 特徴
- セラー専用プロフィールページあり（例: n8nmarket.com/sellers/104）
- ニュースレター・ブログでテンプレートが紹介される可能性あり
- カスタムプロジェクト依頼の受注も可能

## 3. その他のマーケットプレイス

### 3a. n8n公式テンプレートライブラリ (n8n.io/workflows/)

**無料掲載 + アフィリエイト報酬**

- **掲載**: n8n.io/workflows/ に無料で投稿可能
- **収益化**: n8n Cloud アフィリエイトプログラム（n8n.io/affiliates/）と組み合わせ
  - **報酬**: n8n Cloud契約の初年度売上の **30%** がコミッション
  - テンプレートURLにアフィリエイトID付与: `?ref=yourUniqueId&utm_source=affiliate`
  - **最低支払額**: EUR 100（月1回PayPal払い）
- **制限**: 有料広告でのアフィリエイトリンク宣伝は禁止
- **メリット**: 公式サイトなのでSEO・信頼性が高い。8,500+テンプレートが掲載中

### 3b. ManageN8N Marketplace (managen8n.com/features/marketplace)

- n8nワークフローの売買プラットフォーム
- バージョン管理機能付き（更新・ロールバック対応）
- ダウンロード・デプロイメントの詳細メトリクス提供
- 組織向けプライベートライブラリ機能あり
- コミッション率は非公開（要直接確認）

### 3c. Gumroad (gumroad.com)

**汎用デジタルコンテンツ販売プラットフォーム**

- **手数料**: 売上の10%（Gumroad手数料）+ 決済手数料
- **メリット**: 自由な価格設定、バンドル販売可能、即日開始
- **実績**: n8nテンプレートの大量バンドル（4,000〜8,000+個）が多数販売中
- **出品方法**: JSONファイルをデジタル商品として登録するだけ
- **注意**: n8n専門ではないので、自力で集客する必要あり

### 3d. Gumroad以外の汎用プラットフォーム

| プラットフォーム | 手数料 | 特徴 |
|---|---|---|
| **LemonSqueezy** | 5% + 決済手数料 | Gumroad代替、EU VAT自動処理 |
| **Payhip** | 5%（無料プラン） | シンプル、直接販売向き |
| **自社サイト（Stripe連携）** | Stripe手数料のみ(3.6%) | マージン最大、ブランド構築可 |

## 出品優先順位（推奨）

1. **n8n公式** (n8n.io/workflows/) — 無料掲載 + アフィリエイト30%。まず最初にここ
2. **HaveWorkflow** — n8n専門マーケットプレイス。クリエイターページで認知度向上
3. **n8nMarket** — プレミアム路線。高単価テンプレート向き
4. **Gumroad** — バンドル販売やセルフプロモーション用。自由度が高い
5. **ManageN8N** — 組織向け・エンタープライズ寄りの需要がある場合

## 注意事項

- コミッション率が非公開のプラットフォーム（HaveWorkflow, n8nMarket, ManageN8N）は、出品前にサポートに直接問い合わせて確認すること
- n8n公式テンプレートライブラリは無料テンプレートのみ（直接販売は不可、アフィリエイトで間接収益化）
- Gumroadは集客を自力で行う必要があるが、価格・販売条件の自由度が最も高い
