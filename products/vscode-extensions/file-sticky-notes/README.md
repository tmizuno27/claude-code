# File Sticky Notes

Attach virtual sticky notes to any file or line. Notes persist across sessions with color-coded priorities.

## Features

- Add colored notes (yellow/green/blue/pink) to any line
- TreeView sidebar grouped by file
- Gutter decorations and hover tooltips
- Search and filter notes by text or color
- Export/import to .vscode/sticky-notes.json
- Workspace-persistent storage

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
