# Commit Crafter

Generate structured, conventional commit messages with templates and smart analysis of staged changes.

## Features

- Conventional Commit type selection with smart detection
- Auto-detect scope from changed file paths
- Gitmoji emoji prefix support
- Fill VS Code SCM input box directly
- Custom message templates
- Subject length validation

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
