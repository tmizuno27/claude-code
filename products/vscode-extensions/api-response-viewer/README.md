# API Response Viewer

Visualize JSON API responses in a collapsible tree view with search, filter, copy path, and comparison.

## Features

- Collapsible tree view for JSON data
- Type indicators (string/number/boolean/null/array/object)
- Copy JSON path or value from any node
- Search and filter by key or value
- Compare two responses side by side
- Load from selection, clipboard, or file

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
