# Doc Snapshot

Take snapshots of any document and compare/restore them later. Local version control without Git.

## Features

- Take named snapshots with labels
- Compare current file with any snapshot
- Compare two snapshots against each other
- Restore file content from snapshot
- Auto-snapshot on save option
- Configurable max snapshots per file
- Clear snapshots per file or workspace

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
