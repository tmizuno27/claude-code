## Prompt Examples (3 of 50+)

### Prompt #1: Customer Inquiry Auto-Responder

```
You are a customer service specialist for [BUSINESS_NAME], a [BUSINESS_TYPE].

I received this customer inquiry:
---
[PASTE CUSTOMER MESSAGE]
---

Write a professional, friendly response that:
1. Acknowledges their specific concern
2. Provides a clear, actionable answer
3. Offers additional help if needed
4. Maintains our brand voice: [BRAND_VOICE — e.g., "professional but warm, like a knowledgeable friend"]

Our key policies:
- [POLICY_1 — e.g., "30-day money-back guarantee"]
- [POLICY_2 — e.g., "Free shipping over $50"]
- [POLICY_3 — e.g., "Support hours: 9am-6pm EST"]

If you cannot fully answer from the information provided, flag what you need and draft the best possible response with what you have.
```

### Prompt #2: 30-Day Social Media Calendar

```
Create a 30-day social media content calendar for [BUSINESS_NAME].

Business details:
- Industry: [INDUSTRY]
- Target audience: [AUDIENCE]
- Platforms: [PLATFORMS — e.g., "Twitter, LinkedIn, Instagram"]
- Content pillars: [PILLAR_1], [PILLAR_2], [PILLAR_3]
- Posting frequency: [FREQUENCY — e.g., "1x daily on Twitter, 3x weekly on LinkedIn"]

For each post, provide:
1. Date & platform
2. Content type (educational / entertaining / promotional / engagement)
3. Full post text (ready to copy-paste)
4. Suggested hashtags (5-7)
5. Best posting time
6. CTA (call to action)

Rules:
- Max 20% promotional content
- Include trending topic hooks where relevant
- Vary content types to prevent fatigue
- Include engagement posts (polls, questions) at least 2x per week
```

### Prompt #3: SOP Writer

```
Create a detailed Standard Operating Procedure (SOP) for the following business process:

Process name: [PROCESS_NAME]
Department: [DEPARTMENT]
Process owner: [OWNER_NAME]
Current frequency: [FREQUENCY — e.g., "daily", "weekly", "per customer order"]

Describe the process in plain language:
[DESCRIBE WHAT HAPPENS FROM START TO FINISH]

Tools/systems used:
- [TOOL_1]
- [TOOL_2]

Generate the SOP with:
1. Purpose & scope
2. Roles & responsibilities
3. Step-by-step instructions (numbered, with screenshots placeholders)
4. Decision points (if X, then Y)
5. Quality checkpoints
6. Common errors & troubleshooting
7. Revision history table

Format as a clean, printable document with clear headers.
```
