# AsuInk 事業計画書

## 1. 事業概要

- **事業名**: AsuInk（アスインク）
- **事業形態**: Print-on-Demand（受注生産型EC）
- **販売チャネル**: Etsy
- **製造パートナー**: Printful
- **開始時期**: 2026年3月

## 2. ビジネスモデル

在庫を一切持たず、Etsyで注文が入るとPrintfulが自動で印刷・梱包・発送する完全委託型。

```
顧客 → Etsy注文 → Printful製造 → 顧客へ直送
                ↑
          AsuInk（デザイン提供のみ）
```
