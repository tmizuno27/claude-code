# デザイン仕様書 — AsuInk

## 共通設定

- **カラーモード**: sRGB
- **ファイル形式**: PNG（透過背景対応）
- **DPI**: 300
- **最大ファイルサイズ**: 200MB（Printful上限）

---

## Tシャツ（Unisex Bella+Canvas 3001）

| 項目 | 値 |
|------|-----|
| プリントエリア（前面） | 12.5" × 15" |
| 推奨デザインサイズ | **4500 × 5400 px** |
| DPI | 300 |
| セーフゾーン | 端から0.25"（75px）内側 |
| 背景 | **透過必須**（PNG） |
| プリント方式 | DTG（Direct-to-Garment） |

### 注意事項
- 白インクは別レイヤー不要（Printfulが自動処理）
- ダークカラーTシャツ用デザインは明るい色を使用
- 極細線（1px以下）は印刷で潰れるため避ける

---

## マグカップ（White Glossy 11oz）

| 項目 | 値 |
|------|-----|
| プリントエリア（全面ラップ） | 9.5" × 3.58" |
| 推奨デザインサイズ | **2850 × 1074 px** |
| DPI | 300 |
| セーフゾーン | 上下左右0.2"（60px）内側 |
| 背景 | 白または任意の色（不透明OK） |
| ハンドル位置 | デザイン中央から左右にラップ |

### 注意事項
- 取っ手の裏側（約1"幅）にはデザインが印刷されない
- 重要な要素は中央3"以内に配置
- 食洗機対応だがハンドウォッシュ推奨と記載

---

## ポスター（Enhanced Matte Paper）

| 項目 | 値 |
|------|-----|
| サイズ | 18" × 24" |
| 推奨デザインサイズ | **5400 × 7200 px** |
| DPI | 300 |
| ブリード（裁ち落とし） | 不要（フチなし印刷なし） |
| セーフゾーン | 端から0.25"（75px）内側 |
| 背景 | 任意（不透明OK） |
| 用紙 | 189 gsm / 7.5 mil Enhanced Matte |

### 注意事項
- フレームなしで販売（フレームは顧客側で用意）
- 白背景の場合、用紙の白と一体化する
- テキスト主体のデザインは最小フォントサイズ12pt以上

---

## Printful アップロード要件まとめ

| 項目 | Tシャツ | マグカップ | ポスター |
|------|---------|----------|---------|
| 幅 (px) | 4500 | 2850 | 5400 |
| 高さ (px) | 5400 | 1074 | 7200 |
| DPI | 300 | 300 | 300 |
| 形式 | PNG | PNG | PNG/JPG |
| 背景 | 透過 | 任意 | 任意 |
| カラーモード | sRGB | sRGB | sRGB |
| Max | 200MB | 200MB | 200MB |

## ファイル命名規則

```
{niche}-{number}-{description}.png

例:
zen-01-bamboo-forest.png
sa-03-terere-culture.png
bilingual-07-arigatou-gracias.png
nomad-05-laptop-beach.png
quote-10-wabi-sabi.png
```
