# pSEO SaaS比較サイト — プロジェクトガイド

## 概要
メールマーケティングツールの「[A] vs [B]」比較ページを数千ページ自動生成し、Google検索流入→アフィリエイト報酬で収益化するプログラマティックSEOサイト。

## 事業モデル
- **ターゲット言語**: 英語（グローバル市場）
- **初期ニッチ**: Email Marketing Tools（70〜100ツール → 2,500〜5,000ページ）
- **拡張予定**: Phase 2でProject Management、Phase 3でCRM
- **収益化**: アフィリエイト（リカーリング報酬）+ 広告（Mediavine/AdSense）
- **集客**: Google検索のみ（営業ゼロ）

## 主要アフィリエイトプログラム

| ツール | 報酬 | 種別 | Cookie |
|-------|------|------|--------|
| AWeber | 50% | 生涯リカーリング | 365日 |
| GetResponse | 40-60% | 12ヶ月リカーリング | 120日 |
| Moosend | 40% | リカーリング | 90日 |
| ConvertKit (Kit) | 30% | 24ヶ月リカーリング | 90日 |
| ActiveCampaign | 20-30% | リカーリング | 90日 |
| Omnisend | 100%(初月）+ 50% | リカーリング | 90日 |

## 技術スタック
- **フレームワーク**: Next.js (App Router, SSG)
- **ホスティング**: Vercel
- **データ**: JSON/SQLite（ツール情報DB）
- **コンテンツ生成**: Claude API（比較文章）
- **監視**: Google Search Console API + GA4
- **デプロイ**: GitHub → Vercel自動デプロイ

## ディレクトリ構成
```
pseo-saas/
├── CLAUDE.md            ← このファイル
├── config/              ← サイト設定・アフィリエイトリンク
├── data/                ← ツール情報DB（JSON/SQLite）
├── scripts/             ← データ収集・ページ生成・監視スクリプト
├── public/              ← 静的ファイル（画像等）
├── docs/                ← 事業計画・リサーチ
└── site/                ← Next.jsプロジェクト（後で作成）
```

## ペナルティ回避ルール
- 月500〜1,000ページずつ段階的公開（一気に全公開しない）
- 各ページ最低500語のユニークコンテンツ（テンプレコピペ不可）
- 独自データ・分析を含める（Information Gain）
- 内部リンクを自然に構築
- 薄いページは作らない（データだけの表は不可）

## 収益目標
- Month 6: 5,000ページ、月50,000PV、$1,500/月
- Month 12: 10,000ページ、月150,000PV、$5,000〜$10,000/月
- Month 18: 20,000ページ、月300,000PV、$10,000〜$20,000/月
