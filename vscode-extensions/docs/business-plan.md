# VS Code Extensions 事業計画

## 市場分析

- VS Code月間アクティブユーザー: 約3,600万人（2025年時点）
- Marketplace拡張数: 約50,000
- ニッチなユーティリティ拡張は競争が少なく、安定したインストール数を獲得しやすい

## 収益モデル

### Phase 1: 無料公開（Month 1-3）
- 全10拡張を無料公開
- 目標: 各拡張 500+ インストール
- GitHub Sponsors リンク設置

### Phase 2: スポンサー誘導（Month 4-6）
- READMEに「Buy me a coffee」リンク追加
- 月間収益目標: $50-100

### Phase 3: プレミアム機能（Month 7+）
- 人気拡張にPro版を追加
- ライセンスキー販売（Gumroad経由）
- 月間収益目標: $200-500

## コスト

| 項目 | 費用 |
