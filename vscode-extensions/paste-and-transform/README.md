# Paste & Transform

Paste clipboard content with instant transformations. No more switching to external tools.

## Features

- **Case conversion**: camelCase, snake_case, UPPER_CASE, kebab-case, PascalCase
- **JSON**: Format (2 space indent) / Minify
- **Encoding**: Base64, URL encode/decode, HTML entities
- **Lines**: Sort (asc/desc), remove duplicates, trim whitespace

## Usage

1. Copy text to clipboard
2. Press `Ctrl+Shift+Alt+V` (or run "Paste & Transform" from Command Palette)
3. Select a transformation
4. Transformed text is inserted at cursor (or replaces selection)

## Keybinding

| Platform | Shortcut |
|----------|----------|
| Windows/Linux | `Ctrl+Shift+Alt+V` |
| macOS | `Cmd+Shift+Alt+V` |

## Install

Search "Paste & Transform" in the VS Code Extensions panel, or:

```
ext install miccho27.paste-and-transform
```
