# Smart Folding

Custom folding by type (imports, functions, classes, comments), named regions, and fold presets.

## Features

- Fold all imports/requires
- Fold all functions or classes
- Fold all comments/JSDoc blocks
- Named region support (#region/#endregion)
- Save and restore fold presets
- Auto-fold imports on file open
- TreeView of named regions

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
