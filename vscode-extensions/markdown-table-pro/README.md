# Markdown Table Pro

Format, sort, and convert Markdown tables with ease. CSV import/export, column alignment, and more.

## Features

- Auto-format tables with aligned columns
- Sort by any column (ascending/descending/numeric)
- CSV to Markdown table conversion
- Add/remove/move columns and rows
- Rename headers
- Format on save option
- Import CSV files directly

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
