# i18n Helper

Autocomplete translation keys, detect missing translations, and manage i18n JSON files.

## Features

- IntelliSense for t(), $t(), i18n.t() patterns
- Hover shows translation values across locales
- Detect missing translations as diagnostics
- Add/rename/delete keys across all locales
- Sort keys alphabetically
- TreeView browser for all locales

## Installation

Install from the VS Code Marketplace or install the  file manually:

1. Open VS Code
2. Press  and run "Extensions: Install from VSIX..."
3. Select the  file
