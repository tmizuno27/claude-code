## 競合の「穴」— 誰もやっていないこと

調査の結果、以下の**7つの空白**が見つかった。ここを突けば差別化できる。

### 空白1: 一次調査（アンケート・消費者パネル）
```
現状: 全競合がデスクリサーチ（二次調査）のみ
機会: LINE Research、Macromill、Rakuten Insightを使った
     実際の日本人消費者へのアンケート調査を提供できる人がゼロ
影響: これだけで「他の誰もやっていないサービス」になる
```

### 空白2: 業界特化のGig
```
現状: 全競合が「何でも調べます」のオールラウンダー
機会: 「日本の美容・コスメ市場専門」「日本のSaaS市場専門」
     「日本の食品・飲料市場専門」のように業界特化すると
     ニッチ検索で上位表示され、単価も上がる
影響: 特化Gigは一般Gigの2〜3倍のコンバージョン率
```

### 空白3: 月額レポートサービス（サブスク型）
```
現状: 全競合が単発プロジェクトのみ
機会: 「月次日本市場トレンドレポート」「四半期競合アップデート」
     のようなサブスクリプション型Gigを提供している人がゼロ
影響: 安定した月額収入 + 顧客生涯価値が大幅に向上
```

### 空白4: 日本のEC市場特化リサーチ
```
現状: Amazon Japan、楽天、Yahoo!ショッピング、メルカリに
     特化したリサーチGigが存在しない
機会: ECプラットフォーム横断の市場調査、SKU分析、価格比較、
     キーワードリサーチを提供
影響: EC市場2,865億ドルに対して専門リサーチャーがゼロ
```

### 空白5: 日本語ソーシャルリスニング
```
現状: X(Twitter)日本、5ch、Kakaku.com、@cosme、食べログ等の
     日本語SNS・レビューサイトからの情報収集を提供している人がゼロ
機会: 日本語限定のソーシャルリスニングレポートを提供
影響: 英語圏のツールでは収集できない生の日本人の声
```

### 空白6: 動画フォーマットのレポート
```
現状: 全競合がPDF/スプレッドシートのみ
機会: 動画プレゼンテーション形式（日本語ナレーション+英語字幕）
     で調査結果を報告。経営陣への社内プレゼンにそのまま使える
影響: 他にない形式 + 高単価設定が可能

```
