## Phase 1: 最初の5件を獲得（Week 2〜6）

**目標: ★5レビュー × 5件**（利益度外視、レビュー最優先）

### 毎日のルーティン（合計1時間/日）

```
朝 30分:
├── Fiverr: Briefsをチェック（表示されたら応募）
├── Upwork: 新着案件を検索
└── Upwork: 3〜5件に提案を送る

夜 30分:
├── メッセージ確認・返信（12時間以内）
├── Upwork: さらに2〜3件に提案
└── Fiverr: Gig統計を確認、タグ・タイトルを微調整
```

### 営業チャネル

| チャネル | 方法 | 期待値 |
|---------|------|--------|
| **Upwork提案** | 毎日5〜8件送信 | 1〜2週間で初受注 |
| **Fiverr Briefs** | 表示されたら即応募 | 新規セラーは数週間待ち |
| **Fiverr自然検索** | Gigを公開して待つ | 徐々にインプレッション増加 |
| **SNS発信** | LinkedInでサービス紹介 | 長期的な集客 |

### 最初の5件の鉄則

```
❌ 利益を最大化する
✅ ★5レビューを確実に取る

- 納期より1日早く納品
- 求められた以上の品質を出す
- 「追加修正あれば無料対応します」と伝える
- 完了後に丁寧に★5レビューをお願いする
```

### 収入目安


## Phase 4: No.1確立 + スケール（Month 7〜12）

**目標: 月$5,000〜10,000を安定して達成**

### 収入構造の完成形

```
安定ベース（月額契約）:
├── Gig 3 月額レポート × 5社         = $1,000〜5,000/月
└── Upwork 継続契約 × 2〜3社         = $1,000〜3,000/月

変動収入（単発プロジェクト）:
├── Gig 1 市場リサーチ × 月5〜10件    = $500〜2,500/月
├── Gig 2 ソーシャルリスニング × 月3〜5件 = $300〜1,500/月
└── Upwork 単発案件 × 月2〜3件        = $500〜2,000/月

合計: $3,300〜14,000/月
```

### スケールのための施策

| 施策 | 効果 |
|------|------|
| 業界特化Gigを追加（美容、SaaS、食品） | ニッチ検索で上位表示 + 単価UP |
| 単価をさらに引き上げ | Top Ratedバッジで強気の価格設定可能 |
| LinkedInで専門家発信 | プラットフォーム外からの集客 |
| 自分のWebサイト作成 | 直接契約（手数料0%）への移行 |
| 下請け体制の構築 | 受注量を超えたら外注で対応 |
