# PayPal vs Payoneer 徹底比較（日本在住フリーランサー向け）

---

## 結論: どちらがおすすめか？

### 🏆 Payoneer が圧倒的におすすめ

**理由**: Fiverr・Upworkの両プラットフォームとの統合が深く、手数料が安く、日本の銀行への出金がスムーズ。

ただし**PayPalも併用で登録しておく**のがベスト。理由は後述。

## 手数料の詳細比較

### PayPalの手数料構造

| 手数料項目 | 料率 | 備考 |
|-----------|------|------|
| 商用受取手数料 | 4.4% + ¥40（固定費） | 海外からの支払い受取時 |
| 為替換算手数料 | 3.5%（USD→JPY） | 市場レートに上乗せ |
| 日本の銀行への出金 | 無料 | ただし為替手数料に含まれる |
| 最低出金残高 | なし | |

**PayPalでの$500受取シミュレーション**:
```
クライアントが $500 を支払い
  ↓
PayPal受取手数料: -$22.00（4.4%）- $0.30（固定費）
  = $477.70 がPayPal残高に
  ↓
JPYへ換算（為替手数料 3.5%上乗せ）
  例: 市場レート $1 = ¥150 の場合
  PayPalレート: $1 = ¥144.75（3.5%引き）
  = ¥69,162
  ↓
日本の銀行に出金: 無料
  = 最終受取: 約 ¥69,162

※ 元の$500 × ¥150 = ¥75,000 に対して
  実質手数料: 約 ¥5,838（約7.8%）
```

### Payoneerの手数料構造

| 手数料項目 | 料率 | 備考 |
|-----------|------|------|
| Fiverr/Upworkからの受取 | 無料（0%） | プラットフォーム連携の場合 |
| 為替換算手数料 | 最大2.0% | USD→JPY換算時 |
| 日本の銀行への出金 | 為替手数料に含まれる | 追加費用なし |
| 最低出金額 | $30 | |

**Payoneerでの$500受取シミュレーション**:
```
クライアントが $500 を支払い（Fiverr/Upwork経由）
  ↓
Payoneer受取手数料: 無料
  = $500 がPayoneer残高に
  ↓
JPYへ換算（為替手数料 最大2%上乗せ）
  例: 市場レート $1 = ¥150 の場合
  Payoneerレート: $1 = ¥147（2%引き）
  = ¥73,500
  ↓
日本の銀行に出金: 無料（為替に含まれる）
  = 最終受取: 約 ¥73,500

※ 元の$500 × ¥150 = ¥75,000 に対して
  実質手数料: 約 ¥1,500（約2.0%）
```

## メリット・デメリット比較

### PayPal

**メリット**:
- 即日アカウント開設・利用開始可能
- 世界的に知名度が高く、クライアントの信頼度◎
- Fiverr以外のクライアントとの直接取引にも便利
- アプリが使いやすい
- 日本語サポートが充実

**デメリット**:
- 受取手数料が高い（4.4%+固定費）
- 為替手数料が高い（3.5〜4.0%）
- 合計すると**7〜8%以上**が手数料で消える
- 日本ではビジネスアカウントが個人で使いにくい場合がある
- アカウント凍結リスク（突然の利用制限が海外で問題視されている）

### Payoneer

**メリット**:
- Fiverr/Upworkからの受取手数料が無料
- 為替手数料が最大2%（PayPalの半分以下）
- 70通貨以上に対応
- Fiverr・Upwork公式推奨の出金方法
- 複数通貨での残高保持が可能（為替タイミングを選べる）
- 年間数万〜数十万円の節約

**デメリット**:
- アカウント審査に2〜5営業日かかる
- 本人確認書類（パスポート等）の提出が必須
- 最低出金額$30（少額出金に不向き）
- PayPalほど一般消費者には知られていない
- 審査落ちの可能性がゼロではない

## おすすめの使い分け戦略

### ベスト戦略: 両方登録して使い分ける

```
【メイン出金: Payoneer】
├── Fiverr の売上 → Payoneer で出金 → 日本の銀行口座
├── Upwork の売上 → Payoneer で出金 → 日本の銀行口座
└── 理由: 手数料が圧倒的に安い（約2% vs 約8%）

【サブ出金: PayPal】
├── Fiverr/Upwork以外のクライアントとの直接取引
├── 小額の緊急出金（$30未満の場合）
├── 一部のクライアントがPayPal指定の場合
└── 理由: 知名度が高く、直接取引で便利
```

### 登録の優先順位

| 順番 | アクション | 理由 |
|------|----------|------|
| 1位 | **Payoneer に登録** | 審査に2〜5日かかるので最優先 |
| 2位 | **PayPal に登録** | 即日利用可能。待ち時間に登録 |
| 3位 | Fiverr/Upwork で Payoneer を出金方法に設定 | Payoneer審査通過後すぐに |
| 4位 | PayPal も予備の出金方法として設定 | バックアップ用 |

---

## 為替タイミング戦略（Payoneer限定）

Payoneerでは**外貨のまま残高を保持**できるため、為替レートが有利なタイミングで換金できる。

```
例: 円安の時（$1 = ¥155）に換金 → お得
例: 円高の時（$1 = ¥140）は待つ → USD残高で保持

PayPalはこの戦略が使えない（受取時に自動換算されることが多い）
```

**注意**: 為替予測は難しいため、基本は「一定額たまったら出金」が安全。

## まとめ

| 項目 | 推奨 |
|------|------|
| メインの出金方法 | **Payoneer**（手数料2%で圧倒的に安い） |
| 最初に登録すべき | **Payoneer**（審査待ちの間にPayPalも登録） |
| サブの出金方法 | **PayPal**（直接取引・小額出金用） |
| 月収$1,000以上の場合 | **Payoneer一択**（年間10万円以上の差） |
| 月収$500未満の場合 | **Payoneer推奨**だがPayPalでも大差なし |

**最終結論: Payoneerをメイン、PayPalをサブで両方登録がベスト。**
