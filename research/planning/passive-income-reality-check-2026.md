# 放置ビジネス リアリティチェック 2026（12クエリ調査結果）

作成日: 2026-03-15
調査方法: 12の異なる検索クエリで最新データを収集し、マーケティング的な誇張を排除して現実を評価

---

## 結論（先に言う）

**「完全放置で稼げるビジネス」は存在しない。** ただし「週1-2時間のメンテナンスで月5-50万円」は現実的に達成可能なモデルが複数ある。

---

## 全モデル リアリティ評価

### Tier S: 最も放置に近い（構築後 週0-1時間）

#### 1. デジタルテンプレート販売（Notion / Canva / Figma / n8n）
| 項目 | 現実 |
|------|------|
| **実際の週間メンテ** | 0-1時間（プラットフォームのアプデ対応が年数回） |
| **現実的月収** | 初心者: $500-1,000 / 中級: $3,000-5,000 / トップ: $5,000-50,000 |
| **実データ** | Gumroad上のNotion Top sellers: $5,000-20,000/月。Figma Academy創設者: 4ヶ月で$300,000。Framerテンプレ販売者: 19ヶ月で$200,000超 |
| **Claude Codeで構築・維持?** | ◎ 可能。Notionテンプレート・n8nワークフロー・マークダウン教材はClaude Codeで完全作成可能 |
| **失敗モード** | 市場の飽和（特にNotion）。差別化できない汎用テンプレは月数百円で終わる。ニッチ特化が必須 |
| **正直な評価** | ★★★★★ 最も放置に近い。作って出品したら本当にほぼ何もしなくていい。ただし売れるテンプレを作れるかが全て |

#### 2. 電子書籍（KDP / JP+EN同時）
| 項目 | 現実 |
|------|------|
| **実際の週間メンテ** | 0時間（出版後は完全放置） |
| **現実的月収** | 1冊あたり月$5-50（JP）、$10-100（EN）。10冊×2言語で月$300-3,000 |
| **Claude Codeで構築・維持?** | ◎ 執筆→翻訳→フォーマットまで完全自動化可能 |
| **失敗モード** | AI生成書籍の飽和でKDPが規約強化。レビューが付かず埋もれる。低品質だとアカウント停止リスク |
| **正直な評価** | ★★★★☆ 本当に放置できるが、1冊あたりの収益が低い。量産戦略が必要で「質×量」のバランスが鍵 |

#### 3. プロンプト / GPTs販売
| 項目 | 現実 |
|------|------|
| **実際の週間メンテ** | 0-30分（モデルアプデ時にプロンプト修正が必要な場合あり） |
| **現実的月収** | $100-1,000（大半）。ヒットすれば$3,000+ |
| **Claude Codeで構築・維持?** | ◎ プロンプト設計・テスト・販売ページ作成まで全て可能 |
| **失敗モード** | 陳腐化が極めて速い（モデルの進化で不要になる）。競合が多すぎて埋もれる |
| **正直な評価** | ★★★☆☆ 放置度は最高だが、収益の天井が低く不安定。補助的な収入源としては良い |

### Tier A: 半放置（週1-3時間）

#### 4. マイクロSaaS
| 項目 | 現実 |
|------|------|
| **実際の週間メンテ** | 2-5時間（バグ修正、サーバー監視、顧客対応） |
| **現実的月収** | 70%が月$1,000未満。成功すれば$5,000-50,000/月。トップ事例: Nomad List $1.5M ARR、Carrd $1.5M ARR |

| **正直な評価** | ★★★★☆ 収益ポテンシャルは最大だが「放置」ではない。週2-5時間は最低限かかる。200顧客×$99/月で年$240K、利益率41%で$98K |

#### 6. データレポート自動販売

### Tier B: 放置とは言えない（週3-5時間+）

| 項目 | 現実 |

| **正直な評価** | ★★☆☆☆ 2026年時点で最もリスクが高いモデルの一つ。SEO依存型アフィリエイトは衰退中。コミュニティベース・独自データ・一次情報がないサイトは生き残れない |

## 新発見モデル（既存リストにないもの）

### 11. 帯域幅シェアリング（Pawns.app等）
- **週間メンテ**: 0時間（インストールして放置）
- **月収**: $1-5/月（現実）
- **評価**: ★☆☆☆☆ 本当に放置だが稼げない。時間の無駄

### 12. オンラインコース（Udemy / Teachable）
- **週間メンテ**: 1-2時間（Q&A対応、コンテンツ更新）
- **月収**: 平均$3,000/コース。初心者$200-1,000/月。2026年トレンド: マイクロコース（30分の深掘り）がフルコースの3倍のコンバージョン率
- **Claude Codeで構築?**: ○ カリキュラム・台本・スライド作成は可能。動画撮影は人間が必要
- **評価**: ★★★★☆ 一度作れば長期的に売れる。Udemyが集客代行。ただしUdemyの取り分63%は痛い

## 調査で最も重要だった知見

> 「簡単にコピーできる単一の配信チャネルに依存するビジネスは、そのチャネルが変わった瞬間に破綻する」

> 「2026年のSEO: ページ数を減らし、洞察を深め、視点を強くし、複製困難なアセットを作る」

> 「マイクロSaaSの70%は月$1,000未満。ただし成功した1-2%は月$50,000超」

> 「デジタル商品市場は2026年に$124B突破。マイクロコース（30分）がフルコースの3倍のCVR」

> 「『パッシブ』は前もって仕事を集中投下するという意味。構築後も月1-2回の品質チェックは必須」
