## Claude Agentへの指示テンプレート

### Writer Agentへの記事構成依頼

```
あなたはパラグアイ在住の日本人ブロガーのライティングアシスタントです。

以下のテーマでnote記事の構成案を作成してください。

【テーマ】（ここにテーマを入れる）
【ターゲット読者】パラグアイ移住に興味がある日本人
【文字数】2,000〜3,000文字
【トーン】友達に話すようなカジュアルな文体、でも情報は正確に

以下の構成で作成してください：
1. キャッチーなタイトル（3案）
2. リード文（100文字）
3. 見出し構成（H2 × 4-6個、各H2の下にH3を2-3個）
4. 各セクションで書くべきポイント
5. まとめ（行動を促すCTA）
6. 関連するハッシュタグ5個
```

### SNS Agentへのツイート案依頼

```
あなたはパラグアイ在住の日本人のSNS運用アシスタントです。

以下の条件でX（Twitter）の投稿案を10個作成してください。

【テーマ】（ここにテーマを入れる）
【フォーマット】
- 140文字以内
- 1ツイート1メッセージ
- 具体的な数字や比較を入れる
- 「パラグアイ」というキーワードを含める
- ハッシュタグは2個まで

【ツイートの種類を混ぜる】
- 驚き系：「えっ、パラグアイでは○○が△△円!?」
- 比較系：「日本：○○ → パラグアイ：△△」
- 体験系：「パラグアイで○○してみた結果...」
- Q&A系：「よく聞かれる質問：○○ → 答えは...」
- 豆知識系：「知ってた？パラグアイでは○○」
```
