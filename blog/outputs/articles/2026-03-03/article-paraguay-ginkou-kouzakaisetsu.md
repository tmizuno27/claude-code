<span id="パラグアイで外国人が口座を作るには"></span>

## パラグアイで外国人が口座を作るには？

パラグアイで外国人が銀行口座を開設するには、永住権（セドゥラ）を取得しているのが原則。セドゥラさえあれば、Banco ContinentalやBanco Itauなどの主要銀行で、グアラニー建て・ドル建ての口座を即日〜数日で開設できる。

ただ、ここで厄介なのが「永住権を取るには銀行口座が必要 → 銀行口座を開くには永住権が必要」という、いわゆる"鶏と卵"問題。正直、私もこれには頭を抱えた。移住前に調べていて「え、どっちが先なの？」となった人は多いと思う。

この記事では、私が実際にBanco Continentalで口座を開設した体験をベースに、必要書類・手順・鶏と卵問題の突破方法をまとめた。Wise（旧TransferWise）やPayPalとの使い分けも書いているので、パラグアイでのお金まわりの情報はこれ一本でだいたいカバーできるはず。

<span id="口座開設に必要な書類リスト"></span>

## 口座開設に必要な書類リスト

外国人がパラグアイで銀行口座を開設する場合、以下の書類が必要になる（2026年3月時点）。

<span id="必須書類"></span>

### 必須書類

| # | 書類 | 詳細 |
|---|------|------|
| 1 | セドゥラ（Cedula de Identidad） | パラグアイの永住権カード。最も重要な書類 |
| 2 | パスポート原本 | 有効期限内のもの |
| 3 | 住所証明 | 電気・水道の請求書、または賃貸契約書（直近3ヶ月以内） |
| 4 | 収入証明 | 雇用証明書、確定申告書、または海外からの送金明細 |
| 5 | RUC（納税者番号） | 個人のRUCが必要。SET（税務署）で取得可能 |

<span id="あると有利な追加書類"></span>

### あると有利な追加書類

- 銀行残高証明書（日本の銀行のもの、アポスティーユ付き）
- 不動産の所有証明
- パラグアイ国内の知人からの紹介状

銀行によって求められる書類が微妙に違う。なので、いきなり開設に行くんじゃなくて、事前に支店へ行って「口座開設にはどんな書類が必要ですか？」と確認するのが確実。私はスペイン語が挨拶程度しかできないので、Google翻訳アプリを見せながら聞いた。紙にリストを書いてくれたので、それを持って準備した。

<span id="おすすめ銀行2行を比べてみた"></span>

## おすすめ銀行2行を比べてみた

パラグアイには銀行がいくつもあるけど、外国人がまず検討すべきなのはBanco ContinentalとBanco Itauの2行だと思う。

| 比較項目 | Banco Continental | Banco Itau |
|---------|-------------------|------------|
| 外国人対応 | ◎ 外国人の口座開設実績が豊富 | ○ 対応可能だが支店による |
| ATM網 | ◎ 国内最大級 | ○ 主要都市に展開 |
| オンラインバンキング | ◎ アプリの使い勝手が良い | ◎ 国際基準のUI |
| ドル建て口座 | ○ 開設可能 | ◎ ブラジル系で外貨に強い |
| 手数料 | 月額維持費 約500〜800円 | 月額維持費 約600〜1,000円 |
| デビットカード | Visa / Mastercard | Visa / Mastercard |
| 国際送金受取 | ○ SWIFT対応 | ◎ SWIFT対応、Itau国際ネットワーク |
| 最低預入額 | 約5,000〜10,000円相当 | 約8,000〜15,000円相当 |
| おすすめの人 | 初めての口座開設、日常使い | 国際送金が多い人、ドル運用重視 |

<span id="個人的にはbanco-continentalを推す"></span>

### 個人的にはBanco Continentalを推す

外国人が最初に口座を作るならBanco Continentalが無難。理由はシンプルで3つ。

1. 外国人の口座開設実績が多いから、担当者が慣れていて手続きがスムーズ
2. ATMが全国に多い。ランバレやサンロレンソなど郊外でも困らない
3. アプリが普通に使いやすい。残高確認、振込、QR決済がスマホで完結する

もちろん両方持つこともできるので、落ち着いたらBanco Itauでドル建て口座を追加するのもアリ。

<span id="口座開設の手順を5ステップで解説"></span>

## 口座開設の手順を5ステップで解説

<span id="step-1-支店を選んで訪問"></span>

### Step 1: 支店を選んで訪問

まず開設したい銀行の支店を選ぶ。アスンシオン中心部やショッピングモール内の支店は外国人対応に慣れていることが多いので、そこを狙うのがおすすめ。

Webや電話で予約できる支店もあるけど、予約なしで直接行っても対応してくれる。ただし待ち時間は覚悟したほうがいい。午前中の早い時間に行くのがコツ。

<span id="step-2-書類を出して窓口で手続き"></span>

### Step 2: 書類を出して窓口で手続き

必要書類を持参して窓口へ。私はスペイン語が挨拶程度しかできないので、スマホのGoogle翻訳を駆使して乗り切った。「口座を開設したい」くらいの単語はメモして持っていったけど、正直それ以上の会話は翻訳アプリ頼みだった。窓口の人も外国人対応には慣れていたので、ジェスチャーと翻訳アプリでなんとかなった。

<span id="step-3-申込書に記入"></span>

### Step 3: 申込書に記入

担当者から申込書（formulario）を渡されるので記入する。住所、勤務先（フリーランスの場合はその旨）、月収の目安あたりを書く。書き方がわからないところは担当者がジェスチャーで教えてくれた。

<span id="step-4-初回入金"></span>

### Step 4: 初回入金

口座が承認されたら、最低預入額を入金。Banco Continentalの場合、グアラニー建て口座で約5,000〜10,000円相当、ドル建て口座で100 USD程度が目安。

<span id="step-5-デビットカードとオンラインバンキングの設定"></span>

### Step 5: デビットカードとオンラインバンキングの設定

口座開設と同時にデビットカードの発行を頼める。カードは1〜2週間くらいで届く。オンラインバンキングとモバイルアプリの登録も窓口でサポートしてもらえるので、そのままやってしまうのが楽。

<span id="鶏と卵問題をどうクリアするか"></span>

## 「鶏と卵」問題をどうクリアするか

パラグアイの永住権取得手続きでは、銀行の残高証明書が必要書類に入っている。でも銀行口座を開設するにはセドゥラ（永住権カード）が必要。

...って、どっちが先なんだよ、と。

これが「鶏と卵」問題。実際につまずく人が多い。私もめちゃくちゃ悩んだ。で、結論としては以下の方法がある。

<span id="方法1-日本の銀行残高証明書を使うこれが一番メジャー"></span>

### 方法1: 日本の銀行残高証明書を使う（これが一番メジャー）

永住権申請時の残高証明は、日本の銀行の残高証明書（英語またはスペイン語、アポスティーユ付き）でも受理されるケースがほとんど。

やることはこう。

1. 日本の銀行に英文の残高証明書を依頼（郵送対応あり）
2. 外務省でアポスティーユを取得（郵送申請可）
3. パラグアイの翻訳公証人（Traductor Publico）でスペイン語に翻訳
4. この書類で永住権を申請 → セドゥラ取得後に現地銀行で口座開設

<span id="方法2-移民弁護士ヘストールに頼む"></span>

### 方法2: 移民弁護士（ヘストール）に頼む

パラグアイにはヘストール（Gestor）と呼ばれる移住手続き専門のエージェントがいる。銀行との太いパイプを持っていて、セドゥラ発行前でも口座開設の道筋をつけてくれることがある。

費用は1,000〜3,000 USD程度。永住権取得の手続き全体をやってくれるので、初めての移住なら使う価値はある。私はスペイン語が挨拶程度しかできなかったので、日本語対応のヘストールに依頼した。言葉の壁を考えると、この選択は正解だった。

<span id="方法3-仮の在留許可証で口座を開く"></span>

### 方法3: 仮の在留許可証で口座を開く

永住権の申請中に発行される仮の在留許可証（Certificado de Radicacion en tramite）で口座開設に応じてくれる銀行・支店もある。ただ、これは支店や担当者の裁量次第なので、ダメと言われたら別の支店を当たるしかない。

私の場合は、方法1と2の合わせ技で突破した。日本にいるうちに銀行残高証明書（アポスティーユ付き）を準備して、現地では日本語対応のヘストールに永住権申請をお願いした。セドゥラが出た翌週にはBanco Continentalへ駆け込んだ。

## 実際にBanco Continentalで口座を作ったときの話

【写真: Banco Continentalの支店外観、またはデビットカードの写真（個人情報は隠す）】

ここからは私のリアルな体験談。

<span id="開設した口座"></span>

### 開設した口座

- グアラニー建て普通預金口座（Cuenta de Ahorro en Guaranies）
- ドル建て普通預金口座（Cuenta de Ahorro en Dolares）

この2つを同時に開設した。日常の支払いはグアラニー口座、海外からの送金受取はドル口座、という使い分けにしている。

<span id="手続きにかかった時間"></span>

### 手続きにかかった時間

支店での手続きは約1時間半。書類確認、申込書記入、システム登録、初回入金まで全部込み。ただ、混んでいるともっとかかると思うので、午前中の早い時間に行くのがおすすめ。

<span id="持参した書類"></span>

### 持参した書類

1. セドゥラ（永住権カード）
2. 日本のパスポート
3. ANDEの電気代請求書（住所証明として）
4. RUC番号の登録証明書
5. 海外からの送金明細（Wiseの取引履歴を印刷したもの）

【要追記】実際に提出した書類の具体的な詳細や、担当者とのやり取りの詳細を追記する

<span id="困ったこと"></span>

### 困ったこと

収入証明でちょっと焦った。フリーランスだから雇用証明書なんてない。Wiseの送金明細を「これが収入の証拠です」と出したんだけど、「これでいけるのか...？」と内心ドキドキだった。結果的には問題なく通った。

あとRUC。口座開設当日に「RUCは？」と聞かれて、持っていなかった。「え、RUCって何？」状態。幸いSET（税務署）がすぐ近くにあったので、その場で走って取りに行って、なんとか当日中に全部終わらせた。これは事前に取得しておくべき。本当に。

## Wise・PayPalとパラグアイ銀行口座の使い分け

パラグアイの銀行口座だけだと、海外との送金がどうしても不便。なので、以下のサービスを組み合わせている。

| サービス | 用途 | メリット | デメリット |
|---------|------|---------|-----------|
| [Wise](https://YOUR-AFFILIATE-LINK/wise) | 日本→パラグアイの送金 | 手数料が安い（0.5〜1%程度）、レートが良い | パラグアイ国内での支払いには不向き |
| PayPal | 海外クライアントからの報酬受取 | 世界中で普及、受取が簡単 | 手数料が高い（3〜5%）、パラグアイの銀行への出金に制限あり |
| パラグアイ銀行口座（グアラニー） | 現地の日常支払い | QR決済対応、ATM引き出し | 海外送金の手数料が高い |
| パラグアイ銀行口座（ドル） | ドル建て資産の保管 | 為替リスクヘッジ | グアラニーへの両替手数料がかかる |

<span id="私のお金の流れ"></span>

### 私のお金の流れ

```
日本の口座 → Wise → パラグアイのドル口座
                         ↓
                   必要分をグアラニー口座に振替
                         ↓
                   デビットカード / QR決済で日常支払い
```

この流れが手数料を一番抑えられるパターン。日本円 → ドル → グアラニーの2段階で両替するんだけど、Wiseの為替レートが優秀なので、銀行の国際送金（SWIFT）を使うよりずっとお得になる。

Wiseの口座開設はこちら: [Wise公式サイト](https://YOUR-AFFILIATE-LINK/wise)（紹介リンク経由で初回送金手数料が無料になります）

<span id="よくある質問faq"></span>

## よくある質問（FAQ）

**Q. 観光ビザでも銀行口座は作れる？**

A. 原則として無理。セドゥラ（永住権カード）が必要になる。ただ一部の銀行・支店では仮の在留許可証で対応してくれることもあるので、ヘストール（移民弁護士）経由で交渉してみる価値はある。

**Q. 口座開設にスペイン語はどれくらい必要？**

A. 窓口でのやり取りは基本スペイン語。英語が通じる担当者もいるけど、当たり外れがある。私の場合はスペイン語が挨拶程度だったので、Google翻訳アプリをフル活用した。事前に必要なフレーズをメモしておくだけでもだいぶ違う。

**Q. 口座維持費はかかる？**

A. かかる。月額500〜1,000円程度。ただ一定の残高を維持していれば無料になるプランもあるので、開設時に確認してみてほしい。

**Q. 銀行口座を作ればクレジットカードも持てる？**

A. 口座開設後にVisa / Mastercardを申請できる。ただし外国人の場合、審査はやや厳しくて、一定期間の口座利用実績と安定した収入証明が求められる。最初はデビットカードで半年〜1年実績を積んでから申請するのが現実的。

**Q. 日本にいながらパラグアイの口座は作れる？**

A. 現時点では無理。本人が支店に行く必要がある。永住権の取得も含めて、最低1回はパラグアイに来ないといけない。永住権取得の詳細は[パラグアイの永住権の取り方](https://nambei-oyaji.com/paraguay-eijuuken-torikata/)を参照。

## 銀行口座は移住生活の土台になる

パラグアイの銀行口座があると、生活の利便性がまるで変わる。デビットカードでの支払い、QR決済、光熱費の自動引き落とし、海外送金の受取。口座なしだと全部現金でやることになって、これがけっこう面倒。

鶏と卵問題は確かにややこしいけど、日本の銀行残高証明書を事前に用意しておけば突破できる。永住権申請と並行して書類を揃えて、セドゥラが出たらすぐ銀行へ行く。これがスムーズな流れ。

移住前の段階では、とりあえずWiseの口座だけ作っておくといい。渡航後はWise経由で資金を送金する体制を整えておけば、現地口座ができるまでの間も困らない。
