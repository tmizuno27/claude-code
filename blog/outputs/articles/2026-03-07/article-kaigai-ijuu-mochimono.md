# 海外移住の持ち物リスト完全版｜日本から持っていくべきもの＆不要だったもの【実体験】

2回の海外移住で「スーツケース4個+段ボール12箱」を運んだ結果、半分は不要でした。**海外移住で日本から持っていくべきものは、重要書類・日本の医薬品・電子機器・日本食材の4つが最優先です。** 逆に、大量の衣類や日用品は現地調達のほうが合理的。

私は家族4人（妻と娘2人）でマレーシアに2年住んだあと、現在パラグアイに移住して生活しています。「あれ持ってくればよかった...」と後悔したものもあれば、「これ完全にいらなかった」というものも山ほどありました。

この記事では、実体験をもとにカテゴリ別の持ち物リストと、航空便・船便の使い分けまで具体的に解説します。これから海外移住を控えている方は、ぜひチェックリストとして活用してください。

## カテゴリ別・持ち物リスト

### 1. 重要書類（最優先）

書類関係は現地で入手不可能なものが多いため、最も優先度が高いカテゴリです。

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| パスポート（全員分） | 必須 | 残存期間を確認。子供は5年パスポート |
| 戸籍謄本（複数部） | 必須 | 永住権申請やビザ手続きに必要。アポスティーユ付きも用意 |
| 国際運転免許証 | 高 | 日本の免許センターで取得。有効期間1年 |
| 卒業証明書・成績証明書 | 中 | 子供の学校入学手続きに必要な場合あり |
| 健康診断書・予防接種記録 | 高 | 英文のもの。子供の母子手帳も必携 |
| 海外転出届の控え | 必須 | 役所で手続き後に受け取る |
| 在留届（オンライン） | 必須 | ORRnet で渡航後に提出 |
| 各種保険証券のコピー | 中 | PDF化してクラウドにも保存 |

**ポイント**: 原本だけでなく、すべてPDF化してGoogle DriveやDropboxにアップロードしておくこと。紛失時に再発行の手がかりになります。

### 2. 電子機器

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| ノートPC | 必須 | 海外では日本語キーボード配列が手に入らない |
| SIMフリースマホ | 必須 | SIMロック解除を日本で済ませておく |
| 変換プラグ（マルチタイプ） | 必須 | ダイソーで買えば100円。複数個用意 |
| モバイルバッテリー | 高 | 停電が多い国では生命線 |
| 延長コード・電源タップ | 高 | 日本製は品質が段違い |
| ポケットWi-Fi / eSIM | 中 | 到着直後のネット確保用 |
| 外付けHDD / USBメモリ | 中 | 重要データのバックアップ用 |

**注意**: 変圧器は基本的に不要です。最近のPC・スマホの充電器はほぼ100-240V対応。ただしドライヤーなど熱器具を持っていく場合は必要です（でもドライヤーは現地購入を推奨）。

### 3. 医薬品・衛生用品

海外で最も手に入りにくいのが日本の医薬品です。ここはケチらず多めに持っていきましょう。

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| 常備薬（処方薬） | 必須 | 英文の処方箋も取得しておく |
| 目薬 | 高 | 日本の目薬は海外で入手困難 |
| 胃腸薬（正露丸など） | 高 | 食あたり対策。特に南米は必須 |
| 解熱鎮痛剤 | 高 | ロキソニン等。現地のは合わないことも |
| 虫除けスプレー・虫刺され薬 | 高 | 熱帯地域は特に重要 |
| 日焼け止め | 中 | 日本製のほうが品質が良い |
| 歯ブラシ | 中 | 海外のはヘッドが大きすぎることが多い |
| 体温計 | 中 | 日本語表示のデジタル体温計 |

### 4. 日本食材・調味料

「日本食ロス」は海外移住者あるある。特に子供がいる家庭は多めに持っていくのがおすすめです。

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| だしパック・顆粒だし | 高 | ほんだし等。現地では高額 or 入手不可 |
| 醤油（小型ボトル） | 高 | キッコーマンは海外にもあるが高い |
| インスタント味噌汁 | 高 | 軽くて場所を取らない |
| ふりかけ・のり | 高 | 子供のご飯が進む救世主 |
| カレールー | 中 | 日本のカレーは海外では作れない味 |
| 乾燥わかめ | 中 | 軽量でかさばらない |
| お茶パック（緑茶・麦茶） | 中 | 水出しタイプが便利 |
| インスタントラーメン | 低 | 重いので厳選して |

【要追記】パラグアイで日本食材が買えるアジアンショップの具体的な店名・場所

### 5. 衣類

衣類は最も「持っていきすぎた」と後悔しやすいカテゴリです。

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| 下着・靴下（1週間分） | 必須 | 日本製の品質は海外で買えない |
| ユニクロのエアリズム・ヒートテック | 高 | 機能性インナーは日本の独壇場 |
| フォーマル服（1セット） | 中 | ビザ申請・学校行事用 |
| 薄手のダウンジャケット | 中 | ユニクロのウルトラライトダウン推奨 |
| 運動靴 | 中 | 日本人の足に合うサイズがない国も |

**実体験**: 普段着は現地で買ったほうが気候に合っています。パラグアイは暑い日が多いので、日本から冬服を大量に持ってきても出番がほとんどありませんでした。

### 6. 子供用品

| 持ち物 | 重要度 | 備考 |
|--------|--------|------|
| 日本語の絵本・ドリル | 高 | 日本語維持の生命線 |
| 学習用タブレット | 高 | 日本の教材アプリを入れておく |
| 子供用の常備薬 | 必須 | 小児用の薬は海外で入手困難 |
| 水筒 | 中 | 日本製の保温水筒は優秀 |
| 文房具 | 中 | 日本の文房具は世界一の品質 |
| お気に入りのおもちゃ | 中 | 引越し直後の精神安定に重要 |

## 持ってきてよかったものTOP5

実際にパラグアイで生活してみて、「本当に持ってきてよかった」と感じたものをランキングにしました。

### 第1位：日本の医薬品

これは断トツの1位。パラグアイの薬局にも薬はありますが、日本の薬のほうが体に合う。特に胃腸薬と目薬は現地では代替品が見つかりませんでした。

### 第2位：だし・調味料類

パラグアイにもアジア食材店はありますが、品揃えは限定的で価格も高い。顆粒だしとカレールーがあるだけで、食卓のクオリティが劇的に変わります。

### 第3位：子供の日本語教材

海外生活が長くなると子供の日本語力がどんどん落ちていきます。日本語のドリルや絵本は現地で入手する方法がほぼないので、多めに持ってきて正解でした。

【写真: 娘たちが使っている日本語ドリルや絵本の写真】

### 第4位：日本製の文房具

これは子供用だけでなく自分用も含めて。ボールペン、消しゴム、ハサミ、爪切り。日本の文房具の品質は海外に来ると痛感します。現地の同僚や友人へのお土産にもなります。

### 第5位：ユニクロのインナー

エアリズムは南米の暑い気候にぴったり。現地のインナーとは快適さが段違い。ヒートテックも冬（6-8月）の朝晩に活躍しています。

## 不要だったものTOP5

逆に「持ってこなくてよかったのに...」と思ったものも正直に紹介します。

### 第1位：大量の衣類

日本の気候に合わせた服を大量に持ってきましたが、パラグアイの気候には合わないものばかり。現地のショッピングモールで安く買えるので、最小限で十分です。

### 第2位：炊飯器

「海外で美味しいご飯を食べたい！」と思って持っていきましたが、パラグアイでも炊飯器は買えます。しかも変圧器が必要になるケースもあり、重くてかさばるだけでした。【要追記】実際に持っていったかどうか確認

### 第3位：大量のタオル

タオルは現地で安く買えます。バスタオル数枚を船便のスキマに詰めるくらいならいいですが、わざわざ持っていく優先度は低い。

### 第4位：日本の家電

ドライヤー、ヘアアイロンなどの熱器具は電圧の問題で変圧器が必要。重いし壊れるリスクもある。現地で買い直したものがいくつもあります。

### 第5位：紙の本（大量）

読みたい気持ちはわかりますが、紙の本は重い。Kindleに入れれば何百冊でも持ち運べます。子供の絵本・ドリルは紙のほうがいいですが、大人の本は電子書籍に切り替えましょう。

## 航空便 vs 船便の使い分け

海外移住の荷物は、手荷物・航空便・船便の3つに分けて送るのが基本です。

| 方法 | 到着日数 | コスト目安 | 適した荷物 |
|------|----------|-----------|-----------|
| 手荷物（スーツケース） | 当日 | 航空券に含む | 書類、PC、薬、貴重品、到着後すぐ使うもの |
| 航空便（EMS等） | 1〜2週間 | 高い（重量課金） | 調味料、衣類（最小限）、子供用品 |
| 船便 | 1〜3ヶ月 | 安い | 大量の本、季節外の衣類、キッチン用品 |

### 使い分けのコツ

**手荷物に入れるもの**（到着初日から必要）:
- パスポート・重要書類
- ノートPC・スマホ・充電器
- 3日分の着替え
- 常備薬・救急セット
- 現金・クレジットカード

**航空便で送るもの**（1〜2週間以内に届いてほしい）:
- 残りの衣類
- 日本食材・調味料
- 子供の教材・おもちゃ
- 文房具

**船便で送るもの**（届くまで待てるもの）:
- 大量の本
- 季節外の衣類
- キッチン用品
- 思い出の品

【要確認】パラグアイへの船便の具体的な日数と、日本郵便のEMS対応状況（2026年時点）

## 持ち物を減らすためのマインドセット

2回の海外移住を経験して学んだのは、**「迷ったら持っていかない」が正解**ということ。

理由はシンプルです:

1. **現地で買えるものは意外と多い**: Amazonが使える国なら日本製品も手に入る
2. **スーツケースの重量制限は厳しい**: 超過料金は馬鹿にならない
3. **環境が変わると必要なものも変わる**: 日本で「必須」だったものが海外では不要になる

特にパラグアイの場合、ブラジルとの国境が近いため、ブラジル製の家電や日用品が安く手に入ります。生活が落ち着いてから必要なものを買い足していけば十分です。

移住前の準備全体については、[海外移住の準備リスト](/kaigai-ijuu-junbi-list/)の記事で時系列にまとめています。また、現地での出費がどのくらいかかるかは[パラグアイの生活費](/paraguay-seikatsuhi/)を参考にしてください。

## よくある質問（FAQ）

### Q1. 日本食材は現地で買えないの？

国によります。パラグアイの場合、首都アスンシオンにアジア食材店がいくつかありますが、品揃えは限定的で価格は日本の2〜3倍。醤油やインスタントラーメンなど定番品は手に入りますが、だしパックやふりかけなどは見つからないことが多いです。

### Q2. 薬は何ヶ月分持っていけばいい？

処方薬は最低3ヶ月分、できれば半年分を推奨します。英文の処方箋があれば持ち込みでトラブルになることはほとんどありません。市販薬（胃腸薬・目薬など）は半年〜1年分あると安心です。

### Q3. 子供の荷物で最優先すべきものは？

日本語教材（ドリル・絵本）と常備薬です。おもちゃや服は現地で買えますが、日本語教材だけは海外で入手する方法がほぼありません。Kindleキッズなどのデジタル教材も活用しましょう。子連れ移住の詳しい準備については[子連れ海外移住ガイド](/kaigai-ijuu-kosodate/)もご覧ください。

### Q4. 航空便と船便、どちらがおすすめ？

到着後2週間以内に必要なものは航空便、それ以外は船便がコスパ良いです。ただし船便は1〜3ヶ月かかるうえ、国によっては紛失・破損リスクもあるため、代替の効かない貴重品は航空便で送りましょう。

### Q5. 変圧器は必要？

ほとんどの場合、不要です。最近のPC・スマホの充電器は100-240V対応（アダプターに記載があります）。必要なのは変換プラグだけ。ただしドライヤーなど日本専用の熱器具を持っていく場合は変圧器が必要ですが、それなら現地で買い直したほうが安全です。

## まとめ — 「迷ったら持っていかない」で荷造りを始めよう

海外移住の持ち物は、重要書類・医薬品・電子機器・日本食材の4カテゴリを最優先で準備すれば大きな失敗はありません。衣類や日用品は現地調達のほうが気候にも合い、合理的です。

今日からできるアクションは3つです。

1. **この記事のリストをスプレッドシートにコピー**して、手持ちの荷物と照合する
2. **パスポートの残存期間と戸籍謄本の取得**を最優先で進める
3. **医薬品は早めにまとめ買い**しておく（処方薬は英文処方箋も忘れずに）

移住準備の全体像は「[海外移住の準備リスト](/kaigai-ijuu-junbi-list/)」で時系列に整理しています。現地の生活費が気になる方は「[パラグアイの生活費・物価事情](/paraguay-seikatsuhi/)」もあわせてチェックしてみてください。
