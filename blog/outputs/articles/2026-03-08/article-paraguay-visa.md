# パラグアイのビザの種類と取得方法を在住者が徹底解説【2026年最新】

パラグアイのビザは「取得しやすい」と言われることが多い。実際、他の南米諸国と比べるとハードルは低い方だと思う。ただし、2022年10月に法改正があって手続きが変わった部分もあるので、古い情報に惑わされないよう注意が必要だ。

私はパラグアイのランバレに妻と娘2人（8歳・6歳）の家族4人で住んでいる。ビザ取得の手続きは自分で経験しているので、この記事ではリアルな情報をまとめた。

【写真: パラグアイの移民局（Migraciones）の外観】

---

## パラグアイのビザの種類一覧

パラグアイのビザは大きく分けて以下の4種類がある。

| ビザの種類 | 有効期間 | 対象者 | 主な用途 |
|-----------|---------|-------|---------|
| ビザなし入国（観光） | 90日 | 日本国籍保持者 | 観光・短期滞在 |
| 一時居住ビザ（Radicación Temporaria） | 2年 | 外国人全般 | 長期滞在の第一ステップ |
| 永住ビザ（Radicación Permanente） | 10年（更新可） | 一時居住ビザ保持者・投資家・配偶者 | 定住 |
| 外交・公用ビザ | 任務期間 | 外交官等 | 外交・公務 |

---

## 1. ビザなし入国（観光目的・90日）

日本国籍を持っている場合、パラグアイには90日間ビザなしで滞在できる。入国時にパスポートにスタンプを押してもらうだけ。

### 基本情報

- **滞在可能日数**: 90日間
- **必要書類**: 有効なパスポート（残存有効期間6ヶ月以上推奨）
- **費用**: 無料
- **延長**: 移民局で90日の延長が可能（合計180日）【要確認: 延長手数料の最新金額】
- **注意**: 就労は不可

### 観光ビザで入国してからの手順

多くの移住者は、まず観光ビザで入国 → 現地で一時居住ビザを申請、という流れを取る。私もこのパターンだった。

---

## 2. 一時居住ビザ（Radicación Temporaria）— 永住権への第一ステップ

2022年10月18日の法改正により、パラグアイで永住権を取るにはまず2年間の一時居住ビザを取得する必要がある（一部例外あり）。

### 必要書類

| 書類 | 詳細 | 入手先 |
|------|------|-------|
| パスポート原本 + コピー | 残存有効期間6ヶ月以上 | 日本で取得 |
| 無犯罪証明書 | アポスティーユ認証 + スペイン語翻訳 | 日本の警察庁で取得 → 外務省でアポスティーユ |
| 戸籍謄本 | アポスティーユ認証 + スペイン語翻訳 | 日本の市区町村で取得 → 外務省でアポスティーユ |
| 健康診断書 | パラグアイ国内の指定医療機関 | 現地で取得 |
| パラグアイ国内の住所証明 | 賃貸契約書や公共料金の請求書 | 現地で取得 |
| 証明写真 | 指定サイズ | 現地で撮影可 |
| 銀行口座の残高証明 | パラグアイ国内の銀行口座に預金があること | 現地銀行で取得 |

### 費用の目安

| 項目 | 費用（目安） |
|------|------------|
| 移民局申請手数料 | 約3〜5万円【要確認: 2026年時点の最新金額】 |
| 無犯罪証明書（日本） | 無料（手数料はかからない） |
| アポスティーユ認証 | 無料（外務省窓口） |
| スペイン語翻訳（公証人付き） | 約1〜3万円 |
| 健康診断書 | 約5,000〜1万円 |
| セドゥラ（国民IDカード）申請 | 約5,000円【要確認】 |
| **合計** | **約5〜12万円** |

### 取得の流れ

1. **日本での準備**（1〜2ヶ月）
   - 無犯罪証明書の取得（申請から発行まで2〜3週間）
   - 戸籍謄本の取得
   - 両方の書類に外務省でアポスティーユ認証を受ける
   - スペイン語翻訳（パラグアイ現地で公証人の認証が必要）

2. **パラグアイ入国**
   - 観光ビザ（90日）で入国

3. **現地での手続き**（2〜6ヶ月）
   - 健康診断の受診
   - パラグアイ国内の銀行口座開設
   - 移民局（Dirección General de Migraciones）で一時居住ビザ申請
   - 審査期間は数週間〜数ヶ月（時期による）

4. **セドゥラの取得**
   - ビザ承認後、警察署（Departamento de Identificaciones）でセドゥラを申請
   - 2年有効の国民IDカードが発行される

## 3. 永住ビザ（Radicación Permanente）

一時居住ビザの有効期限が切れる90日前から、永住ビザへの切り替え申請ができる。

### 通常ルート

- **条件**: 2年間の一時居住ビザを保持していること
- **追加書類**: 一時居住期間中のパラグアイ滞在証明等
- **セドゥラ**: 10年有効のものに切り替え

### 即時申請が可能なケース（例外）

以下の場合は一時居住ビザを経由せず、直接永住ビザを申請できる。

- パラグアイ人の配偶者
- パラグアイ人の子ども・孫
- **投資家ビザ**（一定額以上の投資を行う場合）

### 投資家ビザの条件

パラグアイ国内の銀行口座に5,000 USD以上の預金を証明できれば、投資家として永住ビザを直接申請できる。他国の投資ビザ（数千万円〜数億円が必要）と比べると、ハードルは圧倒的に低い。

| 国 | 投資ビザの最低投資額（目安） |
|----|--------------------------|
| パラグアイ | 約80万円（5,000 USD） |
| アメリカ（EB-5） | 約1.3億円（800,000 USD） |
| ポルトガル（ゴールデンビザ） | 約8,000万円（500,000 EUR） |
| マレーシア（MM2H） | 約2,000万円相当の資産証明 |
| タイ（エリートビザ） | 約140万円〜（5年プラン）※投資ビザではない |

**パラグアイの投資額の低さは世界的に見ても異例**。ただし「5,000 USDを預金するだけ」なので、本格的な投資ビザというよりは預金証明に近い。

## 4. 日本での事前準備チェックリスト

パラグアイに渡航する前に、日本で準備しておくべきことをまとめた。

- [ ] 無犯罪証明書の申請（住所地の警察署 → 2〜3週間で発行）
- [ ] 戸籍謄本の取得（本籍地の市区町村役場）
- [ ] 外務省でアポスティーユ認証（郵送可、約1週間）
- [ ] パスポートの残存有効期間確認（6ヶ月以上推奨）
- [ ] 国際運転免許証の取得（運転する予定がある場合）
- [ ] 海外転出届の提出（住民票を抜く場合）
- [ ] 年金・健康保険の手続き
- [ ] 在パラグアイ日本国大使館の連絡先メモ

**重要**: 無犯罪証明書とアポスティーユは発行から一定期間内のものが有効。古すぎると受理されない場合があるので、渡航の1〜2ヶ月前に取得するのがベスト。

## ビザ取得で気をつけるべきこと

### 時間に余裕を持つ

パラグアイの行政手続きは、日本のようにテキパキ進まない。書類が足りないと言われて出直し、担当者がいなくて翌日に回される…ということは普通にある。焦らず、余裕を持ったスケジュールで動くこと。

### スペイン語の壁

移民局の窓口は基本的にスペイン語のみ。英語はほぼ通じない。スペイン語ができない場合は、通訳を連れていくか、日系のビザエージェントに依頼するのが現実的。

### 書類の不備に注意

アポスティーユの認証漏れ、翻訳の不備、写真のサイズ違いなど、些細なことで申請が差し戻される。事前に在パラグアイ日本国大使館や経験者に確認するのが確実。

## よくある質問（FAQ）

### Q1: 日本人はパラグアイに何日間滞在できる？

ビザなしで90日間。移民局で延長手続きをすれば最大180日まで滞在可能。それ以上滞在する場合は一時居住ビザの取得が必要。

### Q2: ビザの申請は日本のパラグアイ大使館でもできる？

一時居住ビザ・永住ビザはパラグアイ現地の移民局で申請するのが一般的。日本のパラグアイ大使館では観光ビザ関連の情報提供は受けられるが、居住ビザの発給は現地で行われる。

### Q3: ビザの取得にどれくらいの期間がかかる？

一時居住ビザは申請から数週間〜数ヶ月。時期や書類の準備状況による。永住ビザへの切り替えは、一時居住ビザ取得から2年後に申請可能。

### Q4: 家族も一緒にビザを取れる？

家族それぞれが個別に申請する必要がある。ただし、家族分の書類をまとめて提出できるので、手間は1人分とそこまで変わらない。子どもの場合は戸籍謄本で親子関係を証明する。

### Q5: 永住権を取ったら日本国籍はどうなる？

パラグアイの永住権は「永住ビザ」であって「国籍取得」ではない。日本国籍はそのまま保持できる。パラグアイの国籍を取得する場合は別途手続きが必要で、その場合は日本の国籍法との兼ね合いを確認すること。

### Q6: 永住権を持っていても長期間パラグアイを離れたらどうなる？

永住権保持者がパラグアイを長期間（3年以上【要確認】）離れると、永住権が失効する可能性がある。定期的にパラグアイに戻るか、事前に移民局に届け出をしておくこと。

パラグアイの永住権は、預金5,000 USDの証明で申請できるという点で、世界的に見てもハードルが低い。2022年の法改正で「まず2年の一時居住ビザ → 永住ビザ」という2段階になったものの、それでも他国と比べれば取得しやすい部類に入る。
