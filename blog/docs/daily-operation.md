# 日次運用マニュアル v2.0

「南米おやじのAI実践ラボ」のオーナー向け日常作業ガイド。

---

## Phase 1（Month 1-3）: 1日2-3時間

立ち上げ期間。自動化の調整、記事品質の確認、X（Twitter）運用の確立に集中する。

### 毎日のルーティン

#### 1. 記事確認・追記・公開（30-45分）

1. WordPress管理画面にログイン
2. 「投稿」→「下書き」を確認
3. AI生成記事の内容をざっと読む
4. **【要追記】箇所に自分の体験談を追加する**（これが最重要の作業）
5. アイキャッチ画像を設定
7. 問題なければ「公開」ボタンを押す

**チェックポイント**:
- パラグアイ・海外の視点が入っているか？
- タイトルは自然か？数字が入っているか？
- アフィリエイトリンクは適切に挿入されているか？
- 明らかな事実誤認はないか？
- AI臭い表現が残っていないか？

#### 2. X（Twitter）運用（30分）

1. `outputs/social/` の今日分のツイートを確認
2. 内容を確認し、必要に応じて微調整
3. X（Twitter）に投稿（朝・昼・夜の3回）
4. パラグアイの写真があれば添付（**差別化の最大の武器**）
5. 3-5件のリプ or 引用RTで他アカウントと交流

**Phase 1の鉄則**:
- 最初の3ヶ月は「売らない」。ファンを作ることに集中
- 数字をオープンに（PV、収益、失敗も含めて）
- パラグアイの日常写真を積極的に投稿

#### 3. ココナラ対応（15分）

1. ココナラのメッセージを確認
2. 新規問い合わせがあれば返信
3. 既存の `../business/` ワークフローで対応

#### 4. デジタル商品の作業（30-60分）

1. 現在執筆中の商品を進める
   - Month 1: パラグアイ移住ガイド
   - Month 2: AI副業スタートキット
   - Month 3: 海外AI副業ロードマップ
2. 完成した商品の販売ページをチェック
3. 販売実績を確認（note/Brain/Gumroadの管理画面）

---

## Phase 2以降（Month 4+）: 1日60-90分

自動化が安定した後の運用モード。

### 毎日のルーティン

#### 1. 記事確認→追記→公開（15-30分）
- ドラフト記事をざっと読む
- 【要追記】に体験談を追加
- 公開ボタンを押す

#### 2. X投稿（15分）
- 自動生成済み投稿をコピペして投稿
- 軽いリプ・いいねで交流

#### 3. ココナラ対応（15分）
- メッセージ確認・返信

#### 4. レポート確認 or 商品アイデア検討（15分）
- 月曜: 週次レポートを読む
- それ以外: 商品アイデアやリライト候補の検討

---

## 毎週月曜のルーティン（+15分）

### 週次レポート確認

1. `outputs/reports/weekly-YYYY-MM-DD.md` を開く
2. 二本柱別のPV・収益を確認
3. X（Twitter）のフォロワー増加・エンゲージメントを確認
4. 「来週のアクション」を読み、対応を決める

**判断が必要な項目**:
- 特定記事のリライト推奨 → Claude Codeに依頼
- 新しいKW候補 → content-calendar.jsonに反映
- デジタル商品の売上好調 → 値上げタイミングの検討

---

## 毎月1回のルーティン（+30分）

### 収益と商品の確認

1. 全収益源の月間まとめ（デジタル商品 / アフィリ / AdSense）
2. フェーズ目標との比較（達成率をチェック）
3. 次の商品のアイデアを `inputs/product-ideas/` にメモ
4. アフィリエイト案件の新着を確認（A8.net / もしもアフィリ）
5. 戦略の微調整が必要か判断

## トラブルシューティング

### スクリプトが動かない
```bash
python scripts/[スクリプト名].py
pip install -r requirements.txt
```

### WordPress投稿が失敗する
- config/settings.json のWordPress URL・パスワードを確認
- アプリケーションパスワードが有効か確認

### 記事の品質が低い
- agents/article-writer-agent.md のプロンプトを調整
- Claude Codeに「この記事を改善してください」と依頼

### デジタル商品が売れない
1. Xでの告知頻度を上げる（週2-3回）
2. 販売ページのコピーを改善
3. 価格を下げてみる（最初は安く始める）
4. 購入者レビューをXでシェアする

### Xフォロワーが増えない
1. パラグアイの写真投稿を増やす
2. 数字を公開する投稿を増やす
3. 他アカウントとの交流を増やす（リプ週10件以上）
