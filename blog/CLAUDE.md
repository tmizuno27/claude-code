# CLAUDE.md

このファイルは、Claude Code がこのリポジトリで作業する際のガイドです。

## プロジェクト概要

**事業名**: 南米おやじのAI実践ラボ
**コンセプト**: パラグアイ在住の日本人が、AIを使って副業でどこまで稼げるかをリアルに発信する「実験ログ型メディア」
**ドメイン**: nambei-oyaji.com（既存パラグアイ事業と共用）

**目標**:
- Phase 1（3ヶ月）: 記事15-20本、デジタル商品2つ販売開始、Xフォロワー500人
- Phase 2（12ヶ月）: 月収5-8万円（新規事業のみ）
- Phase 3（18-24ヶ月）: 月収15-20万円（既存事業合算で月20万円）

**収益の優先順位**:
1. デジタル商品（note/Gumroad/Brain）— 利益率90%+
2. 高単価アフィリエイト（サーバー/スクール/VPN）— 1件5,000-50,000円
3. AdSense — サブ収益として月1-2万円

**詳細な事業計画**: `docs/business-plan-v2.md` を参照

## ライティングルール（全エージェント共通）

### 必須ルール
- 一人称は「私」（オーナー本人として書く）
- **「パラグアイから実践している」という文脈を全記事に入れる**
- カジュアルだが信頼感のある文体（友達に話すトーン + 正確な数字）
- 具体的な数字・ツール名を必ず入れる（「安い」→「月1,000円」）
- 実体験に基づかない内容は捏造しない（「【要追記】」と記載）
- 写真が必要な箇所は「【写真: ○○の写真を挿入】」と明示
- **【最重要】ファクトチェック必須**: 記事作成・X投稿の両方で、公開前に必ずファクトチェックを実施する
  - 数字・統計・法律・手続き・料金・税率等の事実情報はWebSearchで最新情報を確認
  - 特にパラグアイの法律・税制・ビザ要件は変更が多いため、毎回最新情報を検索すること
  - 誤情報は読者の人生判断に直結するため絶対に公開しない
  - 確認が取れない情報は「【要確認】」タグを付けてドラフト段階で留める

### 禁止事項
- AI的な表現：「いかがでしたでしょうか」「〜と言えるでしょう」「〜ではないでしょうか」
- 「まとめると〜」で始まる結論
- 「〜することが大切です」の羅列
- 過度な敬語の積み重ね
- 箇条書きだけで段落を埋める
- 体験のない内容を体験したかのように書く

### アフィリエイトリンク
- `config/affiliate-links.json` から取得して自然に挿入
- 1記事あたり上限5リンク、リンク間隔300文字以上
- 高単価案件（サーバー、スクール、VPN）を優先的に配置
- 免責文を記事冒頭または末尾に1箇所挿入

## 技術スタック

- **AI**: Claude API (claude-sonnet-4-6)
- **CMS**: WordPress on ConoHa WING + Cocoon テーマ + Rank Math SEO
- **SEO調査**: Google Suggest API（keyword_research.py）
- **スクリプト**: Python 3.13
- **PDF生成**: WeasyPrint
- **分析**: GA4 API + Search Console API
- **商品販売**: note（主力） + Brain + Gumroad
- **SNS**: X（Twitter）— メイン集客チャネル
- **タスク自動化**: Windows Task Scheduler
