### プロンプト 1-01

**タイトル**：議事録の自動生成

**目的**：会議のメモや録音テキストから、読みやすい議事録を瞬時に作成する

**プロンプト本文**：

```
以下の会議メモをもとに、正式な議事録を作成してください。

【会議情報】
- 日時：[日時を入力]
- 参加者：[参加者名を入力]
- 議題：[議題を入力]

【メモ】
[会議メモをここに貼り付ける]

【出力形式】
1. 会議の概要（3〜5文）
2. 決定事項（箇条書き）
3. 次回アクションと担当者・期日（表形式）
4. 次回会議の議題（案）
```

**活用のヒント**：
- 録音をWhisperやNotionAIで文字起こしした後にこのプロンプトを使うとさらに効果的です
- 「です・ます」調や「だ・である」調など、文体の指定を追加することができます
- 参加者のメモが断片的でも、AIが文脈を補完して整理してくれます

**期待される出力**：
- 構造化された議事録（400〜600字程度）
- 箇条書きの決定事項リスト
- 次回アクション表

### プロンプト 1-02

**タイトル**：メールの返信文作成

**目的**：受信したメールに対して、適切なトーンとフォーマットで返信文を作成する

**プロンプト本文**：

```
以下のメールに対する返信文を作成してください。

【受信メール】
[受信メールの本文をここに貼り付ける]

【返信の要件】
- トーン：[ビジネス丁寧 / カジュアル / フォーマル]
- 返答する主な内容：[返答内容を箇条書きで入力]
- 避けたいこと：[例：過度な謝罪、曖昧な表現など]
- 文字数：[短め(200字以内) / 普通(200〜400字) / 詳細(400字以上)]

【出力】
件名と本文を含む返信メール全文を作成してください。
```

**活用のヒント**：
- クレーム対応、お断りメール、依頼メールなど状況を具体的に指定すると精度が上がります
- 「〇〇さん（担当者名）への返信」と文脈を加えるとより自然な文章になります

**期待される出力**：

# 付録：プロンプトの効果的な使い方
# ========================================

## 付録：プロンプトをさらに効果的に使うコツ

### 基本原則

1. **具体性を高める**：「詳しく」より「300字で」「3つのポイントで」など数値指定が有効
2. **役割を与える**：「あなたはマーケティング専門家です」と前置きすると品質が上がる
3. **出力形式を指定**：「箇条書き」「表形式」「JSON」など形式を明示する
4. **段階的に依頼**：一度に全部頼まず、まず構成→次に本文と段階を踏む

### よく使う指示フレーズ

| 目的 | フレーズ例 |
|------|-----------|
| 短くする | 「200字以内で要約してください」 |
| 長くする | 「各ポイントを2〜3文で詳しく説明してください」 |
| 書き直す | 「もっと[casual/formal/親しみやすく]書き直してください」 |
| バリエーション | 「5パターン案を出してください」 |
| 改善する | 「この文章の改善点を指摘し、改善版を作成してください」 |

### トラブルシューティング

| 問題 | 解決策 |
|------|--------|
| 回答が短すぎる | 「もっと詳しく」「文字数を〇〇字以上にして」と追加指示 |
| 回答がズレている | 「私が求めているのは〇〇です。もう一度お願いします」 |
| 日本語が不自然 | 「日本語ネイティブが書いたような自然な表現にしてください」 |
| 情報が古い | 「〇〇年時点での情報として」または最新情報を自分で追加する |
