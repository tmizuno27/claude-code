Privacy Policy — Quick Currency Converter

Last updated: March 16, 2026

Overview

Quick Currency Converter is a browser extension that converts currencies using real-time exchange rates. Your privacy is important to us. This policy explains what data the extension accesses and how it is handled.

Data Collection

We do not collect any personal information. The only data transmitted externally is the currency pair (e.g., "USD to EUR") sent to an exchange rate API to retrieve the current rate. No personal information, browsing history, or identifying data is ever collected or transmitted.

Data Sent to Third Parties

  • Exchange Rate API — The extension sends only the selected currency codes (e.g., "USD", "JPY") to retrieve exchange rates. No other data is included in these requests.

Data Storage

The extension stores your preferences (last-used currencies, favorite pairs) using the browser's local storage API (chrome.storage.local). This data never leaves your device.

Permissions

  • activeTab — Used for right-click context menu currency conversion on selected text.
  • storage — Used to save your currency preferences and favorites locally.
  • contextMenus — Used to add a right-click conversion option.

Third-Party Services

This extension uses a public exchange rate API solely to fetch current currency rates. No analytics, tracking, or advertising services are used.
