## License Key Generation

For Gumroad, use the "Generate a unique license key" option when creating the product.
Gumroad auto-generates license keys for each purchase. The extension validates by checking
that the key exists and has 8+ characters (simple client-side validation).

For stronger validation in the future, integrate Gumroad's license verification API:
`POST https://api.gumroad.com/v2/licenses/verify`
