Privacy Policy — Page Speed Checker

Developer: t.mizuno27

Last updated: March 16, 2026

Data Collection

This extension sends the URL of the current webpage to Google PageSpeed Insights API to retrieve performance data. No other personal data is collected.

Third-Party Services

This extension communicates with Google PageSpeed Insights API. Google privacy policy applies: https://policies.google.com/privacy

Data Storage

No data is stored externally. The developer does not collect or have access to any URLs you analyze.

Contact

For questions about this privacy policy, contact: t.mizuno27 via the Chrome Web Store developer page.
