# マスタープロンプト v2.0（コピペ用コマンド集）

Claude Codeで以下をコピペするだけで各タスクを実行できます。

---

## 1. キーワード調査（二本柱対応）

```
scripts/keyword_research.py を実行してください。
config/content-calendar.json の二本柱のシードキーワードから関連KWを調査し、
inputs/keywords/ に優先順位付きリストを出力してください。
パラグアイ系6件 + AI副業系14件 = 合計20件。
```

## 2. SEO記事生成

```
scripts/article_generator.py を実行してください。
inputs/keywords/ の最新キューから次のKWを取り、
agents/article-writer-agent.md のルールに従ってSEO記事を生成し、
outputs/articles/ に出力してください。
記事タイプ（集客/収益/キラー/実験）も指定してください。
```

## 3. WordPress投稿

```
scripts/wp_publisher.py を実行してください。
outputs/articles/ の未投稿記事をWordPressにドラフト投稿してください。
投稿後、公開前チェックリストとX投稿文を表示してください。
```

## 4. デジタル商品作成

### アウトライン作成（まずこちら）
```
agents/product-creator-agent.md に従って、
商品ロードマップの [番号] 番の商品のアウトラインを作成してください。
出力先: outputs/products/
```

### 本文生成（アウトライン承認後）
```
上記のアウトラインで本文を作成してください。
販売ページコピーとX告知文も同時に生成してください。
出力先: outputs/products/
```

## 5. X投稿生成（1週間分）

```
agents/sns-scheduler-agent.md に従って、
今週公開した記事と最近のトピックをもとに
1週間分のX投稿案を作成してください。

公開済み記事:
- [記事タイトル1]
- [記事タイトル2]

出力先: outputs/social/
```

## 6. 内部リンク最適化

```
scripts/internal_linker.py を実行してください。
WordPress上の全記事を分析し、パラグアイ記事⇔AI副業記事の
相互リンクを中心に内部リンクを追加してください。
```

## 7. 週次レポート

```
agents/analytics-agent.md に従って、
今週の週次レポートを生成してください。

対象期間: [日付]
現在のフェーズ: Phase 1（Month [X]）

出力先: outputs/reports/
```

## 8. コンテンツカレンダー更新

```
config/content-calendar.json を更新してください。
最新のKW調査結果と公開済み記事を反映し、
来月分のスケジュールを追加してください。
```

## トラブルシューティング

### スクリプトが動かない
```bash
python scripts/[スクリプト名].py
pip install -r requirements.txt
```

### WordPress投稿が失敗する
- config/settings.json のWordPress URL・パスワードを確認
- アプリケーションパスワードが有効か確認
- WordPress REST APIが有効か確認

### 記事の品質が低い
- agents/article-writer-agent.md のプロンプトを調整
- 「パラグアイからの視点が薄い」場合は、パラグアイセクションの追加を指示

### デジタル商品が売れない
1. Xでの告知頻度を確認（週2-3回は告知する）
2. 販売ページのコピーを改善
3. 価格設定を見直す（最初は安く始めて段階値上げ）
4. 購入者レビューをXでシェア

### Xフォロワーが増えない
1. パラグアイの写真投稿を増やす（差別化要素）
2. 数字を公開する投稿を増やす（実験報告）
3. 他アカウントとの交流（リプ・引用RT）を増やす
4. 投稿時間を日本時間に合わせる
