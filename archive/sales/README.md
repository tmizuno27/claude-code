## 運用ルール

- 新しい文字起こしを受け取ったら、内容を分析し**該当するカテゴリのファイルに知識を追記**する
- 1つの文字起こしが複数カテゴリにまたがる場合、それぞれのファイルに分散して追記する
- 発言者（誰のナレッジか）を明記し、再現性を高める
- 議事録として日付ベースで保存するのではなく、**スキル・知識単位で蓄積**する
