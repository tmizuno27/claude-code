# 朝のブリーフィング

以下を順番に実行し、1つのレポートにまとめて報告せよ。

## 1. ログ健全性チェック
`logs/` 配下の全ログファイルを確認:
- `auto-sync.log` の最終同期時刻
- `sheets-sync.log` の最終同期時刻
- `x-auto-post*.log` の最終投稿時刻（3アカウント）
- 各サイトの分析・記事生成・メンテナンスログ
- エラー・警告があれば即報告

## 2. 優先アクション確認
- `memory/dashboard-priority-actions.md` を読み込み
- 昨日の未完了アクションを特定
- 今日の推奨アクション（収益インパクト順）を提示

## 3. 全事業ステータスサマリー
各事業の直近24時間の変化を確認:
- **3ブログ**: 新規記事・PV変動・検索順位変化
- **RapidAPI**: API呼び出し数・売上
- **Gumroad**: 販売数・閲覧数
- **Chrome/VSCode拡張**: ダウンロード数・審査状況
- **X投稿**: エンゲージメント・フォロワー変化

## 4. 今日のToDoリスト
上記を踏まえ、今日やるべきことを優先度順（収益インパクト順）に5〜10個提示。
すぐ実行できるものは「自動実行可」マークを付ける。

## 出力ルール
- 全数値はリアルタイム取得（ハードコード禁止）
- 問題があれば🔴、注意は🟡、正常は🟢で色分け
- コピペが必要な項目はコードブロックで提示
